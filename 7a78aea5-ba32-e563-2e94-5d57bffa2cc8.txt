顾夜瑾真的垂眸看了美甲书。
<br/>
<br/>        夏夕绾看着顾夜瑾俊俏如玉的面容，他的英俊绝对不亚于陆寒霆，精良的西装衣料包裹着他颀长的身躯，斯文俊雅的外表从上往下都透着一股冷静的克制和清寒的疏离。
<br/>
<br/>        这样的男人让人想要靠近，又不敢靠近。
<br/>
<br/>        夏夕绾真的不敢相信刚才叶翎说的，他喜欢最漂亮的女人，美甲也会挑鲜艳的红色，这位顾少应该不是这么肤浅的人吧，所以夏夕绾作死的想要验证一下。
<br/>
<br/>        这时顾夜瑾伸手指了一下，“这个不错。”
<br/>
<br/>        他指的是…红色。
<br/>
<br/>        就是叶翎做的颜色。
<br/>
<br/>        夏夕绾，“…”
<br/>
<br/>        这时叶翎从里面出来了，“哥，你来了？”
<br/>
<br/>        顾夜瑾抬眸看到了她，目光也扫到了她新做的指甲。
<br/>
<br/>        “哥，我做的美甲好看么？”叶翎伸出小手在他面前晃了晃。
<br/>
<br/>        好看。
<br/>
<br/>        怎么会不好看？
<br/>
<br/>        叶翎出身名门，从大院来到商贾簪缨世家，一直是温室里养着的红玫瑰，浑身娇滴滴的。
<br/>
<br/>        她手如青葱，从来没沾过阳春水，这种鲜丽的红色将她白腻的肌肤衬的晃眼，那唇红齿白里都流淌出了一股水媚。
<br/>
<br/>        顾夜瑾眸色暗了几个色调，“好看。”
<br/>
<br/>        叶翎收回自己的小手，拉住了夏夕绾，“绾绾，我们走吧，哥，我们买的衣服辛苦你拿了。”
<br/>
<br/>        能让顾夜瑾帮忙提袋子，夏夕绾觉得十分的荣幸。
<br/>
<br/>        ……
<br/>
<br/>        三个人出了商场，顾夜瑾的座驾银色迈巴赫已经停着等候了，“上车，我送你们回家。”
<br/>
<br/>        叶翎扬了扬手里的车钥匙，“哥，我们开车来的，就不用你送了，夏妍妍和霍璇不是也在里面么，你送她们吧。”
<br/>
<br/>        顾夜瑾看了一眼她手里的车钥匙，“什么时候学开车的？以前你站在大街上就会丢，东南西北都分不清。”
<br/>
<br/>        叶翎抬手将耳畔几缕茶色的卷发掖到了耳后，“哥，我已经长大了嘛，不能总依赖着你呀。”
<br/>
<br/>        顾夜瑾看着她，嗓音淡漠的毫无起伏，“车子放这里，我送你们回家。”
<br/>
<br/>        这时一串悠扬的手机铃声响起了，顾夜瑾来电话了。
<br/>
<br/>        “哥怎么不接，我猜一定是夏妍妍打来的电话，让你送她们回家，不信接接看。”
<br/>
<br/>        顾夜瑾拿出手机，按键接通。
<br/>
<br/>        他没有隐瞒的意思，就这样接了，所以夏妍妍娇美的嗓音迅速传递了过来，“喂，夜瑾，你现在在哪里，可以过来接我和璇璇么，我们的车抛瞄了，需要送到店里去修。”
<br/>
<br/>        顾夜瑾看向叶翎，叶翎一双水媚的眸子也看着他，还对他挑了挑柳眉，看他怎么选。
<br/>
<br/>        顾夜瑾很快就选择了夏妍妍，对着叶翎掀了掀薄唇，“开车小心。”
<br/>
<br/>        叶翎毫无意外，夏妍妍一通电话总是能将他给叫走，“知道了哥，我和绾绾先走了。”
<br/>
<br/>        ……
<br/>
<br/>        银色的迈巴赫疾驰在了路上，顾夜瑾在专心开车，车后面是夏妍妍和霍璇。
<br/>
<br/>        夏妍妍很抱歉道，“夜瑾，真的是不好意思了，原本你是要送翎翎和绾绾的，现在我让你送我和璇璇，翎翎没有跟你闹吧，以前翎翎可是很介意我们在一起的，只要看到了就会跟你发脾气。”

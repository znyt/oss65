　　约莫半个小时左右，服务员将海鲜全部上齐，并且送来了一瓶白酒，两人就边吃边聊了起来。
<br/>　　“最近，集团那边怎么样？”
<br/>　　对于集团方面的事情，慕清霄本不想过问，因为有欧阳紫嫣和雅儿她们在，根本不需要自己操心。
<br/>　　但想到自己也是董事长，顿时就觉得有些尴尬，甩手掌柜做到他这份上，也是没谁了。
<br/>　　一杯白酒下肚，欧阳紫嫣俏脸涌上一抹红晕：“发展的挺不错，各项技术已经占领了市场，随时可以向国外发展。”
<br/>　　“集团的事情，你看着办吧，如果累了就停下来休息休息，出去旅游也是不错的点子。”
<br/>　　欧阳紫嫣毕竟是元婴期的修士，虽说都是天地财宝堆积起来的，但却一点不必普通修士差，甚至更强。
<br/>　　区区打理一些普通事情，根本不可能觉得累。
<br/>　　“嗯，我知道。”
<br/>　　虽说是老夫老妻了，但听到慕清霄关心自己，欧阳紫嫣心里却是甜滋滋的，捋了捋额前的青丝，美眸静静的看着他。
<br/>　　就在两人交流的时候，隔壁桌一名气质沉稳的男子站了起来。
<br/>　　他注意欧阳紫嫣已经有很长时间了，这么漂亮的女人，连电视里的女明星都差远了，心中充满了贪婪与占有欲。
<br/>　　“美女，交个朋友？”
<br/>　　温馨的气氛突然被打断，欧阳紫嫣抬起臻首，看向男子的眼神满是冰冷之色。
<br/>　　如果是普通女子，见到如此有气质的男子，心中恐怕会感到惊喜也说不定，但她却不屑一顾。
<br/>　　见欧阳紫嫣冷冰冰的模样，男子暗自吞咽了口唾沫，强占她的想法更加坚定，这气质这容貌，简直是极品女神啊！
<br/>　　这时，慕清霄也放下碗筷，视线在男子身上打量一圈，剑眉皱在一起，眼中满是诧异。
<br/>　　一来他觉得男子有些面熟，二来则是男子身上居然有一股与他相似的气息。
<br/>　　随后，给欧阳紫嫣投去一个眼神。
<br/>　　欧阳紫嫣心中虽有些困惑，但还是朝男子微微一笑，道：“坐下，我们可以深入交流一下。”
<br/>　　深入交流！
<br/>　　中年男子喉结滚动了一下，立刻乖乖坐下，道：“美女，这里环境不好，附近有个酒吧，不如去那，我请你喝一杯？”
<br/>　　欧阳紫嫣委婉拒绝，套话的同时，慕清霄则静静的看着男子，面色如常，但心中却掀起了惊涛骇浪。
<br/>　　原因无他，因为面前的男子，居然是一名修士！
<br/>　　没有，居然是修士！
<br/>　　虽没有交手，但慕清霄可是练虚九重的修为，而且不是普通练虚期，男子的一切，在他面前，根本无所遁形。
<br/>　　虽说只有练气二重的修为，却犹如深水炸弹，在他的心中炸响。
<br/>　　要知道这可是地球啊，空气中有稀薄的灵气，但哪里来的心法，支撑男子修炼至练气二重？
<br/>　　“我叫李春，静海市本地人。”
<br/>　　李春？
<br/>　　话语刚落，慕清霄眼中闪过一抹思索，不管是名字或者长相，似乎在那里见过听过，印象有些模糊。
<br/>　　“紫嫣，时间不早了，我们回去吧。”
<br/>　　“嗯。”
<br/>　　说罢，欧阳紫嫣就给男子投去一个怪异的眼神，跟在慕清霄身后，转身离去。
<br/>　　从始至终，男子都表现的非常有气度，但眼中贪婪的目光，怎可能躲过慕清霄与欧阳紫嫣的视线。
<br/>　　贪婪的目光一闪即逝，望着欧阳紫嫣的背影，旋即目光落在慕清霄身上，男子双眼一眯，其中杀意翻滚起来。
<br/>　　挣扎片刻，似乎做下了某种决定，冷笑一声后，站起身来就跟了上去。
<br/>　　结账离开美食街，慕清霄就坐上宝马五系，欧阳紫嫣则坐在副驾驶上，美眸中满是困惑，道：“他给的感觉，有些奇怪。”
<br/>　　“他和我们是同一类人，虽然很弱。”
<br/>　　慕清霄的话语，令得欧阳紫嫣瞳孔微微一缩，不可思议道：“我记得，你当初说过，地球不可能有修士？”
<br/>　　瞥了眼倒车镜，慕清霄微微一笑，道：“我当初也是这么觉得，但他的出现，却改变了我的看法。”
<br/>　　说罢，慕清霄脚踩油门，车子就以缓慢的速度驶入主道，望着倒车镜中画面，嘴角牵起一丝笑意。
<br/>　　“真是色胆包天，很快我们就知道结果了。”
<br/>　　欧阳紫嫣可不是单纯的女子，下一秒就理解了慕清霄的意思，望着身后的车流，美眸中满是冷意。
<br/>　　男子贪婪的眼神虽隐藏在眼底深处，但无论如何隐藏，都揭示着他的目的。
<br/>　　本个小时的车程，慕清霄驾驶着汽车，行驶在公路上，但距离市中心却越来越远，周围也原来越接近郊区。
<br/>　　将车停在路边，将灯熄灭，距离宝马五系百米外的地方，同样有一辆黑色别克商务车停了下来。
<br/>　　车内，望着前面的宝马五系，男子眼中满是阴冷的杀意：“刚才装的那么清高，现在却来这里车震，真是对狗男女。”
<br/>　　“咚咚!”
<br/>　　就在这时，车窗敲响，男子转头望去，脸色猛的一变，因为出现在窗外的，可不就是刚才的青年吗？
<br/>　　怎么可能？
<br/>　　心头狂跳，男子顿时有一种不好的预感，他的视线明明一直盯着前面的车，后者什么时候下车的，他怎么没看见？
<br/>　　“嘭...!”
<br/>　　就在男子愣神的同时，车窗陡然爆碎，玻璃碎片扎了他一脸。
<br/>　　伴随着一声惨叫，鲜血顺着男子脸上流淌而下。
<br/>　　刚回过神来，便有一只手直接掐住了他的脖子，硬生生的将他从驾驶座上拖了出去。
<br/>　　“砰！”
<br/>　　望着地上被摔得七晕八素的男子，慕清霄饶有兴趣的笑道：“如果我没记错，将近一年前，你似乎是失踪名单上的其中一人吧？”

<br/>　　将近在一年前，全国各地人口失踪，失踪的人皆在三十岁以下，而其中一名男子，似乎就是叫李春。

<br/>　　仅仅一年的时间，从一名普通人，变成练气二重的修士，慕清霄对此可是非常的感兴趣。

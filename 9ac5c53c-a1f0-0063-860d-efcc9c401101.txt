　　人王城，坐落在轮回神朝的东部地域。
<br/>　　此城之大，无与伦比。
<br/>　　作为轮回神朝的六大主城之一，人王城长宽均有数千里，城墙巍峨，仿佛要耸入云端。
<br/>　　城中人流如织，店铺林立，比之极空城还要繁华。
<br/>　　走在城中的街道上，苏莫已经换了一副衣着。
<br/>　　他穿了一身朴素的青衫，显得毫不入眼，普普通通。
<br/>　　这就是他要的效果，不引人瞩目。
<br/>　　除此之外，他已经做好了一切隐藏的手段，比如吞噬元神、混沌之力。
<br/>　　混沌之力，他分解了一大半，布于丹田之中，将混沌圣丹包裹。
<br/>　　吞噬元神他同样隐藏的极深，亦是被浑厚的五行玄力包裹，不到生死时刻，他不会动用。
<br/>　　苏莫认为，现如今，除非帝一魂和拓跋青云亲来，不然，估计没有谁能看出他是分魂。
<br/>　　苏莫大步而行，一步数十丈，向城北走去。
<br/>　　人王宫，就在城北，占据了整个城北地域。
<br/>　　良久之后，他来到了城北，目光往见了人王宫。
<br/>　　人王宫，用一堵高耸的院墙，将其和人王城分隔了开来。
<br/>　　人王宫的宫门，仿佛是青铜打造，如同巨大的城门，雄伟壮阔。
<br/>　　宫门前，伫立着十六名守卫，分两侧而立，个个气息浩大，实力惊人。
<br/>　　人王宫非常巨大，宫内建筑连绵，或金碧辉煌，或美轮美奂，可谓是琼楼玉宇，愧丽不凡。
<br/>　　苏莫没有接近人王宫，就在人王宫宫门对面不远处，找了一家客栈安顿了下来。
<br/>　　他要等，等那位沁儿姑娘回来。
<br/>　　他想要进入人王宫，还需要从此女身上着手。
<br/>　　盘膝坐在房间之中，苏莫一边守株待兔，留意着人王宫的宫门，一边暗自思索。
<br/>　　若是可以的话，他估计要在人王宫待很久了。
<br/>　　这里是最危险的地方，但同时，也是最安全的地方。
<br/>　　其一，他要借助人王宫的资源，提升修为实力。
<br/>　　其二，是慢慢接触宫中的强者，和人王宫中的那些强者搞好关系，然后再徐徐策划。
<br/>　　当然，最主要还是，他需要想办法得知，人王宫的宫主，对于帝一魂的态度。
<br/>　　时间缓缓流逝，苏莫也不再修炼，就安心的等待了起来。
<br/>　　就这样，过去了三日之后，沁儿回来了。
<br/>　　当然，还有褚彦和骆妤两人，与之同行。
<br/>　　苏莫见此，立刻离开客栈，向沁儿飞掠而去。
<br/>　　“沁儿姑娘！”
<br/>　　就在沁儿三人，快要走到人王宫宫门之时，苏莫快速而来，朗声喊道。
<br/>　　“嗯？”
<br/>　　听到喊声，沁儿转头一看，见到是苏莫找她，顿时微微一怔。
<br/>　　“又是你！”看到苏莫再次纠缠，褚彦和骆妤两人脸色一沉。
<br/>　　不过，很快褚彦的嘴角，便勾起了一丝笑意。
<br/>　　现在已经到了家门口，他也不会有任何的顾忌了。
<br/>　　“段惊天，你有何事？”沁儿好奇的问道，她不太明白，眼前的青年找她有何事。
<br/>　　“沁儿姑娘，在下想拜入人王宫，不知姑娘能否引荐？”苏莫来到沁儿的身前，郑重的抱拳说道。
<br/>　　“你想拜入人王宫？”沁儿闻言，顿时小脸上露出惊讶之色。
<br/>　　“呵呵！”
<br/>　　这时，褚彦突然笑了起来，不屑的道：“难怪你三番二次的接近我们，原来是想要拜进人王宫！”
<br/>　　“不错，三位能帮忙吗？”苏莫点了点头，环视三人。
<br/>　　骆妤脸色平淡，并没有开口的意思。
<br/>　　“这……！
<br/>　　沁儿面现迟疑之色，她没有立刻同意。
<br/>　　毕竟，她虽然心思单纯，也也并不傻。
<br/>　　她若是举荐，自然没有什么问题，只是她对苏莫一无所知，自然不能举荐一个陌生人。
<br/>　　毕竟，若是苏莫真有什么歹心，那不是养虎为患吗？
<br/>　　“帮忙？”
<br/>　　褚彦轻笑一声，不屑道：“你身份不明，居心叵测，居然还想要加入人王宫？”
<br/>　　“在下来自终极剑域，乃是一个散修，只是想要投靠一个大势力而已，并无什么不良居心！”苏莫耐着性子，沉声说道。
<br/>　　“你给我滚！”
<br/>　　然而，褚彦却是丝毫不给面子，声音一冷，厉声喝道：“三息之内，不要让我在见到你，不然后果自负！”
<br/>　　苏莫闻言，脸色一沉，心中不禁恼怒，恨不得一巴掌乎死这个褚彦。
<br/>　　“沁儿姑娘！”苏莫没有立刻离开，看向沁儿姑娘，等待对方回答。
<br/>　　“你天赋怎么样？我们人王宫一般不招收外人，只招天才弟子！”沁儿略微沉吟，轻声问道。
<br/>　　“我不足百岁，修为已达虚神四变，天赋很高！”苏莫沉声道。
<br/>　　闻听苏莫之言，骆妤美眸中露出玩味之色。
<br/>　　不足百岁，虚神四变？
<br/>　　这也算是天赋很高吗？
<br/>　　这种天赋之人，人王宫内一抓一大把。
<br/>　　毕竟，不足百岁虚神四变，和不足百岁达到真神，这完全是两个概念。
<br/>　　“沁儿，不必理会他！”
<br/>　　褚彦面现不耐之色，随即，大袖一挥，一股气浪卷向苏莫。
<br/>　　褚彦的实力极强，因为和苏莫相距极近，气浪顿时击中了苏莫。
<br/>　　嘭！
<br/>　　一声闷响，苏莫的身形立刻倒飞了出去，急速远离。
<br/>　　“可恶！”苏莫心中大怒，但是，他强忍住没有出手，不然的话会有很大的麻烦。
<br/>　　“沁儿姑娘，我在对面的客栈之中，等待你的答复！”
<br/>　　苏莫意念传音，向沁儿传去了一句话，随即，他借势远离了人王宫，仿佛被气浪击到了九霄云外。
<br/>　　褚彦的一击，并没有伤到苏莫，别说对方也没下杀手，就是下了杀手，也不可能伤到苏莫。
<br/>　　今日，有褚彦在此，苏莫想要获得沁儿的帮助怕是很难，所以，他只能等待机会了。
<br/>　　“蝼蚁！”
<br/>　　见苏莫被自己随即一挥击飞，褚彦面带不屑之色，不屑的摇了摇头。
<br/>　　“我们走吧！”骆妤见此，招呼一声，率先向人王宫内走去。
<br/>　　沁儿瞥了一眼急速离去的苏莫，面带迟疑之色，她自然是听到了苏莫的传音。
<br/>　　对方，居然要等她！
<br/>　　“沁儿放心，我没有伤他！”褚彦见此，微笑着说道，他没有下狠手，那个段惊天不会受多大的伤。
<br/>　　“嗯！”沁儿点头，随即没有多言，和褚彦一起，进入了人王宫之中。

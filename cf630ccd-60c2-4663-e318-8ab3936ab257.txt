　　花费了很久的时间，刘哔从无数道具中挑选出来了两件装备。
<br/>　　而这个两件装备，对他来说也是再适合不过了。
<br/>　　第一件装备是一件通体黑色的长袍，上面刻画着一些暗红色的不知名图案，像是某种图腾，给人的感觉十分玄妙。
<br/>　　相对于其他装备来说，这件长袍的防御力并不强悍，甚至在总体数据上，都和同等价位的装备相差不少。
<br/>　　促使他选择这件装备的原因，是因为黑袍自带的一个主动技能。
<br/>　　和这件黑袍绑定的主人，可以消耗能量，任意进行一次方圆最高不超过2公里的短距离迁跃，冷却时间，三小时。
<br/>　　以刘哔现在变态的再生能力，他根本不需要考虑防御的事情。
<br/>　　反正他只要头部不受到损害，身体就会一直修复，除非是实力高出他许多的敌人，否则他很难彻底死亡。
<br/>　　如果和他交手的，真是那种实力高出他许多的敌人，那两者之间的差距，并不是一件增加防御属性的装备可以解决的。
<br/>　　所幸，他还不如牺牲一些防御数据，来换取其他方面的优势，比如黑袍上自带的迁跃技能，性价比上就高出很多。
<br/>　　更不要说接下来的计划中，这件黑袍也能派上很大的用场。
<br/>　　至于第二件装备，是一副拳套，虽然商城将它列入了防具分类，但实际上，它更像是一件武器。
<br/>　　它并没有太过于花里胡哨的东西，仅有的优势，就是那超高的攻击强度。
<br/>　　很多比这幅拳套等阶要高出许多的武器，攻击强度都不分上下，唯一的缺点，就是这幅拳套并没有什么特殊的功效罢了。
<br/>　　倒也正常，如果没有这个缺点，那它的价值自然也会比现在高出许多倍。
<br/>　　随着这两件装备被兑换出来，他账户中本来还算富裕的积分，就彻底被消耗干净了。
<br/>　　某天，分不清是上午还是下午，反正囚室内依旧漆黑一片，刘哔像往常一般，盘坐在地面上进行修炼。
<br/>　　虽然相对于吞噬技能来说，像这样修炼能带来的成果实在是太过于缓慢，但蚊子腿再小也是肉，他不会放弃任何一个变强的机会。
<br/>　　哪怕是慢一些，常年坚持下来，也会见到很大的成效。
<br/>　　就在刘哔处于深层次修炼状态的时候，囚室中突然传来一股晃动，他屁股底下的金属地面，似乎都颤抖了几下。
<br/>　　强行被人从修炼状态中打断，刘哔体内气血有些翻涌，心底莫名升腾上了些许戾气。
<br/>　　但很快的，他就平静下来了自己的心情，CCG的这座名为库克利亚的喰种收容所，都是用很特殊的金属材质进行建造的，整个建筑，都仿若是一座整体。
<br/>　　为了更好的监禁效果，身处封闭状态囚室中的犯人，几乎不会感受到外界的任何声响，而这股颤动，也只有一种可能。
<br/>　　有人袭击了监狱，而且动作不小，导致整座收容所都开始摇晃了。
<br/>　　刘哔站起身来，从空间戒指中掏出黑袍和手套，穿戴完毕，他心中估摸着，青铜树那伙人应该开始行动了。
<br/>　　没有任何犹豫，他等待的就是这次机会，待一切准备就绪之后，猛然发动了黑袍自带的迁跃技能。
<br/>　　一道华丽的金光闪过，有些类似传送的拉扯感从刘哔身上传来，但力道要比穿梭位面的时候弱小了许多。
<br/>　　过程十分短暂，大概只有一两秒的时间，他的身影出现在了囚室外面。
<br/>　　因为对这里的地形不太熟悉，他将传送点设立在了自己囚室的门口，之前有马贵将将他进行关押的时候，行走的路线他还大致记得。
<br/>　　但四周的情况和他预想的不太一样，周围很安静，没有什么人影，也没有什么声响发出，根本不像被入侵了的样子。
<br/>　　如果不是空中飘荡的淡淡白雾，他还以为自己选错了时间，不过，他出来的似乎要晚了一些。
<br/>　　对于这层白雾，他有些印象，原著中称这玩意叫做，据说会对喰种起到一定程度的限制效果。
<br/>　　不过因为他的特殊体质，任何毒素之类的东西对他都起不到什么效果，故此他也不需要担心什么。
<br/>　　怕有什么意外发生，为了保险起见，刘哔还是闭上了呼吸，反正喘不喘气对他来说也没有多大影响。
<br/>　　辨别了一下方向，刘哔顺着记忆中的路线，从一旁的金属侧门中走了出去。
<br/>　　走过一条长廊，他来到了一处环形楼梯，顺着这个楼梯，他可以去很多地方。
<br/>　　之所以选择不乘坐电梯，是因为他对这里的地形太不熟悉，想找到枭，他需要一些东西来验证现在的时间。
<br/>　　楼梯上，四处都是死去的尸体，有些是喰种，有些是这里的守卫人员，对于这些尸体，他也没有丝毫客气，顺手将其吞噬了。
<br/>　　或许是因为他现在的力量太高，虽说吸收了很多尸体，但他体内的力量并没有强大多少，更没有吞噬出什么特别的属性。
<br/>　　似乎晋入SSS阶之后，这些等阶稍低的家伙，就对他没多大作用了。
<br/>　　发现了这个现象，刘哔有些苦恼，不过他很快将注意力移到了别的地方，模糊之间，他听到了一阵微弱的打斗声音。
<br/>　　有人在战斗，这样说来，入侵还没有结束。
<br/>　　略微沉吟了一下，刘哔向声音来源的地方赶去。
<br/>　　……
<br/>　　这是一处长长的过道，目测四五十米的距离，地上横七竖八的躺了一些尸体。
<br/>　　过道的尽头，一个面色有些苍白的病娇少女，手持一柄与她外形极为不协调的紫色镰刀，蹲伏在出口，似乎在守护着什么。
<br/>　　和她遥遥相对的，是两名带着一模一样面具的双胞胎少女，虽然两名双胞胎外形和人类少女没什么区别，但红色的独眼，却表明了她们的身份。
<br/>　　“我想要更强大的对手。”病娇少女摇摇晃晃的从地上站起，抬起了头，缝着红线的嘴角露出一抹病态的笑容。
<br/>　　看到什造熟悉的脸庞，双胞胎明显一惊，目光微微呆愣了一下。
<br/>　　铃屋什造，这家伙，不知道为何跑到了这里，这在计划中，可是没有说明的事情。

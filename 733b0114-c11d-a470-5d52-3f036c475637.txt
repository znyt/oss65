　　第255章
<br/>　　戎族的王城位于草原腹地，依湖而建，那片湖就是戎族的圣湖，唤作女儿湖。出夕照关到女儿湖，大概要走八百里，中间是无边草原。
<br/>　　关克乐是跑这条路的行商，帝国的铁器、茶叶、绸缎在戎族可是抢手货，而戎族的皮毛裘革在帝国也能卖上好价钱。只不过，帝国不许和戎族经商，关克乐这种行为属于走私。
<br/>　　关克乐只是个小商人，整个商队不过十几个人。
<br/>　　“大家快点，天黑找地方歇脚，娘娘皮的，再敢偷懒，都让马贼把你们杀了。”
<br/>　　关克乐大声骂着，他赶着车，回头看了一眼，见道上行过来两个人。
<br/>　　男人牵着毛驴，女人骑在毛驴背上，那女人模样生得可是真俊，让人想看又不太敢看，再看男人模样，怎么说呢，感觉有些不太搭。
<br/>　　“兄弟，往哪儿去啊，一起走吧？”
<br/>　　草原上可能走几天都见不着一个人影，只要路上碰见，对方又不是马贼，那就不介意一起走，若有某一方缺少口粮的，给吃给喝都不是太大问题。
<br/>　　男人笑了笑，道：“不了，不太方便。”
<br/>　　“有什么不方便的，这条路上可是有马贼，你们两个人，不怕撞见马贼么？大家一起走，还是有个照应。”
<br/>　　程大雷说得不是不太方便，而是不太安全。听到这话他顿了顿，道：“你们难道不怕遇到马贼？”
<br/>　　“嘿，老哥我有刀啊。”关克乐显然是一个自来熟，立刻已经与程大雷称兄道弟起来，他道：“不瞒兄弟说，老哥我练过五年刀，平常的马贼根本不放在眼里。”
<br/>　　程大雷看了看他，下意识点点头：“蛮好，蛮好。”
<br/>　　“兄弟，咱们一起走吧，看上去你也不像坏人。”
<br/>　　关克乐正说着，忽然见男人已跃上毛驴，和女人双乘一驴，遥遥离去。
<br/>　　“唉，不想交朋友呐。”关克乐道：“不过，这头毛驴还真是快啊，一般的马都赶不上。”
<br/>　　他招呼伙计继续赶路，其实做这种买卖，是拿命还钱，真如果撞上马贼，一条命也就埋在草原了。所以大家一路上最怕碰到的，还是马贼。
<br/>　　继续向前走，约过了一个时辰，就看到草原上一队人马奔向前方。
<br/>　　“东家，马贼，是马贼！”有伙计惊慌道。
<br/>　　关克乐也看到了，却发现这帮马贼没有搭理己方，赫赫奔过。
<br/>　　“东家，是不是他们没看见咱们。”
<br/>　　“不应该吧。”关克乐也拿不准。
<br/>　　“要是看见了，怎么会放过我们？”
<br/>　　“会不会不是冲我们来的。”
<br/>　　“哪……是冲谁来的？”
<br/>　　心里七上八下，关克乐不敢再向前走了，在原地待了一个时辰，才鼓起勇气上路。
<br/>　　一路上心脏仍在砰砰打鼓，此时天色已经暗下来，西方一轮没有热度的红日还露着半个脑袋。
<br/>　　太阳快要落山了，这一幕似乎象征着什么，令关克乐心中更加恐惧。
<br/>　　正埋头走着，他忽然一愣，因为空气中传来一股血腥味。下意识抬起头往前看，眼前的景象把关克乐的魂儿都吓了出来。
<br/>　　只见绿色的草地上，倒着一具具尸体，不多不少，十三具，正是刚才过路的马贼。
<br/>　　他们死了，同时死的，脸上留着惊愕，似乎临死之前都不相信自己被杀死了。
<br/>　　更可怕的是，在距离尸体不远处，一男一女正坐着吃东西，一头毛驴在周围悠闲吃草。
<br/>　　正是那不久前还见过面的一男一女。女人依旧很俊，男人粗糙一张脸，专注咀嚼手中的肉干，看上去很平静。
<br/>　　平静得有点吓人。
<br/>　　这时候男人看见了关克乐，伸手向这边打了个招呼，竟迈步走了过来。
<br/>　　关克乐吓得后退，男人进一步，他退一步，男人再进，他退得更加厉害。
<br/>　　忽然，脚下一跌，倒坐在地上，抬起头男人已经在眼前。
<br/>　　“你想干嘛，我可是有刀的，我练过五年刀法，我很厉害的。”
<br/>　　关克乐险些哭出声来，仓皇去拔刀，这腰上的刀不知怎么回事，偏偏此刻拔不出来了。
<br/>　　“喂，有衣服么？”男人问。
<br/>　　“衣服……？
<br/>　　“男人穿的，还有女人穿的。”男人摊开双臂，关克乐才看到他一身鲜血：“没法穿了。”
<br/>　　打劫衣服的马贼，可他骑的也不是马啊，那是驴贼。马贼要钱要命，驴贼只要衣服？
<br/>　　“有，有。”
<br/>　　关克乐还真的有，这次去戎族做生意，车上还真带了一些衣服靴子，既有男的也有女人穿的。
<br/>　　“太好了，太好了。”
<br/>　　男人显得很兴奋，冲女人挥挥手：“快过来，快过来。”
<br/>　　两人开始在车上翻捡，靴子衣服都翻了出来。
<br/>　　“呀，这里还要件袍子，不错不错。”女人。
<br/>　　“东西挺全么。”男人也忍不住赞道：“还有马车呢，这马车也不错，要了要了。”
<br/>　　“来来，把你们的货搬下来，我看上的东西都给我装到车上。”
<br/>　　男人大声道，伙计们不敢不听，关克乐也是不敢不听，只能咬着牙，心里暗骂：抢吧，抢吧，你们这对贼公贼婆，活该你们做贼，你们一辈子都做贼。
<br/>　　“嗨，真多谢你了。”男人不知何时来到关克乐面前。
<br/>　　关克乐心里冷哼一声，嘴上却道：“您看上什么尽管拿，您尽管拿！”
<br/>　　“差不多够了，你看这够不够。”说着，男人从怀里掏出一颗夜明珠塞到关克乐手中。
<br/>　　关克乐有辨宝的能力，见到这颗夜明珠眼睛立刻亮了。有这一颗夜明珠，自己两三年都可以在家里歇着。
<br/>　　“够了就好，这条路最近不太平，小心些。”
<br/>　　说着，男人把毛驴挂在车上，他和女人上了车，一挥马鞭向北方去了。
<br/>　　此时关克乐还没回过神来，看着走掉的二人，再看看手中的夜明珠。
<br/>　　原来不是抢，是买。嘿，早知道自己就该多推销一些东西，可以多换一颗夜明珠。
<br/>　　“东家，咱们把货物收拾一下吧。”
<br/>　　“不要了，不要了。”关克乐挥挥手：“回去，赶紧回去。”
<br/>　　这些货物在戎族的确抢手，不过在帝国其实不值什么银子。现在自己有了这颗夜明珠，回去后，打死也不会再来了。

　　苏莫急速而行，向时空界域的方向而去，这一次去时空界域，他要利用时空之城的时间法则，再次突破境界。
<br/>　　这样的话，能大大的缩短修炼的时间。
<br/>　　苏莫的飞行速度，非常的快，如同利剑刺穿长空，响起刺耳的破空之声。
<br/>　　“不知道帝一魂有没有在找我！”苏莫一边飞行，一边心中暗想。
<br/>　　他离开人王宫这么久了，离开之时，又杀死了裘蒙，他离开人王宫的消息，肯定早已泄露了。
<br/>　　那么，帝一魂肯定派人在到处找他，只是不知道，有没有找到终极剑域？
<br/>　　“以后，还是要加倍小心！”苏莫叹了口气，在没有足够的实力之前，他只能继续蛰伏。
<br/>　　好在，他现在有天幻神丹，现在的面貌与气息，除了绫裳之外，没有人能认出他的真实身份。
<br/>　　终极剑域距离时空界域，并没有多远，以苏莫的速度，飞行了两个多月之后，便距离时空界域不远了。
<br/>　　而此时此刻，在苏莫的后方，距离他约莫一个时辰的地方，慕容雷利和许焱两人，依旧在追踪。
<br/>　　“慕容师兄，不会追错方向了吧？”
<br/>　　一边飞行，许焱一边询问慕容雷利，这都追了两个多月了，居然还没有追到秦陨。
<br/>　　他深深的怀疑，他们追错了方向，那秦陨虽然实力强大，但飞行的速度，不可能比他们快。
<br/>　　“应该不会错！”慕容雷利低声道，他精神感知力弥漫四方，能感觉到空气之中，残留着微弱的气息，就是那秦陨留下的。
<br/>　　这种气息，过的时间越久，便会越发淡薄，直到消失无踪。
<br/>　　而他能感知到这种气息，想必那秦陨，刚离去不久，最多半个时辰。
<br/>　　“那就好！”许焱点了点头，他对于慕容雷利还是比较相信的，对方精神力很强大，远超同阶之人，感知力比他强大的多，应该不会有错。
<br/>　　“继续追吧，我看看他到底要去哪里？”慕容雷利声音低沉的道，他和许焱并不擅长速度，不然也不会追这么久了。
<br/>　　不过，他还就不信了，那个秦陨会一直飞行而不停留。
<br/>　　随即，两人都不再多言，继续追击。
<br/>　　……
<br/>　　神剑山，终极神王殿，在今日迎来了一位贵客。
<br/>　　一名白衣飘飘，气质独特、神俊非凡的年轻男子，来到了神剑山的山门之前。
<br/>　　“鄙人天尘宇，求见终极神王！”
<br/>　　白衣男子的声音响起，平淡而悠远，在神剑山守门的弟子耳边响起。
<br/>　　“无上神王……天尘宇！”
<br/>　　一众守山弟子，个个瞪大了眼眸，仅仅的盯着眼前的白衣青年。
<br/>　　无上神王天尘宇，居然来到了他们神剑山，这可真是稀客啊！
<br/>　　神域有十大无上神王，天尘宇是一个、紫霄神王是一个。
<br/>　　但是，天尘宇的修为，虽然低于紫霄神王，而名气却是更在紫霄神王之上。
<br/>　　“嘘！”
<br/>　　天尘宇向众弟子，坐了个禁声的手势，微微一笑道：“莫要声张，快去通报终极神王吧！”
<br/>　　这几年，他辗转天池各大界域，搜寻苏莫的下落，并不想弄的人尽皆知。
<br/>　　“是！”
<br/>　　一名为首的弟子，点头应声，随即立刻上山通报去了。
<br/>　　弟子通报的速度很快，片刻之间便回来了。
<br/>　　“殿主有请！”
<br/>　　随后，弟子恭敬的带路，引领着天尘宇，向神剑山上走去。
<br/>　　不多时，天尘宇便来到了神剑山的山巅，走进了终极大殿之中。
<br/>　　大殿中，主座之上，端坐着身穿明黄色长袍，身材矮壮的老者。
<br/>　　老者下巴上留着短短的胡茬，每一根都犹如钢针一般，此人正是终极神王殿殿主，终极神王。
<br/>　　“哈哈，无上神王天尘宇，驾临鄙殿，有失远迎！”终极神王面带爽朗的笑容，若是一般的低级神王求见他，他可不会这么客套。
<br/>　　“呵呵，殿主过奖了，晚辈不请自来，还望恕罪！”天尘宇微微一笑，便向终极神王抱了抱拳。
<br/>　　“哪里哪里？”
<br/>　　终极神王摆了摆手，笑着道：“无事不登三宝殿，不知阁下此来何事？”
<br/>　　终极神王心中疑惑，他们终极神王殿，和天王宫可没有什么交集，对方此来必有目的。
<br/>　　“既然殿主爽快，那晚辈也就直说了！”天尘宇笑着道。
<br/>　　终极神王点了点头，静等天尘宇开口。
<br/>　　天尘宇略一沉吟，凝视着终极神王，沉声道：“晚辈此来，是想要劳烦殿主帮忙找一个人！”
<br/>　　“找一个人？”终极神王闻言一怔，对方居然让他找人。
<br/>　　“前辈，此人对晚辈来说，至关重要，如果终极神王殿能帮助晚辈找到此人，晚辈可以答应任何要求！”天尘宇郑重的说道。
<br/>　　“任何要求？”终极神王闻言惊讶，这个诺言倒是让他惊奇，什么人值得对方如此重视？
<br/>　　“不错，任何要求，只要晚辈能做到，就绝不推辞！”天尘宇重重的点了点头。
<br/>　　“何人？”终极神王好奇的问道。
<br/>　　“此人名为苏莫，修为目前应该是二品，修混沌之力，拥有万化神剑和不败战气元神……！”
<br/>　　随即，天尘宇将苏莫的大致信息，与个人特点，一一告知了终极神王。
<br/>　　“此人很可能改头换面，改名换姓，所以，展露的容貌和气息，很可能都是假的。”天尘宇又补充道。
<br/>　　“此人在终极剑域？”终极神王又问道，既然对方来这里，那么说明这个苏莫就在终极剑域。
<br/>　　“不一定，此人行踪飘忽不定，在任何地方都有可能！”天尘宇摇了摇头。
<br/>　　“天尘宇，既然此人如此弱小，难道衍算不出其行踪？”终极神王问道，在神域很多强者，都懂得衍算之道。
<br/>　　对这种弱小之人，只要是神王级别的强者，很轻易就能找到此人。
<br/>　　“无法衍算，此人有特殊手段，能瞒过一切衍算之法！”天尘宇摇了摇头，其实，他心中清楚，苏莫因为是轮回之身，所以才无法被衍算出行踪。
<br/>　　若是苏莫只是虚神之境，他们或许还能衍算出一二，但是，到了真神之境的轮回之身，就算是至强者，都不一定能衍算到其行踪。
<br/>　　“哦？”终极神王闻言，眼眸微微一眯，他是什么人物，自然能猜到，这所谓的苏莫不简单。
<br/>　　不然，怎会劳动天尘宇亲自寻找，还愿意付出一切代价。
<br/>　　“你放心，此事，本座会吩咐下去，若是真的找到了此人，再通知于你！”终极神王点了点头，不管找与不找，口头上自然要答应下来。
<br/>　　“多谢殿主，此事还需保密，有劳了！”
<br/>　　天尘宇向终极神王抱拳，便提出了告辞：“前辈，晚辈还要去其它界域，就不多留了，替我向紫霄兄问好！”
<br/>　　“嗯，不送！”终极神王点了点头，他知道对方和紫霄关系不错。
<br/>　　毕竟，两人同为无上神王，自然有些交流。
<br/>　　只是，对方让此事保密，那就有些耐人寻味了。
<br/>　　找一个普通的二品修为之人，还需要保密吗？此人是何等身份？能让天尘宇如此郑重？
<br/>　　“告辞了！”天尘宇微微抱拳，便欲转身离开。
<br/>　　这时，终极神王略一沉吟，又道：“且慢，不知你天王宫的那位，何时能晋阶高级？”
<br/>　　“快了，具体只有他自己知晓！”天尘宇含糊的道，他知道对方问的是帝一魂何时能达到高级神王？
<br/>　　现如今的神域，虽然表面风平浪静，但各大势力的强者，都在等待着。
<br/>　　等待这帝一魂晋阶高级神王，或者说也在等待着拓跋青云，何时能达到高级神王。
<br/>　　谁若是率先突破，那么，大局就定下了，到时候威临神域，无人可挡。
<br/>　　“嗯！”终极神王点了点头，对于帝一魂和拓跋青云两人，很多势力的强者，也都在关注着。
<br/>　　毕竟，这两人的天赋太强了，远超无上神王，任何一人，都有成为至强者的潜力。
<br/>　　当然，最终分出胜负之后，能不能超越前世的轮回天尊，就没有人能知晓了。
<br/>　　对于两人的争斗，所有势力都没有插手，也不敢贸然插手。
<br/>　　随即，天尘宇走出了终极大殿，离开了神剑山。
<br/>　　离开了神剑山的天尘宇，又向另一个界域而去。
<br/>　　只有广撒网，才能尽快找到苏莫。
<br/>　　今日的一幕，已经发生了很多次，他在很多个界域，都是如此。
<br/>　　当然，为了消息不传到地王宫，遭到有心人的注意，他所去的大势力，俱是与天王宫交好的实力。
<br/>　　他在编制一张大网，一张网罗苏莫的大网，但是，最终能不能找到苏莫，连他自己都不清楚。
<br/>　　毕竟，他若是苏莫，肯定躲藏了起来，不轻易的抛头露面。

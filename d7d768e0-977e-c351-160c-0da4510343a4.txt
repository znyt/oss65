　　鲁初雪的心脏开始激烈跳动起来，她无法想象下一秒这个慌张的汇报人员，会说出怎样的噩耗，更无法推测在难得平静的这些天里，又会突发什么状况。
<br/>　　深吸一口气，鲁初雪做好接受一切噩耗的准备，沉静开口：“说。”
<br/>　　那汇报人员也在这短暂的沉默里深吸好几口气，道：“东北部发生了一场大战，规模很大，我们没有提前得到消息，察觉到的时候战争已经快要结束。”
<br/>　　“东北？”鲁初雪愣住：“那里不是无战地带么？难道异种还是丧尸绕去了那里的海域，对人类基地发起了进攻？”
<br/>　　“不是！”汇报人员连忙摇头，焦急道：“参战者没有丧尸或异种。”
<br/>　　“那是谁？内战？”鲁初雪更加迷惑，全华夏的战力几乎都在南方边境抵御尸潮，根本爆发不了内战才对。
<br/>　　“不能确定。”汇报人员的神色带着复杂和古怪之色，道：“但经过情报分析，似乎是神秘家族之间展开了大战。”
<br/>　　“神秘家族？！”鲁初雪的声音猛然提高了一个八度，不可置信的霍然而起。
<br/>　　神秘家族一向是所有人心中最畏惧的存在，比异种王或丧尸王更加可怕，这种畏惧让除却楚涵那个天不怕地不怕的人之外，根本没有其他基地有胆量去正面挑战，也更别说与之开战。
<br/>　　而四大神秘家族在普通人心中更是一个整体，他们内部竟然爆发了战争，让鲁初雪有些难以置信。
<br/>　　“是，貌似是高家和罗家之间。”汇报之人继续道：“但是也有其他势力参与其中。”
<br/>　　“其他势力？还有什么势力？狼牙战团？不可能，他们压根不在东北地带！”鲁初雪越发的一头雾水，她忽然发现这世上有很多东西都超脱了她的预料，也完全不在正常理论的范围内。
<br/>　　“这……我不是太敢说啊！而且也不确定！”汇报之人满头大汗，最终心一横，道：“是牧司令和洛老，还有部分猎者联盟的人。”
<br/>　　鲁初雪双眼瞪得老大，表情呆傻的坐在原位，一种充满迟疑和古怪的气氛在屋内蔓延，让这间不大的空间安静了良久。
<br/>　　也不知过了多长时间，鲁初雪终于缓了过来，问道：“这件事上将知道了么？”
<br/>　　“已经派人前去汇报。”那人立即回复道。
<br/>　　“文奇胜将军呢？”鲁初雪又问。
<br/>　　“还没有通知，这件事只是我们段氏基地内部的情报系统发现的，事关两位老者，我也不知道该不该说。”汇报人员神色迷茫。
<br/>　　鲁初雪嘴角抽了抽：“那场战役的情报拿到没有？牧司令他们是渔翁得利还是跟一方合作？”
<br/>　　汇报人员摇头：“还没有拿到手，但应该快了，那边随时有人在跟进，我这里拿到消息的第一时间就过来通知了。”
<br/>　　鲁初雪冷静了下来，点头道：“有消息立即汇报，段上将那里也别落下，哦对了，给狼牙情报部也送去一份。”
<br/>　　“他们还需要我们的情报？”汇报人员下意识的脱口而出，表情古怪道：“那帮人肯定比我们知道的早，也知道的更全面。”
<br/>　　“试探一下。”鲁初雪瞥了他一眼，神色隐晦：“正是因为知道狼牙情报部一定知道的比我们多，所以我们才需要示好，以获得更多的信息。”
<br/>　　汇报人员呆了一秒，而后立即点头：“我立即去办。”
<br/>　　在鲁初雪这里接到这则消息的同一时间，待在狼牙基地守着楚涵和叶永阳的陆玟丞，也同样拿到了有关这次大战的消息，并且也正如鲁初雪所猜测的那般，陆玟丞拿到的这则情报，涵盖了更全面的情况，也有更多不为人知的细节在内。
<br/>　　因为这次事件的情报来源有两处，一是常规往狼牙送消息的暗部，二是来自参与了这次事件的人员之一猎王。
<br/>　　对于猎者联盟在这段时间里的沉寂，很多基地都报着围观的态度，外加南方边境的战事为重点，于是众人也渐渐将重心放在了其他方便，而从忽略了猎者联盟的动向。
<br/>　　而让人没有想到的是，在所有人都以为猎者联盟会就此一蹶不振，亦或是低调不出面的时候，猎王却已经与牧司令私下达成了协议。
<br/>　　原因很简单，那就是有关上一次猎者联盟被针对的事件，两位老者怀疑始作俑者是神秘家族的白家。
<br/>　　猎王虽然有另外的想法，但在牧司令和洛冥找上门的时候，却推波助澜的参与了行动。
<br/>　　人类势力首次正面与神秘家族开战，看上去像是以卵击石，但猎王之所以会答应参与，却是因为知道了某个秘密。
<br/>　　“叶老，过来看看。”陆玟丞拿着这份情报，放在了叶永阳的面前。
<br/>　　这些天在狼牙基地渐渐恢复健康的叶永阳神色好了很多，点头看了眼那份情报后，便不由震惊：“牙？这又是什么组织？”
<br/>　　一旁的顾凉晨不由心惊：“我知道，这是狼牙的前身，文明时代就成立的一支预备战队，全部都有女子组成，现在更名为‘牙’？这是什么情况？貌似也和以前不一样了啊！”
<br/>　　“这是高家出资，两位老者培养的一批普通人战队。”陆玟丞同样心惊，道：“这次参与者中有我们的人在，所以情报中有很多不为人知的秘密，洛老在文明时代培养了十几名女子组成一支预备战队，那只是一个明面上的幌子，实际上培养的是隐藏在这支预备战队之下的另一支队伍，取名为‘牙’，与龙牙、虎牙或未成形的狼牙都不同，这支队伍从一开始就是冲着神秘家族去的。”
<br/>　　“而高家，更是不得了。”陆玟丞说着，有些心惊肉跳：“他们也是胆子够大，提供了一批特殊金属武器，可以让神秘家族的血脉者行动受限。”
<br/>　　“可普通人依旧杀不死神秘家族的成员才是。”叶永阳提出了这一观点。
<br/>　　“所以动手给予致命一击的都是高家人，因为高家的人数不多，所以需要普通人的战队协助。”陆玟丞说着，表情复杂：“这次罗家死了不少人，高家也死了不少人，真的死了不少人……”

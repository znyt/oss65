<br/>        “傅总，您手背上的伤口，还是处理一下吧！”
<br/>
<br/>        落以柔将创可贴轻轻地放在了傅夜沉手旁的桌面上。

<br/>        傅夜沉握着钢笔的手停滞了一下，目光落向自己手背上的那两排齿印的时候，脑海里回想起今早的画面，嘴角情不自禁地微微上扬。

<br/>        这事也都怪他。
<br/>
<br/>        天才蒙蒙亮的时候，他一睁开眼睛，就看到一幅活色生香的画，充斥着十足的诱惑，直勾勾地撞进他的眼底。

<br/>        筱颖恬静地侧躺在他身边，身上白色睡裙的衣襟半敞，香肩圆润，酥胸半露，丝质被单也被她一脚踢开了，裙摆下是连着蜜桃臀的那双美腿修长而匀称。
<br/>
<br/>        本来已经休憩的细胞，瞬间被她的美色唤醒。

<br/>        他抑制不住体内的那股冲动，缓缓地伸出食指，沿着吸引他视线的路径，从她的后颈脊柱，轻轻地描绘到尾椎，一路蜿蜒向下。
<br/>
<br/>        所以……

<br/>        他没忍住……
<br/>
<br/>        趁着她在熟睡的时候，偷袭了她。

<br/>        而且，每一次下口都很重。

<br/>        思绪到这里，傅夜沉束手无策地抚了抚额头，这事还真有些伤脑筋。

<br/>        其实，那时候傅昼景说筱颖的那张膜是补的，他压根就不在意。
<br/>
<br/>        因为是不是补的，他都已经尝过了，一来无所谓，二来他心里也清楚。

<br/>        只是，他发觉自己，对筱颖的那种欲望，似乎得节制一下了。

<br/>        刘婶进屋后，将窗帘给拉开了，耀眼的太阳光瞬间透过落地窗，洒得室内一地的金光闪闪，童筱颖此刻在床上滚了好几圈，脸上仍旧带着浓浓的睡意。
<br/>
<br/>        唔……
<br/>
<br/>        好刺眼……
<br/>
<br/>        不想起床……
<br/>
<br/>        她举起纤细的手腕想遮住太阳光，可是才稍微移动手臂，每根骨头都在跟她抗议，全身酸痛。
<br/>
<br/>        一觉醒来，骨头就像散了一样……
<br/>
<br/>        “大少奶奶，大少爷出门前特意吩咐过了。说，午饭前让我一定要把您叫起床。已经错过了早饭，不能再错过午饭。”刘婶一脸和蔼可亲地说道。

<br/>        童筱颖这才从床上爬起来，待神志逐渐清醒后，昨晚和今早的记忆如浪潮般汹涌而至，全回到她的脑海里。
<br/>
<br/>        一巴掌拍在自己的额头上，她觉得，大概或许……

<br/>        童筱颖觉得，不能让大哥傅夜沉把自己宠成“智障”的女人。

<br/>        所以，她得去上班！

<br/>        多做一些脑力活，这样才能变得聪明点，以防再次被大哥给反勾引。

<br/>        吃过午饭后，童筱颖拿着工作证，去了傅氏集团总部的设计部，还专门让部长孙莉莉给她安排了一张办公桌，方便她天天来这里坐班工作。

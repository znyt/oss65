<br/>    一线天峡谷。
<br/>    
<br/>        仙魔诸王汇聚在峡谷的两侧，数月以来的厮杀，如今终于是到了决战的时刻，诸王们看向在峡谷之上对坐的凌仙和李素衣身上。
<br/>    
<br/>        这一场燕山师兄弟之间的战斗，同样会决定这一场战争的胜负。
<br/>    
<br/>        “不知道这一场战斗到底谁胜谁负！”
<br/>    
<br/>        “同出一门，却是一个魔修，一个正道，还真是诡异啊。”
<br/>    
<br/>        “不知道燕山之主还收不收徒，我虽然年纪大了点，但是当个记名弟子应该还是没问题吧？”
<br/>    
<br/>        李素衣和凌仙最近在一线天附近已经闯出了偌大的名头，颇有诸王之中所向无敌的气势。
<br/>    
<br/>        两人可谓是都是一人成军的恐怖存在，李素衣身上环绕一只青色饕餮，吞噬周围的灵气，瞬间炼化为法力，凌仙一身道骨，法力源源不绝！
<br/>    
<br/>        本来实力就在诸王之巅，加上法力几乎用之不尽，让这两人成为更加可怕的存在。
<br/>    
<br/>        而两人的恐怖实力，也是让众人对于陈明越发感到敬畏，一人门下，教出如此绝代双骄，该是对于大道的领悟到了何种境界？
<br/>    
<br/>        加上陈明最近行走于人间，天劫不断，对于陈明的实力，更是高山仰止。
<br/>    
<br/>        无论是魔宗还是古仙门，都从来没有想到过，这一场诸王之战，最后的胜负居然落在了燕山的这一对师兄弟的身上。
<br/>    
<br/>        李素衣和凌仙对坐，都没有说话，原因可能是凌仙知道李素衣在战斗时不会说话，所以也没有开口。
<br/>    
<br/>        两人对坐良久，终于是起身，李素衣伸手掀开背后的血棺,血海魔刀出鞘，血气斑驳了一片。
<br/>    
<br/>        凌仙手握神木印，先天大道之气弥漫，环绕身躯。
<br/>    
<br/>        “凌仙和哑魔要动手了！”
<br/>    
<br/>        “这一战必定流传千古！”
<br/>    
<br/>        “今生能够见到绝代双骄一战，无悔修行！”
<br/>    
<br/>        万众瞩目之下，李素衣凌空便是一刀直接劈来，凌仙头悬神木印，双手施展搏仙术！
<br/>    
<br/>        轰！
<br/>    
<br/>        恶蛟与穷奇起舞，刀光弥漫一片，两人在一线天之前，骤然出手！
<br/>    
<br/>        众人分明可以看得到，这一次，虽然凌仙处在劣势，但是并没有如同上一次一般一招落败，两人交手，看得周围的诸王大气都不敢出。
<br/>    
<br/>        李素衣血海魔刀施展的出神入化，凌仙双拳搏仙术无边大势沟通整个天地，李素衣的身上血气弥漫不散，凌仙的身上，先天大道之气环绕。
<br/>    
<br/>        两人如同神子一般在空中交战，轰鸣之声不绝于耳。
<br/>    
<br/>        凌仙和李素衣都对对方知根知底，所以上来就是大杀招，根本没有半点藏私。
<br/>    
<br/>        穷奇嘶吼飞天，恶蛟盘旋裂地，刀光横纵数十丈，大地在两人的战斗下变得千疮百孔，天空中的浮云，也在这一刻全部被打散。
<br/>    
<br/>        李素衣挥刀落入血棺之中，血海魔刀在血棺之中搅动，引出一道道血浪，他的周围，瞬间变成一片血海。
<br/>    
<br/>        凌仙沟通神木印，法力不断地灌注其中，先天大道之气涌出，如同行走在人间的神子，浑身青光璀璨，青光触碰到血海，将血海隔绝在外。
<br/>    
<br/>        两人在顷刻之间交手十次，而后没有半分停息，又是朝着对方冲去，搏仙术横空，镇裂苍穹，刀光贯穿，将一座座山峰轰碎。
<br/>    
<br/>        两人在不断地交手，更为可怕的是，周围的诸王已经看了一天了，两人竟然还是龙精虎猛，各种术法不断打出！
<br/>    
<br/>        “他们修行的功法，当真是强大的可怕！”
<br/>    
<br/>        “哼哼哼，那是当然，凌仙可是修行的整个大陆之上最难修行的道经，一旦到了悟道境，就一飞冲天了！”
<br/>    
<br/>        “哑魔大人乃是修行的燕山之主传下的白骨饕餮，乃是大陆上最邪恶的一门功法！”
<br/>    
<br/>        夜里，战斗依旧没有任何停息，两人依旧是在疯狂对战，但是诸王们明显可以看得出，凌仙已经是落于下风，“凌仙毕竟还只是神府境，若是到了法相境，凝聚出了自己的法相，哑魔必定不是对手！”
<br/>    
<br/>        “哼，哑魔大人的法相乃是饕餮，岂是一般的法相可以匹敌的！”
<br/>    
<br/>        “不过是欺负凌仙还没有凝聚法相而已，猖狂什么！”
<br/>    
<br/>        “哑魔大人，才是真正的诸王之首！”
<br/>    
<br/>        空中的搏杀仍旧在继续，又是两天过去，凌仙已经浑身浴血，不过他咬着牙，依旧想要一战，毕竟差了一个境界，他还是不是李素衣的对手。
<br/>    
<br/>        白骨饕餮太邪门了，修为提升的太快了。
<br/>    
<br/>        李素衣的身旁，血海三千，他的身子上，一只青色的巨兽盘旋，胸口有着一张巨口，那是饕餮，也是他的法相。
<br/>    
<br/>        二师兄，能够和你这样畅快的大战三天三夜，还真是我的荣幸。
<br/>    
<br/>        李素衣一刀劈飞凌仙，朝着仰倒在地再也爬不起来的凌仙走去，他横刀，步履坚定。
<br/>    
<br/>        诸王们终于是看到了这一场对战的结果，“哑魔赢了！”
<br/>    
<br/>        魔王们大喝道:“杀了他，杀了他！哑魔才是真正的诸王之首！”
<br/>    
<br/>        人王们自然不可能放任李素衣杀了凌仙，众多人王齐齐涌出，“一定要救走凌仙，没有了凌仙，将再也没有人能够抵挡哑魔！”
<br/>    
<br/>        “凌仙若死凝聚法相，哑魔绝对不是凌仙的对手！”
<br/>    
<br/>        见到众多人王齐齐朝着凌仙飞去，想要保护凌仙，魔王们也是齐齐涌出，飞上空中，去阻挡人王。
<br/>    
<br/>        数万诸王，瞬间战作一团，无数的术法之中，李素衣如同闲庭散步，朝着凌仙走去，没有任何人能够阻挡他的脚步，因为他是诸王之首。
<br/>    
<br/>        李素衣随意地挥刀，将三个阻挡他前进的修行者劈飞，不过刚刚走到凌仙的面前，他深深地看了凌仙很久，凌仙透过李素衣血红色双眼，可以看得到李素衣的神情正在剧烈的变化，“我本来只是想和你一战的，二师兄，我并不想杀你，可是，魔念横生，苦海无边啊！”
<br/>    
<br/>        “我已经，回不了头了！”
<br/>    
<br/>        李素衣举刀，忽然感觉到一阵心血来潮，剧烈的危险感应然让他瞬间后退数十步。
<br/>    
<br/>        天空之中，一柄巨剑落下，正好落在李素衣之前的位置上，将整个战场都震了震，所有人看向那一柄巨剑，在巨剑的剑柄上，站着一个身穿红衣的女孩，那个女孩，脸上带着一个滑稽面具，而她的脚下那柄剑，叫做巨阙。
<br/>    
<br/>        一时之间，诸王们都停手，四野无声寂静，只有卓青瑶的声音传荡在整个战场之上，“还真的有很多王级在这儿啊，看来我没有来错地方！”

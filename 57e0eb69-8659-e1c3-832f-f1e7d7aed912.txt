        &lt;h3&gt;第三卷第二百零六章  宁家&lt;/h3&gt;
<br/>
<br/>        凌辰和宁雅见面了，但也是匆匆而来，匆匆而往，他在宁家露了个脸，宁雅的父亲似乎对他的感觉还不错。
<br/>
<br/>        在宁家吃了晚饭，几个年轻人在一起谈天说地，这里面年龄大一些的就是陈子昂，年龄最小的是刘诚之和董亚子。
<br/>
<br/>        事业最成功的也是陈子昂，宁雅的父亲也是陪着几个年轻人谈天说地从做生意到聊现状，再到每个人看了些什么书。
<br/>
<br/>        宁雅从来都不知道她的父亲会这么健谈，和他们完全是溶于到一块了，而且和陈子昂聊得很投机。
<br/>
<br/>        就着生意上巴拉巴拉了很多问题。
<br/>
<br/>        “子昂啊，我们两家公司其实应该有同曲异妙的感觉，我们做的是设备，你们做的是软件，不知道后期有没有合作的机会。”
<br/>
<br/>        “其实，我们算是集成商，在软件开发的同时也需要很多硬件厂家的支持，之前有接触过伯伯您们公司的产品，不过是在您的代理商手中拿的货。”
<br/>
<br/>        “你们硬件的需求值大概率是多少？”
<br/>
<br/>        “综合全国各地区的硬件设备，只算您这边的模块的话，大概能上十位数。”
<br/>
<br/>        “这么说如果你用我家的产品我会有亿的销量？”
<br/>
<br/>        “不全是您家，还有别家的。”陈子昂实话实说，谁能将市场一把垄断？
<br/>
<br/>        “哈哈哈，我都不知道我的身边还隐藏着这样的一个大客户。”宁雅的父亲确实很心疼，做生意的人谁愿意丢掉客户？况且他女儿还在这个公司上班。
<br/>
<br/>        “叔叔，想不想和我合作一把？我今年要去甸城，那边需求量也是很大的。”陈子昂也笑着说，从生意人的角度出发，她和宁家达成合作伙伴，中间就少了很多环节，利润点相对会提高很多。
<br/>
<br/>        从私人角度出发，宁雅在她公司干了几年，而且俩个人的私人关系也很好，和宁家合作也是给宁雅一份面子。
<br/>
<br/>        “哈哈哈哈，想啊，当然想啊，和你们年轻人一起做事情干脆爽快。”宁雅的爸爸哈哈大笑。
<br/>
<br/>        “那，后期我安排人来您公司洽谈合作意向？”陈子昂也很开心。
<br/>
<br/>        “好，上班你就安排人过来吧，一切都好说！”签订一个长期的合作伙伴对双方都有利。
<br/>
<br/>        凌辰、宁雅和刘诚之简直就是惊呆了，还有如此操作？
<br/>
<br/>        董亚子眼冒星星的看着陈子昂，难怪被安教授拿来做教材，这做生意的本领是无孔不入啊。
<br/>
<br/>        凌辰虽然听着陈子昂和宁父的聊天，但是他的心思完全都在宁雅的身上，他觉得他有很多话想给宁雅说，但是碍于大家都在，又无法说出口。
<br/>
<br/>        “凌辰哥哥，我还没有去过山西，山西好吗？”宁玥坐在凌辰的旁边低声问凌辰。
<br/>
<br/>        “怎么说呢，你喜欢了他就好，你不喜欢了他就不好。”凌辰回答着宁玥眼睛却看着宁雅。
<br/>
<br/>        “我倒是对山西心有所往，有机会凌辰哥哥带我去看看如何？”宁玥语气轻柔，似乎真对山西情有所钟。
<br/>
<br/>        “当然可以啊，你想什么时候去都行。”宁雅的妹妹想去山西，他凌辰能不好好招待？只要的关于宁雅一切的东西都是好的，都会牵动凌辰的心。
<br/>
<br/>        “那太好了，凌辰哥哥这一次回去就带着我去看看如何？”宁玥纯真而默默含情的看着凌辰笑着问。
<br/>
<br/>        “好啊，刚好诚之和亚子也会去，大家一起。”凌辰并没有拒绝宁玥，董亚子作为厂家这一次是会到山西的，刘诚之目前还是公司的员工，肯定是要回去上班的，既然宁玥想去那就邀请她过去，如果宁雅能去就更好了。
<br/>
<br/>        “爸爸，我过几天和凌辰哥哥一起去山西。”宁玥打断了宁父和陈子昂的谈话，宁雅和陈子昂都看向宁玥。
<br/>
<br/>        “你去山西干什么？”宁父皱着眉头，这个小女儿啊，总是喜欢自作主张。
<br/>
<br/>        “爸爸，我想去看看凌辰哥哥的家呢。”宁玥有点撒娇的看着宁父。
<br/>
<br/>        “姐姐，我们一起去吧，我和凌辰哥哥都说好了。”宁玥又看向宁雅。
<br/>
<br/>        “我快要上班了，事情还很多，要去你自己去吧。”宁雅看着宁玥，似乎她对凌辰的感觉不一样？
<br/>
<br/>        “你不去爸爸是不让我去的。”宁玥的语气都有些央求了。
<br/>
<br/>        “爸咋可能不让你去呢，走亲访友的不是很正常吗？”宁雅有些无语了，你和凌辰是什么关系啊？
<br/>
<br/>        “爸。”宁玥又看向她的父亲。
<br/>
<br/>        “去吧，诚之是不是也要去山西？”宁父转过话语问刘诚之。
<br/>
<br/>        “我们明天都会到山西，我上班，亚子是去视察。”刘诚之原本就是想着将凌辰介绍给宁家，宁父对凌辰的印象还不错，如今已经达到目的了，后面追妻之路怎么样就靠他凌辰自己努力了。
<br/>
<br/>        “那就让宁玥随着你们一起过去玩几天，这孩子天性贪玩，你们到时候多担待点。”
<br/>
<br/>        “没问题，我保证将宁玥照顾好。”刘诚之微微一笑。
<br/>
<br/>        陈子昂看了看宁雅，又看看凌辰，宁雅一脸淡定，凌辰却是看着宁雅一脸期待，而宁玥却是满脸兴奋。
<br/>
<br/>        再看看宁父似乎有点尴尬的样子，而刘诚之和董亚子是一副无所谓的神情。
<br/>
<br/>        “哈哈哈，现在是你们年轻人的时间。”宁父抬起手看了看手中的腕表，时间已经不早了，他到了需要休息的时候。“我就不掺和你们年轻人的时间了，子昂啊，你阿姨已经将你住的地方给你收拾好了。”
<br/>
<br/>        “诚之，你们也留下来吧。”宁父对刘诚之和陈子昂说。
<br/>
<br/>        “不了，我们离得这么近，我就回家了，还得收拾收拾，明天出发。”刘诚之也是看着时间不早，就提出告辞。
<br/>
<br/>        他得开车将董亚子和凌辰送到酒店，然后回家。
<br/>
<br/>        陈子昂是宁雅邀请来的客人，住在宁家情有可原，如果他们住在宁家就不合适了。
<br/>
<br/>        凌辰见到了宁雅，满腔的热血沸腾，却连话都没有说上几句，不过他已经开心了，他已经向宁雅靠近了一步，等宁雅回北京他就去向宁雅表白。
<br/>
<br/>        走出宁家大门，风，有些微凉，南方的冷和北方的冷不是一样的，南方的冷是那种刺骨的冰凉凉，似乎让人无处躲藏。
<br/>
<br/>        凌辰：走过你来时的路，就当一路相伴。吹你吹过的风，就当拥你入怀。看你见过的风景，就算牵手而行。我从来没有奢求什么，我只是感动了自己，谈了一场你不在的恋爱，现在我想牵着你的手，走过风走过云，走过春夏秋冬，走过未来的每一程！
<br/>
<br/>        在恰好的时间，我恰好碰见你，我多想陪你一起生长，陪你一起走过山高水长，我多想，在你的生命里留下阳光，照亮我们彼此前进的方向。

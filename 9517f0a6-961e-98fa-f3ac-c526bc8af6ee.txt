　　整个青俞皇城，寂静无声。
<br/>　　众人凝望着不停道歉，并吓的脸色苍白的三长老，心中掀起惊涛骇浪。
<br/>　　堂堂的武皇境九重武者，北海冰宫三长老，居然如同一个犯了错的孩童一般，哪里还有半点超级强者的风范！
<br/>　　青谷和青雅等一众青俞皇室之人，心中的震撼最为强烈。
<br/>　　北海冰宫是北海地域的霸主，三长老是北海冰宫的掌权者之一。
<br/>　　青俞帝国是依附于北海冰宫的势力，他们自然清楚三长老的威严，三长老的强大。
<br/>　　现在他们亲眼看到，以往威严无比、高高在上、只能让人仰望的三长老，满脸的惊恐，大声的认错。
<br/>　　一众青俞皇室之人，心中的震撼可想而知。
<br/>　　目光望向苏莫，众人只感觉有些恍惚，这个传闻中的妖孽人物，威慑力真是恐怖！
<br/>　　都不用出手，便能让超级强者低下高贵的头颅！
<br/>　　“现在知道错了？”苏莫见此嗤笑一声，暂停了出手，因为他有了新的想法。
<br/>　　“错了！我真的知道错了！”三长老闻言立刻点头，同时他心中亦是大松了一口气，苏莫能给他认错的机会便好。
<br/>　　辛亏他赌对了，没有选择逃走！
<br/>　　“哪里错了？”苏莫再次问道。
<br/>　　“我……我一时被仇恨蒙蔽了双眼，不该辱骂苏宫主！”三长老低下了头颅，低声说道。
<br/>　　他感觉脸上火.辣辣的疼痛，当着无数人的面，向别人认错道歉，这真的是丢人丢到姥姥家了，他恨不得找个地缝钻进去。
<br/>　　但他却不得不为，虽然面子很重要，但与性命相比显然不值一提。
<br/>　　“对于辱骂我的人，下场只有一个！”苏莫面色淡漠，眸中寒光闪动。
<br/>　　三长老闻言顿时心中一颤，心中悔的肠子都青了，暗骂自己嘴贱。
<br/>　　“苏宫主，我罪大恶极，不求您宽恕，只求您能饶我一命！”三长老再次向苏莫抱拳，深深的一拜。
<br/>　　苏莫见此，微微一笑，点头道：“我可以饶你一命！”
<br/>　　“多谢苏宫主！”三长老闻言顿时大喜，急忙向苏莫道谢，他的心中暗暗的松了口气。
<br/>　　“不过，死罪可免活罪难逃！”
<br/>　　苏莫摇了摇头，继续道：“一个月后，你和你北海冰宫的宫主，一起到苍穹神宫，负荆请罪吧！”
<br/>　　听闻苏莫之言，三长老再次心中一颤，随即面上布满了苦涩。
<br/>　　此事，本只是他的事情，现在居然要把他们北海冰宫的宫主，也牵连在内了！
<br/>　　待他回到宫中，宫主知道了此事，定然会雷霆震怒吧！
<br/>　　不过，苏莫既然开口了，他也不敢反抗。
<br/>　　“是！苏宫主！”三长老立刻点头。
<br/>　　“滚吧！”苏莫向对方挥了挥手，便不再理会。
<br/>　　“是！”三长老应声，随即一刻也不愿意多待，身形立刻冲天而起，急速远去。
<br/>　　慕冰雪和其他十数名北海冰宫的强者见此，深深的看了苏莫一眼，便也立刻跟随着三长老而去。
<br/>　　转瞬之间，所有北海冰宫之人，便全部离去。
<br/>　　“现在可以回答我的问题了吧？”苏莫目光望着手中的屈覃国师，眸中杀机森然，冷冷的问道。
<br/>　　“苏……苏宫主，我真的不知道你在说什么？我怎么可能知道黄泉魔宗的人在哪里？”屈覃沉声说道。
<br/>　　“我会让你说的！”苏莫见对方不承认，顿时冷笑一声，随即直接便将对方收进了虚界神图之中。
<br/>　　他要将此人带走，好好的逼问一番。
<br/>　　随即，苏莫目光望向大殿中一众皇室之人，他不确定青俞皇室之人，和黄泉魔宗有没有关系？但想来应该不会。
<br/>　　虽然屈覃是国师，但并不是皇室之人，皇室应该不知道他的真实身份。
<br/>　　当然，这也不是绝对的事情。
<br/>　　而后，苏莫没有理会青俞皇室之人，转身踏空离去。
<br/>　　一众皇室的强者，望着苏莫渐渐远去的背影，久久无声。
<br/>　　什么样的人为强者？什么样的气概为无敌？
<br/>　　这才是真正的强者！这才是真正的无敌！
<br/>　　俯瞰天下，众生臣服！
<br/>　　强如北海冰宫，都只能低头！
<br/>　　至于苏莫擒拿了屈覃，青俞皇室的人根本不敢说什么！
<br/>　　青雅公主见到苏莫即将消失在视线中，立刻咬了咬银牙，仿佛做出了某种决定。
<br/>　　而后，她身形一闪，如同一只青色的蝴蝶，急速向苏莫追了过去。
<br/>　　“青雅！”
<br/>　　“公主！”
<br/>　　青谷等人见此，顿时心中一惊，青谷身形一闪，立刻向自己的女儿追了过去。
<br/>　　“嗯？”
<br/>　　苏莫踏空而行，但刚走出十几步，便停了下来，转头看向身后追来的青雅公主。
<br/>　　唰！
<br/>　　瞬息之间，青雅公主便裹挟着一阵香风，来到了苏莫的身前。
<br/>　　“何事？”苏莫疑惑的出声问道。
<br/>　　“苏……苏宫主，我能不能加入你的苍穹神宫？”青雅深深的吸了口气，高.耸的胸.脯一阵颤动，满脸忐忑的问道。
<br/>　　她很担心苏莫拒绝，不过她还是鼓起勇气前来相问。
<br/>　　苏莫闻言眉头微皱，打量了青雅一番，随即点了点头，轻声道：“可以！”
<br/>　　此女的天赋还是非常不错的，双十年华，武王境三重的修为。
<br/>　　这还只是在青俞岛这种小地方，若是此女身处那些超级势力，得到重点培养，进入万尊榜前几百名，轻而易举。
<br/>　　“真的？”青雅公主闻言，俏脸上顿时充满了惊喜。
<br/>　　能加入苍穹神宫，这可是天大的幸事，前途将不可限量！
<br/>　　最主要的是，加入了苍穹神宫，她便可以经常见到面前这位神一般的男子！
<br/>　　“嗯！一个月后你直接去苍穹神宫吧！”苏莫点了点头，便转身离去。
<br/>　　让对方一个月后前去，是因为一个月后他肯定会回去了，那时候苍穹神宫才会打开山门。
<br/>　　不然的话，对方现在前去，根本踏不进苍穹神宫半步。
<br/>　　三长老和北海冰宫的宫主，也是同样的情况，苏莫均是让他们一个月后再去。
<br/>　　苏莫虚空踏步，一步万丈，不多时便远离了皇城，最后他进入了青俞岛边缘的一座城池之中。
<br/>　　这座城池之中，有一处天涯海阁的分阁，他来到了分阁之中。
<br/>　　分阁中有不少天涯海阁的强者，正是查到这次线索的人。
<br/>　　苏莫传下了命令，让人监视着青俞皇室的动向，他便找了一个宽敞的房间，在分阁中暂住了下来。

　　蓬莱仙岛，不，现在是永恒仙岛了，永恒仙岛上，灵根密布，仙草遍地，美轮美奂。
<br/>　　元始并没有收走这些灵根仙草，因为此岛如今都是他的了，他也不必如此着急。现在他要做的就是将这仙岛全部侦查一遍，看一看都有些什么。
<br/>　　来到永恒岛中心核心地带，元始发现一棵非常巨大的果树，果树有数十丈之高，树上九颗果子五颜六色，泛着异彩光芒，各种玄妙尽在其中，果香飘散，香气四溢，最是诱人。
<br/>　　果子上有“黄中”二字，像是道纹，又像是仙纹，充满道的韵味。
<br/>　　“这就是先天十大灵根之一的黄中李？！”元始瞪大眼睛，目瞪口呆，他早想到东海三仙山肯定会有顶级灵根灵宝，但没想到会是先天十大灵根之一的黄中李。
<br/>　　先天十大灵根，每一种都得天独厚，气运非凡，要是化形修炼，资质也是最顶级的。先天十大灵根中的绿柳，就是大神扬眉，先道祖一步成道的存在，菩提树化形而出，为后来的准提圣人。其余灵根也各有缘法，没一个是简单的。
<br/>　　就连元始，之前也只是遇到过先天七葫芦这一种，至于先天芭蕉树，那并没有被排在先天十大灵根之中。先天十大灵根为：黄中李、蟠桃、绿柳、苦竹、人参果、菩提树、扶桑树、桂花树、仙杏、先天七葫芦。
<br/>　　这十大先天灵根，每一个在后世都是鼎鼎大名，元始没想到，自己在蓬莱岛上会遇到一种，而且还是排名前三的黄中李。
<br/>　　黄中李，十万年一开花，十万年一结果，再过十万年，方能成熟，三十万年也只有九个果子。每一颗果子中都蕴含着一种道。
<br/>　　因为相传黄中李会吸收破碎的规则及法则来成长，故而，此果不仅吃起来美味可口，而且还能够助人悟道，助人领悟法则。是不可多得的顶级灵根，就算拿其他的灵根来换，元始也不会交换的。
<br/>　　元始摘了一个白色果子，吃了起来。果子入口即化，蓬勃的力量顿时散满全身，但更多的是来到元神深处，元神瞬间变得通透明目，让他的精神一下子提升了好多。更多乳白色的能量化为清流进入元神深处的神魂晶体之中。
<br/>　　神魂晶体随着这些年融入的神术越来越多而变得越来越小，现在随着这股清流的进入，这魂核变得更小了，直至缩小为一个点。
<br/>　　元始能够感受到，这股乳白色能流之中，有着一些奇异的神纹碎片，只是不多，元始猜测，这就是法则碎片，若是能够领悟，那将意味着多了一种法则，尽管这只是法则皮毛。
<br/>　　“要到第五重了吗？”元始感受到魂核已经压缩到了极致，就知道自己的《神元九转》已经到了第四转的极限，快要进入第五转界成。
<br/>　　“罢了，既然快要突破了，那么，也就趁着这个机会吧，这些年积累的也够多了。”元始默默道。
<br/>　　既然做出了决定，元始就不再拖泥带水，直接运转《神元九转》第五重功法。
<br/>　　轰！
<br/>　　那元神深处被压缩到极致的一点，瞬间就在元神之中爆炸开来。化为无数碎片，碎片之中，蕴含着无数神通。
<br/>　　一分为二，二为阴阳，阴阳印神通缓缓旋转，阳者上升为天，阴者下降为地。
<br/>　　两仪四象出，四象印神通一出，地水火风涌动，化为风水山川。
<br/>　　金木水火土，五行印神通瞬间按照五行相生排列，山川草木，冰水河流，天雷地火，世界始成。
<br/>　　之后又是各种神通出现，为这新生的世界增添许多色彩。
<br/>　　虽说这是在元始识海深处爆炸，但元始的神海并没有受损，反而随着元神世界的生成，元神修为大涨。
<br/>　　轰！
<br/>　　直至良久，元始的识海深处才停止响动，元始知道，自己已经修成第五转了。
<br/>　　内视自己的识海世界，元始发现自己的识海世界之中并非多姿多彩，反而是孤寂枯竭，他知道，这是因为自己的元神修为及大道领悟还不够的原因。
<br/>　　但元始并没有沮丧，能够开辟出元神世界，这已经是值得高兴的事了，至于其他的，以后慢慢积累就是了。他现在的元神，已经堪比准圣极限了，只差一步，就能进入混元强者的元神修为了。不过，元始知道，这一步之遥，会卡死很多人，也会卡主他很久。
<br/>　　这一枚黄中李果子让元始的元神修为达到《神元九转》第五转，修为虽然也有所增加，但并不是很明显，更多的是法力上的积累。到了元始现在的境界，修为可不是那么容易增加的。每进一步，都是值得高兴的。
<br/>　　虽然因为元神的突破，并没有让元始感悟到什么法则，但他已经心满意足了。黄中李，果然名不虚传。
<br/>　　元始祭出因果莲台，将黄中李连根带土及树下的神泉收入混沌空间，与混沌悟道古茶树为邻。此地只留下一个巨大的深坑。
<br/>　　元始心念一动，深坑渐渐被填了起来，只是光秃秃的，不怎么好看。
<br/>　　“嗯，这样。”
<br/>　　元始想了想，取出玉虚宫，放了出来。玉虚宫瞬间变大，落于此处。
<br/>　　“嗯？玉虚宫的档次有些低了，对不上我元始天尊的名号，应该加强一番。”元始如是想到。
<br/>　　元始收起玉虚宫，再取出一些这些年四处游走收集到的神玉神材，祭出神火，将神玉神材炼化，加入到玉虚宫之中。
<br/>　　元始的神火乃是众多真火神火融合而成，由赤帝火皇执掌，威力巨大。至于炼器手段，元始更是在行，先有盘古神尊遗留下的炼器手段，再有元始自己的一些炼器手法，经过多年磨合，元始已经创出了属于自己的大道炼器手段。
<br/>　　用火炼器，比得上用大道炼器吗？真正的炼器巅峰应该是用后天神材练出先天灵宝，只不过元始还没有达到那个级别而已。但就算不到那个境界，炼制一些接近先天的后天至宝，还是容易的。
<br/>　　元始用神火将神材全部炼化，再用大道手段将这些神材全部返回本源，最后用大道之力合成。再加入元始领悟的神纹，阵法，铭刻、禁制等，使之更加坚固辉煌。
<br/>　　就这样，一座崭新的华丽的晶蓝色宫殿出现在元始眼前，美轮美奂，不似人间所有，更胜天堂宝殿。
<br/>　　宫殿放大，坐落其中，元始用大法力、大神通，将周围布置一番，岛上的顶级灵根仙草，全部被元始收到玉虚宫中的灵草堂中，其他一些，则被元始散落到仙岛各处。
<br/>　　宫殿周围，元始移植了许多仙草仙花，更有一个神泉湖泊，其内鱼虾嬉戏，荷花开放。神光纵横，仙光缥缈，金光四射，荧光飞舞。
<br/>　　人间仙境，不外如是！
<br/>　　宫殿之后，假山密布，石窟丹崖，怪石嶙峋，神树灵植随处可见，彩凤双鸣，神鸟鸣叫，玄猿跳跃，白鹤振翅，灵鹿奔跑，热闹异常。
<br/>　　从仙岛入口，至玉虚宫殿门前，又被元始用大法力改造，一条用白玉铺成的台阶，拾级而上，直达宫殿。白光闪烁，玉色琉璃，清净洁白，尽显高贵。
<br/>　　仙岛之外，云雾缭绕，仙光袅袅。从岛外到岛内，从宫前到宫后，无不尽显仙家气派。
<br/>　　“好，如此地方才配得上我元始天尊住的道场！”元始满意一笑道。

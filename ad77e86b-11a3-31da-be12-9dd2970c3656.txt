　　海面之上，万千刀气，铺天盖地，杀机无限。
<br/>　　苏莫面色凝重如水，面对这铺天盖地的无差别攻击，他根本避无可避。
<br/>　　既然避无可避，那就以攻为守。
<br/>　　轰！
<br/>　　霎时之间，苏莫身上的气势爆开，狂暴的气息让他周身千丈之内的空间，立刻纷纷爆炸。
<br/>　　这一刻，苏莫动用了大三宝术，立刻实力大增。
<br/>　　不过，如今的大三宝术，仿佛成为了昨日黄花，效果越来越差。
<br/>　　他足足抽取了五成的精气神，才勉强增加了接近两倍的战力。
<br/>　　估计等到苏莫晋升武帝境之后，大三宝术的效果，便会微乎其微了。
<br/>　　嗡~~
<br/>　　体内玄力如滚滚长河，手中长剑亦是爆发出璀璨无比的三色剑光。
<br/>　　唰唰唰！！！
<br/>　　剑光舞动，剑影闪烁，密密麻麻的三色剑气，逆斩而上，迎向袭来的万千刀气。
<br/>　　瞬息之间，剑气便与刀气相击在了一起。
<br/>　　轰！轰！轰隆隆！
<br/>　　爆炸连连，惊天巨响不断，爆裂的刀气和剑气，形成无尽的狂涛，席卷八方。
<br/>　　嗖！
<br/>　　苏莫身形急退，瞬息间便后退千里。
<br/>　　他的实力还是要比对方弱不少，爆裂的冲击波，大多都是向他所在的方向冲击。
<br/>　　不过，即便如此，狂猛的冲击波，冲在他的护体玄力之上，依旧让他全身气血沸腾。
<br/>　　“嗯？”冷江见苏莫居然挡住了他的攻击，顿时眸光一凝，心中颇为惊讶。
<br/>　　他虽然早已从何传岩的口中得知，苏莫的实力很强，但也没想到会强到这种地步。
<br/>　　毕竟，他的实力，可是比何传岩强大太多了。
<br/>　　这一刻，十几座小岛上的近千名神武学府的弟子，都不禁惊讶了。
<br/>　　“厉害啊！这个苏莫若是没隐藏修为的话，的确不凡！”
<br/>　　“冷江的实力不弱，此人越三重修为，居然能挡住，不可思议！”
<br/>　　“本以为没有什么悬念，会是一边倒的碾压，没想到还有些看头！”
<br/>　　很多人都来了兴趣，一名准帝境的武者，挡住了武帝境三重武者的攻击，这可是很惊人的！
<br/>　　苏莫最终若是能保持不败，那也算是奇迹了！
<br/>　　因为到了武帝境界，还能越三重修为战斗的人，在整个神武学府都不多。
<br/>　　除了神子和十八位圣子，在武帝境之时，有能力越三重修为战斗之外，其他能有如此越级战斗能力的人，屈指可数。
<br/>　　“有趣！”
<br/>　　太史轩嘴角勾起了一丝笑容，这种人物，若是能拉到他们太史世家派系，那也不失为一件好事。
<br/>　　当然，前提是苏莫能在此战中活下来，毕竟看冷江的模样，可是动了必杀之下。
<br/>　　瞥了一眼远处古树下的木璃，太史轩身形一闪，瞬息间便来到了对方的身边。
<br/>　　“木璃导师，这个苏莫是什么背景啊？”太史轩笑着向木璃问道。
<br/>　　他认为木璃肯定了解苏莫的背景。
<br/>　　毕竟，上次在广场之上，他对苏莫发难，对方还帮助了苏莫，并且说出苏莫是火元圣师看重的人。
<br/>　　“不清楚！”木璃淡淡的摇了摇头。
<br/>　　太史轩闻言眉头微皱，心中不免有些怒气，这个女人实在太冷了，对任何人都不假辞色。
<br/>　　不过，他就是喜欢这种味道。
<br/>　　征服这种女人，才最有成就感！
<br/>　　“木璃导师，我有一些修炼上的问题，明日去向你请教！”太史轩不再谈苏莫的话题。
<br/>　　“嗯！”木璃淡淡的点了点头，她虽然不喜对方，但作为学府导师，对于弟子的请教，她也不好拒绝。
<br/>　　此刻，木璃不想理会太史轩，目光紧紧的盯着海面上的苏莫。
<br/>　　她的心中非常的震惊，她只知道苏莫精神力了得，炼器天赋不凡，却根本不知道，苏莫的实力居然也如此的不凡。
<br/>　　准帝境修为，抗衡武帝境三重武者，这种实力，一般的顶尖天才，都很难做到。
<br/>　　不过，即便如此，苏莫也是必败无疑吧！
<br/>　　抗衡几招应该已经是极限了，想要保持不败，几率实在太小了。
<br/>　　海面上空。
<br/>　　冷江没有急着出手，双眸微眯了下来，冷冷的盯着苏莫。
<br/>　　两招没有打败苏莫，听着周围之人的议论，他感到面上无光，心中的杀机也彻底的爆发了。
<br/>　　“冷大哥，苏莫是剑阵师，你小心他的灵剑阵！”
<br/>　　这时，何传岩的声音响起，在提醒冷江。
<br/>　　此刻何传岩脸色也是有些不好看，他本以为冷江挥手间便能震杀苏莫，没想到苏莫的实力，居然到了能抗衡冷江的地步！
<br/>　　所以，他决定提醒一番冷江，免得对方着了苏莫的道。
<br/>　　冷江闻言，眸光一凝，苏莫还是剑阵师？
<br/>　　不过，他也没有太过在意。
<br/>　　神武学府中也有不少剑阵师，剑阵师虽然厉害，但也要看是什么样的剑阵师。
<br/>　　以苏莫的修为，若是只有几柄灵剑，或者是十几柄灵剑，根本不能威胁到他。
<br/>　　但更多的灵剑，以苏莫的修为境界，估计很难催动。
<br/>　　“苏莫，刚才不过是开胃菜而已，现在，我就让你尝尝什么叫绝望！”冷江居高临下，冷冷的凝视着苏莫。
<br/>　　他手中的长刀，微微转动，耀眼的淡金色刀芒，暴涨万丈，将下方的海水都洞穿了。
<br/>　　苏莫面无表情，心中却在急思，如何才能打败对方！
<br/>　　不过，他也知道，对方估计还没有用出真正的实力，对方的深浅，他还不清楚。
<br/>　　“囚天刀法——囚天一式！”
<br/>　　冷江陡然大喝一声，双手握刀，一刀重重的劈了下来。
<br/>　　刹那之间，一柄长达万丈的恐怖刀影，如长虹贯日，又如泰山压顶，狠狠的向苏莫斩了下去。
<br/>　　巨大的刀影，速度如风驰电逝，威势惊天，锋芒绝世，破碎一切。
<br/>　　在刀影出现的刹那，苏莫便感觉到了一股奇异的力量，这股力量仿佛凝固了空间，束缚住了一切。
<br/>　　“什么？”
<br/>　　苏莫顿时心中一惊，因为他感觉到自己居然难以动弹了，一股强大的凝固之力，从巨大的刀影上倾洒而下，将他的身体束缚住了。
<br/>　　不仅如此，他周围百里之内，空气、海水，一切都静止了，仿佛时间都停止了一般。
<br/>　　囚天刀法，囚天、囚地、囚人，囚困一切！

陆子羡和夏夕绾进了手术室，厉老夫人脑部的肿瘤相当凶险，这场手术被判定成功率绝不超过百分之十，所以这一场中西医的联手手术引起了整个医界的轰动，可谓万众瞩目。
<br/>
<br/>        这场手术整整进行了八个小时，从清晨到傍晚时分，手术室的红灯一直亮着，没有任何人出来。
<br/>
<br/>        晚上六点，当黑夜缓缓降临时，“吱呀”一声，手术室的大门终于拉开了，有人出来了。
<br/>
<br/>        穿着白衣大褂的夏夕绾走了出来。
<br/>
<br/>        李文清院长，周主任，双双，还有很多人快速的围了上来，但都没有徐少南速度快，徐少南冲上前一把捏住了夏夕绾的肩膀，“丑丫头，手术怎么样了，我外婆现在怎么样了？”
<br/>
<br/>        夏夕绾伸手摘下了脸上的白色口罩，露出了里面的轻纱，八个小时让她白腻的额头上出了一层细小的汗珠，在所有人的期待里，她露出微笑，“手术…成功了。”
<br/>
<br/>        哇！
<br/>
<br/>        大家不可思议的欢呼了一声，这场被医学界判定必败的手术竟然真的成功了，成功了！
<br/>
<br/>        徐少南被打了一巴掌的怒火也被这个好消息给冲淡了，看夏夕绾也看顺眼了些，“丑丫头，这一次你走运，我先放过你…”
<br/>
<br/>        话还没有说完，夏夕绾双腿一软，整个人往地上滑去。
<br/>
<br/>        “喂，丑丫头！”徐少南吓了一跳，迅速伸手搂住了夏夕绾的腰肢，将她给拉住了。
<br/>
<br/>        这一搂徐少南的手直接僵了一下，女孩的腰又细又软，堪堪的盈盈一握，他搂过不少女孩，还没有搂住这么软的。
<br/>
<br/>        丑丫头是丑丫头，但是身体像没有骨头似的，徐少南脑袋里突然蹦出了一个邪恶的念头，身体软成这样的不知道扑倒在床上是什么销魂的滋味。
<br/>
<br/>        但是下一秒夏夕绾就被双双给抢走了，双双将夏夕绾扶坐在了椅上，紧张关心的问，“绾绾，你怎么了，你没事吧？”
<br/>
<br/>        李文清院长过来笑道，“八个小时的手术时间太长了，绾绾是体力不支，喝杯水休息一下就行了。”
<br/>
<br/>        这时护士将厉老夫人推了出来，手术过后的麻药还没有醒，厉老夫人的各项身体指标都正常，被送入病房里了。
<br/>
<br/>        徐少南看了夏夕绾一眼，然后扭头去看自己的外婆了。
<br/>
<br/>        陆子羡是最后走出来的，八个小时的手术他身上的白衣大褂依然干净的纤尘不染，气质清冷出尘，刚洗过手，他将一杯热水递给了夏夕绾，“夏医生，你今天表现的很好。”
<br/>
<br/>        他叫她夏医生。
<br/>
<br/>        夏夕绾休息了一会儿已经恢复体力了，她接过水杯看向陆子羡，“陆院士，你今天也很棒。”
<br/>
<br/>        两个人两年前在帝都大街上就合作过，这一次手术更是有一股难言的默契，现在相视一笑。
<br/>
<br/>        “子羡，夕绾，你们都很厉害，走，收拾一下，今晚我请客，请大家都去喝一杯。”李文清院长宣布道。
<br/>
<br/>        所有人都开始欢呼，“夕绾万岁，陆院士万岁，校长万岁！”

<br/>        ……

<br/>        他已经在第一时间里知道了这场跨界手术的胜利，因为微博和各大电视台都争先恐后的播报了这场手术的成功。

今天陆寒霆穿了一件黑色衬衫，这种低调内敛的黑色衬的他整张俊脸犹如天工雕琢般精致完美，衬衫的衣袖往上卷了两道，戴着名贵腕表的大手按在方向盘上，他整个人看着沉稳又矜贵，是商界成功男士的成熟气场。
<br/>
<br/>        陆寒霆扭头，一下子就看到了站在路边的夏夕绾。
<br/>
<br/>        女孩今天穿了一件暗红色的短款针织衫，下面黑色铅笔裤，纤柔窈窕，少女气十足，清纯的长发散落肩头随着晚风徐徐拂动，连这空气里好像都散发着她身上那股芬芳。
<br/>
<br/>        “陆先生…”这时女孩唤了他一声。
<br/>
<br/>        陆寒霆看着她，女孩纤柔的身影牢牢的抓住了他整个眼球，让他舍不得移开目光，不过他掀了掀薄唇，冷酷道，“我现在没时间，你自己回家吧。”
<br/>
<br/>        “陆先生，我…”
<br/>
<br/>        夏夕绾想说话，但是“嘟嘟”两声，陆寒霆已经把电话给挂断了。
<br/>
<br/>        陆寒霆必须强迫自己把电话给挂断了，不让她软糯的嗓音贴在他耳畔蛊惑着他，只要她再多说一句，哪怕多说一个字，他大概就会控制不住自己出现在她的面前。
<br/>
<br/>        现在他的手机里还静静的躺着一条短信，是陆子羡发来的夏夕绾中了花毒，让她跟我回帝都，我可以救她。
<br/>
<br/>        陆寒霆将英挺的后背倚靠在座椅里，英俊的眉心里染上了一层颓废和阴鹜，二次试毒让她染上了花毒！
<br/>
<br/>        她中毒了！
<br/>
<br/>        陆子羡要带她去帝都，刻不容缓！
<br/>
<br/>        陆寒霆敛了一下俊眸，薄唇边勾出了一道黯淡自嘲的弧线，他已经跟她提出离婚了，可是，一点时间都不能给他。
<br/>
<br/>        这时耳畔传来了惊呼声，“让开，快让开！”
<br/>
<br/>        陆寒霆抬眸一看，有个人骑着滑板车疾驰而来，眼见着就要撞上夏夕绾了。
<br/>
<br/>        那个人在大叫，提醒夏夕绾，但是夏夕绾好像忘记了反应，只是傻乎乎的站在原地。
<br/>
<br/>        陆寒霆瞳仁一缩，身体先于大脑的伸手就拉开了车门，健步跑了过去。
<br/>
<br/>        在滑板车要撞上夏夕绾时，一个路过的大妈比陆寒霆更快一步的伸手扯过了夏夕绾，“姑娘，小心！”
<br/>
<br/>        夏夕绾被拉扯的往旁边避了一下，那个滑板车擦着她走了，相当惊险。
<br/>
<br/>        陆寒霆停下了脚步，刚才他的心脏就要跳出来了，精健的胸膛上下喘动着，他紧盯着夏夕绾纤柔的身影，深怕她在他面前再发生什么意外。
<br/>
<br/>        但是他不敢上前，只能停在离她几步远的地方看着她。
<br/>
<br/>        “大妈，谢谢你。”夏夕绾脸色有点白，她大概猜到发生什么事情了，柔声道谢。
<br/>
<br/>        好心的大妈摇头，还不忘唠叨道，“姑娘，这车都来了，你怎么都不避一下，刚才太危险了，慢一步你都要被撞翻的。”
<br/>
<br/>        “大妈，我不是故意的，我眼睛看不见了。”
<br/>
<br/>        她说什么？
<br/>
<br/>        陆寒霆高大英挺的身躯直接僵在了原地，她那句“我眼睛看不见了”在他耳畔炸开，然后不停的回荡。
<br/>
<br/>        她的眼睛看不见了？
<br/>
<br/>        陆寒霆走上前，来到了夏夕绾的面前。
<br/>
<br/>        女孩那双翦瞳依然清澈盈亮，但是她以往纤尘水灵的眸子已经失去了所有的色彩，眸底毫无聚焦。
<br/>
<br/>        她的眼睛真的看不见了！

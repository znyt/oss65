　　金身佛尊这位强者，楚风眠也曾经听闻过。
<br/>　　这金身佛尊，便是在九域古籍之中所记载的一位净土宗强者，是佛门金刚宗一系的强者。
<br/>　　佛门金刚宗一系，主修肉身，以纯粹的肉身对敌。
<br/>　　这金身佛尊便是将肉身修炼到极致的一尊强者，以纯粹的肉身之力，羽化登仙，踏入天人境的一尊强者。
<br/>　　羽化登仙，踏入天人境，本就是无比困难的事情。
<br/>　　历史之中无数惊艳的天才，最终都无法跨越这一步，在如今这个没落的时代之中，天人境更是一种神话之中的境界。
<br/>　　哪怕是号称距离天人境最近，最为耀眼的天才的少帝，也距离真正的天人境，相差不少。
<br/>　　想要以单纯的肉身之道，羽化登仙，踏入天人境，比起寻常以武道踏入天人境，还要困难得多。
<br/>　　能够成就这一道的人，寥寥无几，金身佛尊便是其中之一。
<br/>　　据说这金身佛尊，已经是将肉身凝炼到了极致的地步，堪称无敌，甚至是在一些传闻之中。
<br/>　　金身佛尊的肉身，已经是凝炼到了不朽真身的地步。
<br/>　　在古老的时代之中，金身佛尊便是净土宗赫赫有名的战佛之一，不知道是帮助净土宗，斩杀过多少的敌人。
<br/>　　但是最终，这金身佛尊也已经陨落。
<br/>　　没想到在这藏地之中，就然是连金身佛尊的陵墓，都有显现而出。
<br/>　　让楚风眠所动容的，却是这金身佛尊陵墓之中可能出现的宝藏。
<br/>　　“六翼蜈蚣，藏金精，只怕是都藏在这金身佛尊的陵墓之中。”
<br/>　　方才在外的时候，在佛寺还未开启的一刻，无数宝光便是显现而出。
<br/>　　在这无数显现而出的宝光之中，楚风眠最为想要得到的，就是六翼蜈蚣跟藏金精。
<br/>　　这两件宝物，都是炼体之道的圣物，是凝炼不朽真身，将肉身凝炼到极致必不可少之物。
<br/>　　这两件宝物，在佛寺之中，最有可能存在的，便是这金身佛尊的陵墓之中。
<br/>　　就像是万念佛珠，就是在万念佛尊的陵墓之中出世的。
<br/>　　这金身佛尊的陵墓之中，哪怕是没有六翼蜈蚣，藏金精这两件至宝，只怕也会有着其他炼体的至宝。
<br/>　　楚风眠利用神树，吞噬了青铜天魔之后，肉身已经是开始想着半步不朽真身转变，现在已经是达到了瓶颈。
<br/>　　得到一些炼体之道的至宝，也足以来帮助楚风眠来突破瓶颈，肉身再进一步。
<br/>　　“看来楚兄是有兴起，来跟我们一同去探索那金身佛尊的陵墓了？”
<br/>　　小耀王寒夜，在一旁看到楚风眠动容的神色，不由的开口道，他倒是没有任何的意外。
<br/>　　金身佛尊的威名，不止是在九域，在大罗天星域之中，都是威名显赫，他的陵墓，其中蕴含的无数宝物，没有人会不心动。
<br/>　　“是有兴趣，不过这金身佛尊陵墓之中的宝物，要如何分配？”
<br/>　　楚风眠看了一眼寒夜，突然开口道。
<br/>　　金身佛尊的陵墓，其中的宝藏定然不少，而且只怕是想要得到，并不容易。
<br/>　　不然这寒夜，也不会将这金身佛尊陵墓的消息告诉楚风眠了，而是会直接选择独吞了。
<br/>　　这寒夜既然是想要叫人一同去探索这金身佛尊的陵墓，便是显然是以这寒夜一个人的实力，无法将这金身佛尊陵墓，收入囊中。
<br/>　　如果联合，宝物分配倒是一个问题。
<br/>　　“这金身佛尊陵墓极大，其中处处都是遗留下来的佛门禁制，寸步难行，进入其中，所有的宝藏，凭实力而得，谁先得到，就属于谁。”
<br/>　　小耀王寒夜身形一动，从椅子上站了起来，对着楚风眠说道。
<br/>　　“楚兄如何？”
<br/>　　“凭实力得到？好。”
<br/>　　楚风眠点了点头，这是最好是方式了。
<br/>　　“楚兄，那我们现在尽快出发，免得那金身佛尊的陵墓，被其他人所察觉。”
<br/>　　寒夜看到楚风眠答应下来，便是立刻化为了一道遁光，冲天而起。
<br/>　　于此同时寒夜的手心之中，也打出了几道玉符，当即是向着远方飞去。
<br/>　　“这是？”
<br/>　　楚风眠看到寒夜的举动，有些疑惑的问道。
<br/>　　“是这样，这金身佛尊陵墓中的禁制，以我们二人的实力，也难以开启，我还需要在呼唤几位强者到来，不过楚兄不用担心，规矩不变。”
<br/>　　小耀王寒夜的目光之中，闪过了一丝不易察觉的狡猾，旋即化为了一道遁光，冲天而起。
<br/>　　无数圣院宗圣子，也同样是催动起遁光，冲天而起，向着远方飞去。
<br/>　　“楚兄？我感觉此事有诈。”
<br/>　　在这时，炎撼世走了过来，小声对着楚风眠说道。
<br/>　　“这小耀王，性格狂傲无比，今日楚兄打了他的脸面，他绝不会善罢甘休才对，怎么可能如此客气，前去只怕是会有危险。”
<br/>　　“无妨，一会出事，你们不用插手，保护好自己就是，那寒夜敢轻举妄动，我一个人就可以杀了他。”
<br/>　　楚风眠眼神冰冷无比。
<br/>　　这寒夜要是乖乖的真的打算来跟楚风眠一起联合探索陵墓的话，那还好说。
<br/>　　俗话说的好，伸手不打笑脸人，但是这寒夜要是来敢算计楚风眠的话，那今日就是此人的死期。
<br/>　　“我们走。”
<br/>　　楚风眠身形一闪，冲天而起，化为了一道遁光追了上去。
<br/>　　炎撼世等五灵宗圣子，也紧随其后，化为了一道道遁光，跟随过去。
<br/>　　数百道遁光，在这藏地之中穿行。
<br/>　　看到如此之多的强者一同行动，不少独行的古帝，都纷纷退避开来，生怕招来祸端。
<br/>　　一路穿行，这一片藏地，看似不大，但是实际上却是近乎无穷无尽，似乎是可以一直延伸下去。
<br/>　　“缔造出这一片藏地的强者，实力无法想象。”
<br/>　　楚风眠观察着整个藏地，不由的感叹一声。
<br/>　　创造一方小世界，踏入天人境的强者，就足以做到，像是现在剑道门所在的哪一方小世界，就是由楚风眠的大师兄剑凌创造而出的。
<br/>　　但是那一方小世界，跟眼前这一方藏地相比，简直是云泥之别，这一方藏地单单是大小，就比起那一方小世界大上千倍万倍，而且其中空间也是无比的稳定。

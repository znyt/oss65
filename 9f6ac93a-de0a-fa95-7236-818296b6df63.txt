　　“已经七天时间了。”
<br/>　　萧逸的眼眸，异常冰冷。
<br/>　　紫炎火翼振动下，周遭火焰气息环绕，炙热至极，却浇不熄他眼中冷意。
<br/>　　这些天，他虽然已经不断加快剿杀邪修的速度。
<br/>　　但奈何六龙地域内，邪修、还有邪修分部的数量，着实惊人。
<br/>　　再加上龙家还有六龙卫的处处阻挠、追踪，生生拉了他不少时间。
<br/>　　“顶多再有5天，以毒袍的手段，必能恢复伤势。”
<br/>　　萧逸眼眸眯了眯。
<br/>　　5天之内，若再无法逼毒袍现身，事情将变得异常麻烦。
<br/>　　他能和毒袍鏖战，能压毒袍一头，是因为他手段尽出，还有六种火焰自成轮回下带来的增幅。
<br/>　　可六色火焰化作轮回，包括体内元力也一并化作轮回，他现在还未能瞬间做到。
<br/>　　一旦毒袍伤势痊愈，他将再难以击杀。
<br/>　　这七天时间，他在快速剿杀邪修，同时，也在以九阳轮回诀功法，不断调动着元力。
<br/>　　他要让体内庞大的元力，愈加快地能瞬间契合九阳轮回诀。
<br/>　　元力和火焰，以及功法，三者间的契合速度，已经原来越快了。
<br/>　　但这终究需要不短的时间。
<br/>　　算起来，若是修为能再提升一大截的话，他的实力，应当能有一次不小的提升。
<br/>　　只是，以他的庞大气泉，修为别说提升一大截，就算提升一小截都异常困难。
<br/>　　到哪去寻那么多修炼资源？
<br/>　　“呼。”萧逸深呼吸了一口气。
<br/>　　修炼之路，难如登天。
<br/>　　武者的一生，若真要追求那极致武道，真要乘风破浪，便早就该有劈波斩棘、不惧前路的觉悟。
<br/>　　现今这丁点儿困难，又算得了什么。
<br/>　　“既是这些小型分部找不着你，那便直接去大型分部。”萧逸眼眸一冷。
<br/>　　六龙地域这边，隐藏在暗处的小型分部，数量极其庞大。
<br/>　　而每个地域，都必会有一个大型分部，统领着地域之内的各个邪修，以及小型分部。
<br/>　　这些天，萧逸一直没有去大型分部。
<br/>　　因为他知晓，以毒袍的谨慎，绝不会去大型分部的，反倒会藏身在数量众多的小型分部中。
<br/>　　故这些天，萧逸一路横扫着这些小型分部。
<br/>　　一来，打算通过击杀大量邪修，逼毒袍现身。
<br/>　　二来，或者中途就能寻到毒袍藏身之地。
<br/>　　但现在，他放弃了这个想法。
<br/>　　龙家和六龙卫总是处处阻挠，他的剿杀速度大减，根本无法在短时间内击杀足以让毒袍现身的庞大邪修数量。
<br/>　　既如此，那便直接去大型分部。
<br/>　　大型分部，作为统领六龙地域内各小型分部的存在，里头强者云集，邪修汇聚。
<br/>　　“我便不信你能看着整个大型分部被屠。”萧逸咬牙自语。
<br/>　　……
<br/>　　半个时辰后。
<br/>　　五虎门，山门前，萧逸的身影，轰然降落。
<br/>　　指尖轻弹，一道火焰飞跃而出，瞬间将山门炸裂。
<br/>　　大手一挥，一个屏障布下，包裹了整个五虎门。
<br/>　　嗖…
<br/>　　萧逸瞬间闪身进入五虎门内，沿途所过，五虎门弟子死伤殆尽。
<br/>　　宗门广场处。
<br/>　　一道道身影，疾速而来。
<br/>　　萧逸如此大的动作，自是引起了五虎门强者的注意。
<br/>　　一个中年人，率着十数老者，拦下了萧逸。
<br/>　　中年人，正是五虎门门主，十数老者，则是门内长老。
<br/>　　“阁下是何人，为何毁我五虎门山门，还大肆杀戮？”五虎门主，冷声质问。
<br/>　　“呵。”萧逸淡漠一笑，“五虎门主，何必明知故问。”
<br/>　　“我毁你山门，沿途又击杀门内弟子；你身为门主，却未立即对我出手，证明你已经认出我是谁了。”
<br/>　　不错，萧逸如此动作，大肆杀戮。
<br/>　　若是换了正常人，一位门主，早就怒而出手了。
<br/>　　可五虎门主，竟未出手，反而冷声质问。
<br/>　　“紫炎，易霄。”五虎门主，眯着眼，直视萧逸。
<br/>　　萧逸笑笑，“能让易某亲自上门，且毫无废话便立刻出手，想来，你们也知道原因。”
<br/>　　五虎门主皱了皱眉，“易霄分殿主，当我五虎门是邪修聚集之地，邪修分部？”
<br/>　　“你说呢？”萧逸冷笑一声，“六龙地域，素有六龙五虎之誉；谁能想到，二者之一，竟是邪修分部。”
<br/>　　六龙地域，素有六龙五虎的说法。
<br/>　　六龙，说的自然是龙家。
<br/>　　五虎，便是这五虎门。
<br/>　　五虎门，六龙地域第一宗门；同时，亦是龙家之下第一势力。
<br/>　　当然，谁也想不到，五虎门，竟会是个邪修分部。
<br/>　　可别人想不到，却瞒不过萧逸的天地窥探。
<br/>　　在他之前的窥探中，六龙地域的邪修数量，异常庞大。
<br/>　　而五虎门这里，散发的黑气，最是浓郁澎湃，远超别的地方，显然这里就是大型分部。
<br/>　　也是这里，统领着六龙地域所有的各邪修，各小型分部。
<br/>　　事实上，前些天，萧逸遇到的大型分部，有好几个，都是地域内的最强宗门，如火牙宗，还有这五虎门。
<br/>　　但其实，倒并非每个地域的最强宗门，都是大型邪修分部，这终究只是极少一部分罢了。
<br/>　　而但凡是最强宗门乃邪修分部的地域，无不是邪修猖獗异常，数量过人，便如这六龙地域。
<br/>　　“本门主，实在不知易霄分殿主在说什么。”五虎门主摇了摇头。
<br/>　　“我随你装蒜，我也懒得和你争辩。”萧逸冷笑一声。
<br/>　　“我来，只是杀人。”
<br/>　　嘭…萧逸手中一道紫炎凝聚。
<br/>　　霎那间，紫炎滔天。
<br/>　　五虎门主，脸色一变。
<br/>　　对，易霄不是来和他讲道理的，而是来杀人的，他狡辩也无用。
<br/>　　“各长老听令，随门内弟子，带走全门之物，立刻撤离。”
<br/>　　“是。”身后，十数长老应答一声。
<br/>　　“想走？”萧逸冷笑一声。
<br/>　　滔天紫炎，瞬间暴走。
<br/>　　五虎门主，脸色一冷，“一个毛头小子，也敢来我这里放肆？笑话罢了。”
<br/>　　五虎门主，瞬间出手，一掌拍出。
<br/>　　不过，其结果，却是被萧逸反震十数步。
<br/>　　“虎护法。”原本欲要离去的十数长老，脸色一变。
<br/>　　“不必管我。”五虎门主冷喝一声，“立刻走，这里本护法会处理。”
<br/>　　“护法？”萧逸眼眸眯了眯。
<br/>　　双手，早已化作兽手；五色圆环，也已凝聚，火焰流光转动不停。

　　不一会碧清进来了，也刚洗完澡，穿着睡衣，来到叶少阳身边，找了个凳子坐，冲叶少阳微微笑着，表情有点讨好。
<br/>　　“咋的了？”叶少阳问。
<br/>　　“你是不是不想我收了那个苏沫？”
<br/>　　叶少阳沉着脸说道：“你真要这么做，我也没办法，但如果你陷入危险，我也不会救你了。”
<br/>　　“为什么，她明明是你的仇人，你为什么要护着她。”
<br/>　　“她跟我有仇，是为了宗门，不是私仇。我不是护着她，我只是觉得……你是你，她是她，虽然你们曾是一体，但现在是两个人——”
<br/>　　“我不想听这些道理，我是非收了她不可的，不管你帮不帮我，我都会做。”
<br/>　　叶少阳听她这么说，也就不再说话了，本来他也不是非要保护苏沫不可。
<br/>　　碧清也不再说什么，回到自己房间去了。
<br/>　　叶少阳一个人在阳台上坐了半天，听见有人敲门，过去开门一看，吓了一跳，门外站着十来个人，都笑吟吟的看着自己。
<br/>　　美华、小青、小白、秋莹站在最前面，后面还有几个，叶少阳看了一眼，有个面熟的，居然是朱智慧，炭头，还有几个小鬼。
<br/>　　“你们怎么都来了！”
<br/>　　叶少阳有点吃惊。
<br/>　　“老大，大帅说这两天要打架，让我们都来参加，我也就来了，嘿嘿，老大你可想我没有啊。”朱智慧嘿嘿傻笑着，当初从幻境里出来，他就想留在叶少阳身边，叶少阳觉得自己身边不需要这么多人跟着，就让他跟林三生去了空界。
<br/>　　叶少阳摸了摸他的猪头，无奈道：“你们为啥不直接进屋，来外头敲什么门。”
<br/>　　“嘿嘿，这是美华姐交代的，万一老大正在做什么见不得人的事，被我们撞到了……”小白捂嘴笑，叶少阳在她脑袋上敲了一下，瞪眼道：“真是人小鬼大，我能有什么见不得人的事！”
<br/>　　让他们进屋，房间里瞬间人就满了，全坐在床上，叶少阳一个个打招呼，问他们林三生为啥没来。
<br/>　　“他呀，正在亲自操练士兵，暂时来不了。”美华说道。
<br/>　　“这时候了，操练什么士兵？”叶少阳皱眉。
<br/>　　美华笑道：“这才是军师高明的地方啊，黎山老母他们是偷偷来的人间，也担心祭祀仪式被人破坏，也派了眼线盯着军师呢，军师这是故布疑阵，麻痹他们。他说等行动之前，他跟小九一起过来。”
<br/>　　小白道：“老大，军师让我告诉你，他得到情报，黎山老母他们大概会在明天晚上行动，所以我们应该早点上山埋伏，只是要小心被识破了。”
<br/>　　叶少阳道：“军师有什么计划？”
<br/>　　“没有，他说听老大你的安排就好。”小青笑笑，“他说这种法术界的具体战斗，老大其实比他安排的更好。”
<br/>　　敢情全让自己包办了。
<br/>　　叶少阳先让找了一张纸，把白天踩点后记下的地形要点，在纸上画下来，然后设计战术，思来想去，问题还是很多，叶少阳一下子看到秋莹，眼前一亮，让她晚上就到山顶上去，隐身藏好，一旦发现黎山老母他们，立刻尾随，确定祭祀的地点，就尽快通报，秋莹满口答应。
<br/>　　大伙正在讨论着，叶少阳接到老郭的电话，让他到外面去。
<br/>　　叶少阳起身出去，在客栈外面的山路上看到一辆轿车，老郭从车上下来，把后备箱打开，招呼叶少阳过去看。
<br/>　　有很多口袋，里面有鸡喉骨、硝石粉、朱砂、生石灰等等一干作法药，还有好些个啤酒瓶，里面装着深色液体，叶少阳想拿一瓶看，老郭连忙挡住，笑嘻嘻地说道：“这里头是狗血和人中白，味道太难闻了，不要动。”
<br/>　　叶少阳赶紧缩回手，道：“弄这些干什么，我的法器也怕这些东西。”
<br/>　　“以防万一啊，真到那一步，就照着两败俱伤的来，反正有备无患。怎么办，现在就去布置？”
<br/>　　叶少阳想了下，也觉得事不宜迟，点头答应。“只是这么多东西，靠我们两个怎么搬上去？”
<br/>　　老郭笑着拍了拍胸口，“我这个后勤部长办事，什么时候让你操心，人我都找好了，你在这等一会。”
<br/>　　说完开车下山，不一会又上来，车门打开，后排下来三个汉子，都光着上半身，挑着扁担，叶少阳一看就明白了，这些都是挑山工，之前进山时候遇到过，景区大门外蹲了好些个，帮人挑行礼的。
<br/>　　“就这些东西，你们给我搬到山顶。”老郭把后备箱打开，让三人看着。
<br/>　　三人研究了一下，讨论之后，表示要两趟才能搬完，要五百块，老郭爽快答应，于是让他们动手开工。
<br/>　　两趟下来，东西全运到了山顶帝君庙附近的空地上，这些挑山工很朴实，也没问他们这些东西是干啥用的，老郭多给了他们五百，三人满意而归。
<br/>　　“你平时不是很抠的吗，怎么今天大方了。”叶少阳好奇。
<br/>　　“他们很不容易，长年挑东西，身子都累垮了。”老郭看了一眼三人的背景，叹道，“人间有三苦，撑船打铁磨豆腐，我看这挑工不比这三样轻松，要不是为了吃饭，谁干这个啊。”
<br/>　　叶少阳听了，心里挺感动。
<br/>　　老郭采购的物品中，还有一个大号的旅行包，打开来，里头有两根折叠扁铲，还有四个坛子，这些都是叶少阳交代买的。
<br/>　　这时候天还没黑，还有些游人，不方便动手，老郭去不远的一家商店里买了几瓶啤酒和花生米之类的，跟叶少阳一起坐在草地上聊天喝酒，一边闲聊起来。
<br/>　　“小师弟，你想过什么时候收山没有？”老郭啃着一只鸡爪，看似漫不经心地问道。
<br/>　　“收山？”
<br/>　　“常在河边走，哪能不湿鞋。你自己想想，这一年来你过的是什么日子，不是跟这家怼，就是跟那家怼，也是运气好，大家一直都没事。”
<br/>　　“这……我也是没办法，我倒是想清静，但各种事情逼的我清静不下来啊。”
<br/>　　老郭道：“我知道啊，但是你想过没有，日子不能一直这样啊，总有个尽头，天劫之乱，也不是你一个人的事，嗯？”
<br/>(本章完)

　　段氏基地的会议厅内，段江伟及一干基地高层依次落座其内，现场一片安静之中，猎王带着几名猎者联盟的管事踏入。
<br/>　　猎王等人的入场，引来了众人集体的注视，近些天在段氏基地周边发生的事，让整个段氏基地管理层都心头慌乱，也在这位名声显赫的猎者之王面前不敢与之对视。
<br/>　　“猎王，请坐。”段江伟语态平静，示意着会议厅下首的几个位置。
<br/>　　只是在他话音刚一落下之后，一名跟随猎王进入的高级猎者便冷哼出声：“这就是你们段氏基地的待客之道？”
<br/>　　“你什么意思？”一名基地高层管理皱眉出声：“我们段氏基地对你们有什么招待不周吗？你们在此的一切所需，我们都没有吝啬吧？”
<br/>　　那名高级猎者面色彻底冷了下来，他嘲讽的看着在场众人：“是，的确在物质上你们有求必应，但这也是我们猎王接下那一则任务内的备注条件，虽然你们也说不清楚那任务是谁发布，可猎王来此，引发了大批猎者而来，从而让段氏基地在这么长一段时间内的安然无忧，区区一部分的物资难道也要拿出来特地说明吗？”
<br/>　　看着这名猎者相当不满的样子，鲁初雪与段江伟对视一眼后起身，道：“我们当然对猎者联盟的行动很感激，也没有要斤斤计较的意思，只是这次会议很重要，还请各位先入座。”
<br/>　　几名猎者依然面带不满，但猎王本人却直接走到了座位上坐下，一声不吭也没什么表情变化，让人看不出他到底在想什么。
<br/>　　跟随猎王进场的几名猎者见到这一幕后，也没有再继续惹事，依次落座。
<br/>　　鲁初雪有些心中没底，她摸不准这样状况下的猎王，到底会不会就此在这里发难，毕竟……
<br/>　　“开始吧。”此时段江伟的声音响起，打断了鲁初雪的思绪飘忽。
<br/>　　鲁初雪回过神，拿起桌上的一份文件扫了眼，道：“此次会议的主要内容和近期基地周边的丧尸群有关，众所周知这批丧尸来自南方的埋骨之地，是由聚集的尸潮分散后在基地周边游荡形成，经过当下最权威的研究院狼牙研究部的袁熙叶院长分析，这些尸潮分散成游荡状态徘徊在基地周边，很大几率为智慧丧尸的刻意指使。”
<br/>　　“也就是说这一切都是一个陷阱，我们所看到的丧尸群分散不成形，似乎可以派出小队分批解决的状况，为假象。”
<br/>　　说着鲁初雪顿了顿，余光轻轻一瞥坐在下方的那些猎者，道：“于是段上将第一时间便发出了召回命令，派出大量传讯人员，将分散在周边地带的各位猎者寻回，以免出现被丧尸反埋伏的情况。”
<br/>　　话说到这里，一名猎者当即坐不住，愤怒的一跃而起大声道：“说的倒是振振有词，好像你们段氏基地一点动机也没有的样子！”
<br/>　　其他猎者也难以压制内心的愤怒，开始当场反驳。
<br/>　　“可是结果呢？你们口口声声说的如此大义炳然，最后却是我们猎者联盟死了这么多人！”
<br/>　　“我就很奇怪了，你们的人跑到边境之外都没有受到丧尸围攻，我们猎者联盟的猎者在基地附近都能遇到高阶丧尸群，运气真有这么差？”
<br/>　　“一次两次是巧合，可每一批猎者小队都是如此，每一个人的遭遇都相同，甚至那些丧尸对你们段氏基地的人没有任何进攻举动，连就在旁边都当没看见一样，这就太不对劲了吧！？”
<br/>　　“这件事你们要是不给猎者联盟一个解释，我今天就掀了这基地！”
<br/>　　随着猎者们的情绪爆发，段氏基地的高层也起身，与这些猎者对峙起来。
<br/>　　“我们哪来什么动机？我们全部的行动都一直在配合你们猎者联盟，这次事件更是出动了全部的防守人员，就是为了把你们找回来。”
<br/>　　“你们有一半的人都是因为跑太远而出事，我们在接到消息的第一时间就行动起来，哪来什么不纯动机？”
<br/>　　“倒是你们行动的时候，毫无章法直接就冲进丧尸密集的地带，很多人进去就出不来，我们也没要求你们做这些啊！”
<br/>　　猎者一方想讨说法，基地一方则是想澄清责任，眼看对峙即将升级到骂战，段江伟当即抬手示意众人安静下来。
<br/>　　说的激动的几名基地高层连忙住嘴，安静的待在原位不再吭声。
<br/>　　猎者们见到对方硒鼓，也全体不再多说，目光都放在了一直没有说过话的猎王身上。
<br/>　　“猎王。”段江伟看向猎王，斟酌着开口道：“这件事你有什么看法？”
<br/>　　猎王抬起眼，面色平淡的环顾一圈，随后道：“接任务，前来基地，大批猎者无偿前来支援，这些都属于任务之外的，客观上来说猎者联盟的确认为段氏有些不厚道。”
<br/>　　这番话一出，几名基地高层立马急了，只是刚想开口反驳时，段江伟便再次抬手让他们安静。
<br/>　　“非客观上呢？”他静静的看着猎王，心头忽然冒出了某种古怪的想法。
<br/>　　眼前这个猎者联盟的创始人，传说中战力恐怖无边的猎者之王，在这个地方这个场景中的反应，怎么让他有种无比熟悉的感觉？
<br/>　　听到段江伟如此问，猎王的嘴角一勾：“非客观上啊……”
<br/>　　说着他的声音一顿，而后目光猛然一变，犀利无比如同剑芒扫射：“当然是觉得这事有诈，是不是你们段氏基地勾结丧尸，想要对猎者联盟实行一次铲除计划？！”
<br/>　　随着猎王锋利的反问，大厅内顿时安静无比，连鲁初雪都被镇住，没想到这名从进来到前一刻都看起来随意平淡的人，竟然一下子爆发出了这等可怕的气势。
<br/>　　段江伟却是在此时彻底愣住，他终于知道为何会在猎王身上有熟悉的感觉了，眼下的这种情况让他感觉面对的并非猎王。
<br/>　　而是……
<br/>　　另一个楚涵！
<br/>　　这种风格，以及这种强势，还有毫不给人面子，也丝毫不讲道理的施压，抓·住一个点便给予致命反击的模式。
<br/>　　实在太像楚涵！

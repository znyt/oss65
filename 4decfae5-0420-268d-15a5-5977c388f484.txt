　　第716章
<br/>　　事情一点一点向不可控发展，白空集与孙别山都是武将出身，性子自然谈不上好。他们在用自己的行动向程大雷施加压力，而钱通玉则乐意看到这一幕发生。
<br/>　　必须向程大雷施加压力，让他将探出来的手脚收回去，同时也令他明白，凉州不是你姓程的凉州。
<br/>　　而程大雷恍若对身前的剑拔弩张视若无睹，他抿了杯酒，放下杯子后，才慢吞吞道：“依三位看，这件事该如何办才好？”
<br/>　　三人对视一眼，由钱通玉开口：“程当家，我们对你没有不敬的意思，我们三家已经商定好，从此以后，咱们互不干涉，你走你的阳关道，我们过我们的独木桥，井水不犯河水。”
<br/>　　程大雷轻轻拍掌：“很好，很好，那么咱们今后就这样办了。”
<br/>　　三人都愣了愣，程大雷这答应得也忒痛快些，痛快得和假的一样。
<br/>　　白空集黑着脸道：“假若你当面一套，背后反悔又该如何？”
<br/>　　程大雷眨了眨眼睛：“依你看本当家该如何，是否该写一封保证书，交给三位保管。假若我日后反悔，你们就骂我是无信之徒？”
<br/>　　呃……
<br/>　　三人同时无语，骂上程大雷几句，似乎也不解决任何问题。
<br/>　　孙别山开口道：“我们三家若联起手来，也是一道屏障，想程当家也并非战无不胜，攻无不克。”
<br/>　　程大雷点点头：“说得在理。”
<br/>　　三人又一次无语，程大雷一直顺着聊，倒像是拳脚打在棉花上，有力气无处使。
<br/>　　钱通玉抬起头，发现程大雷拎着酒杯，脸带促狭的笑意看着三人。
<br/>　　“程当家有话直说无妨？”
<br/>　　程大雷点点头，道：“请问三位，自比杨龙停如何？”
<br/>　　钱通玉一楞，同时闭上了嘴巴。
<br/>　　“比戎族大军如何？”程大雷再问。
<br/>　　“比宋伯康又如何？”
<br/>　　程大雷一连串追问，钱通玉三人都没有回答，委实也是答不上来。杨龙停曾是帝国战神，声望如日中天。钱通玉当初就算给杨龙停提鞋都不配。
<br/>　　至于戎族大军，号称天下无双，三人还没有资格与他们交手。宋伯康曾是凉州王，三人本就是宋伯康的跟班。
<br/>　　“如今，他们又在何处？”
<br/>　　钱通玉三人一激灵，目光看向程大雷有几分忌惮。杨龙停被何人所杀，戎族大军被谁人所挡，那宋伯康现在又在何处……
<br/>　　程大雷恶名远博，让人厌恶，但同样让人畏惧。在这方面，他可并不是浪得虚名，而是他一个人一个人杀出来的。
<br/>　　此刻，这尊魔头就坐在三人面前。大河流水，落日西沉，可三人却感受到一股扑面而来的血腥气。
<br/>　　程大雷又自顾自斟满酒，饮尽后把玩着酒杯，道：“你们如果比不上他们，又有什么资格坐在我面前，与我拍桌子，谈条件？”
<br/>　　程大雷抬起头，杀机毕露。
<br/>　　钱通玉三人似乎被这目光锁定住，忽然孙别山晃了晃肩膀道：“今日之凉州，不是幽州，也不是长安，凉州只是凉州。真如果两军对垒，沙场之上论高低，孙某纵然不敌，也想和程当家较个高下。”
<br/>　　程大雷忍不住拍手：“言之有理，言之有理。”
<br/>　　白空集长身而起，手摁住刀柄冷冷道：“不说别的，如果我们不答应，你能活着离开这座楼么？”
<br/>　　银眸忍不住向前跨出一步，表情依旧冷漠，却是杀气四溢。
<br/>　　程大雷挥挥手让她退下，不冷不热道：“本当家既然敢来，自然就有把握离开。”
<br/>　　程大雷扬了扬眉毛，道：“你有胆量在本当家面前拔剑么？”
<br/>　　白空集掌心攥紧，似乎随时会拔刀出鞘。此时，他的目光落在程大雷手上，不知何时，程大雷也同样摁住了剑柄。
<br/>　　这柄剑吸引了白空集的注意力，那是匹夫剑，程大雷自从出世以来，用这柄剑杀过许多人。如果自己真的敢动手，有把握比他的剑更快么？
<br/>　　自己险些忘了，面前这人不仅是头恶贼，也是天下超一流的剑术高手。他的权力抵达不了千里之外，但十步之内还是他说了算。
<br/>　　纵然程大雷不能从此地脱身，自己的人头也保不住。况且，程大雷真如果死在此地，他手下的铁骑是吃素的么？
<br/>　　山贼大多没什么文化，没文化的人大多很野蛮。
<br/>　　钱通玉这时候忙站起来打圆场，笑呵呵道：“息怒，息怒，白城主多饮了几杯，一时失言，还请程当家见谅。”
<br/>　　白空集趁势下了台阶，道：“程当家想做这凉州之主，也得有这个资格，靠以前杀过几个人吓唬人，嘿，我们也不是吃素的。”
<br/>　　程大雷将剑摁在桌上，道：“三位可认得这柄剑？”
<br/>　　三人目光同时落在程大雷身上，不知他想说什么。
<br/>　　程大雷淡淡道：“这柄剑是明帝送给我的，我和明帝他老人家的感情，那可真称得上是亲如兄弟啊。他老人家给我这柄剑，就是让我上斩昏君，下斩不臣。靠这柄匹夫剑，不知我有没有资格在凉州说了算？”
<br/>　　三人觉得程大雷大概真的吃多了酒，说话也有些胡闹。但这柄剑毕竟不是假的，而程大雷手中的兵，也能撑起胡乱扯来的虎皮。
<br/>　　“其实，这凉州之主，我做不做没有关系。”程大雷道：“我是为三位着想啊？”
<br/>　　“程当家，此话怎讲？”钱通玉。
<br/>　　“三位可知道，当今陛下要召天下藩王入京，他通知你们了没有？”程大雷问。
<br/>　　三人无语，凭三人的资格怎么会李乐天通知。
<br/>　　“不巧，他恰恰通知了我。”程大雷道：“这次八月十五，我便要去长安，摘下这凉州王的封号。三位可知道，相府如今如何了？”
<br/>　　“都说大树底下好乘凉，可崔相这棵大树说倒也就倒了。也有句话，良禽择木而栖，贤臣择主而事，三位可以不这么打算。但等我从长安回来，有名有实，三位真如果想要和我在沙场上碰一碰，赢了倒好说，可假若输了。”
<br/>　　程大雷笑看着三人：“明人面前不说暗话，我是山贼出身，脾气……一向是不是很好。”
<br/>(本章完)

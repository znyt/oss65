　　梧桐的后背，顶在了墙上，脸被迫埋在叶少阳后腰上，什么也看不见，不由问道：“蝾螈死了？”
<br/>　　回答她的是蝾螈的一声吼叫，而且从声音判断，好像在从远处迫近。
<br/>　　之前那种奇怪的安全感消失无踪，梧桐情急中问道：“你怎么不动了？”
<br/>　　“没什么好怕的。”叶少阳沉声说道，眼睛直勾勾地盯着蝾螈，似乎在等待什么。
<br/>　　妙心等人从石门出去之后，并没有走太远，都趴在石门边上朝里面观望着，墓室里点着很多油灯，因此能够大致看见墓室里的情况。之前叶少阳抱着梧桐，身手仍然十分矫捷，在蝾螈的横冲直撞之下，一次次轻松躲过，大伙不管对他态度怎么样，心中都是有些惊讶，甚至有点暗暗的佩服，本以为他或许能就这样把蝾螈耗死，没想到他突然背靠着对面的墙壁站住了。
<br/>　　谁也不知道他打的什么主意。
<br/>　　“跑啊，愣着干什么！”情急之中，妙心大声提醒道。
<br/>　　叶少阳没动，只是静静地看着冲过来的蝾螈。蝾螈的脚步已经趔趄，眼神也浑浊了，看上去是撑不了多久了。
<br/>　　也许正是因为这是生命里的最后一次攻击，蝾螈用上了体内的最后一丝力气，猛然提速，想要跟叶少阳同归于尽。
<br/>　　“太慢了。”叶少阳轻声说了一句，就算蝾螈冲到理自己三米左右远的地方，身形猛然一矮，抱着梧桐一个翻滚，逃到了几米之外。
<br/>　　蝾螈也不知道反应过来没有，就算反应过来，也根本没有时间调整方向了，一头撞在了墙上，轰的一声巨响，居然将墙壁撞出了一个大洞，半个身子钻了进去，石块掉落在他浑身是伤的身体上。
<br/>　　“大哥，再努力一把，你不能死啊！”叶少阳紧张地喊道。
<br/>　　蝾螈果然挣扎着从自己撞出的洞里爬了出来，结果脑袋刚移出来，浑身猛地一个抽搐，倒在地上彻底不动了。
<br/>　　死了。
<br/>　　这个两三米高的大块头，总算是被叶少阳给玩死了。
<br/>　　叶少阳仍旧没有放梧桐下来，走过去，从蝾螈的脑袋上拔出了一直插在上面的七星龙泉剑，用自己的衣服擦了擦，看着剑身说道：“对不住啊，回去再给你洗澡。”
<br/>　　说完把剑插回到剑鞘里。
<br/>　　这个时候，他才转过身，对着石门外吼了一嗓子，说道：“进来吧没事了。”
<br/>　　妙心等人也待要走进墓室，突然又是一阵水花掀动的声音，从那口巨大的水井里传来。
<br/>　　“靠，不是吧！”叶少阳心中升起了一丝不详的预感，结果很快这预感就成了现实：一个庞然大物，在一片水花之中从水井里跳了出来。
<br/>　　又是一只蝾螈！
<br/>　　体型上比地上躺着那只小了快一半左右，但叶少阳知道，这一头可能更不好对付：体型小，意味着更敏捷，速度更快。目标也小的多。
<br/>　　原来这水井里不止一只蝾螈！
<br/>　　其实这本是非常容易理解的事实：蝾螈能从水井下面上来，说明水井下面一定有地下水系，有一只蝾螈，多半就有两只：不然这只蝾螈总不能是石头缝里蹦出来的。
<br/>　　至于为什么只出来一只，叶少阳也不清楚，也许刚好就有一只在附近，总归是他们运气好，如果刚才打到一半出来眼前这只，那情况就不知道有多危险了。
<br/>　　这只小一点的蝾螈跳出来之后，先是环顾了一眼周围，立刻发现了那只死在叶少阳脚下的蝾螈，然后看到了叶少阳。
<br/>　　“对不住啊兄弟，这是你爹还是你哥，或者你姐姐？”
<br/>　　这种时候，叶少阳居然还开的出玩笑。梧桐趴在他肩上，气得快要背过气去，双手拍打着他的后背，嚷道：“快放我下来，一起应敌！”
<br/>　　叶少阳没理她。
<br/>　　那蝾螈突然仰天大吼，吼声中带着明显的悲伤，然后二话没说就冲叶少阳冲了过来。
<br/>　　“再见！”叶少阳冲那蝾螈吐了吐舌头，然后弯腰拨开地下那只蝾螈的脑袋，从被它撞出来的洞穴钻了进去……
<br/>　　小蝾螈也如期而至，却发现仇人钻进了洞穴里。它迫切地想要冲进去，结果……洞穴大半都被地上那只大蝾螈的尸体挡住了，它虽然是小蝾螈，但也是相比地上那只来说，比人类的身体还是要大上几号，叶少阳抱着梧桐能钻进去的洞穴，它根本钻不进去，等它用力拨开地上那只蝾螈的尸体，钻进去的时候，叶少阳两人已经不见了踪迹。
<br/>　　面前只是一条长长的洞穴，小蝾螈用力朝里面爬，只是洞穴在朝下的一个转弯之后，后面越来越窄，根本钻不进去。
<br/>　　“嗷——”小蝾螈盛怒至极，大吼一声，实在无奈之下，只好爬了回去，然后，它就闻到了石门外面还有人的气息，顿时把怒气都转移了过去，飞快地扑了过去，与妙心等人打了个照面，二话不说，厮杀起来……
<br/>　　在梧桐强烈的抗议声中，叶少阳终于把他放了下来，结果梧桐单手结印，出其不意地按在了叶少阳的额头上，只要喷吐法力，叶少阳顷刻间就会重伤。
<br/>　　“干啥你这是！”叶少阳瞪大眼睛，十分不解地看着她。
<br/>　　“你轻薄我！”梧桐愤然说道。
<br/>　　“轻薄……什么时候？”叶少阳一脸地无辜。
<br/>　　“你抱了我这么久，之前还……”梧桐对他打自己屁股的事耿耿于怀，却怎么样也说不出口，气的满脸通红。
<br/>　　“大姐啊，我是为了救你啊，你可别恩将仇报！”叶少阳摊着手，更加无辜。
<br/>　　梧桐冷冷地看着他，说道：“你不怕我杀了你？”
<br/>　　叶少阳笑笑，“我怎么说也是你救命恩人，不至于吧。再说……就算你按着我命门，你还真杀不了我。”
<br/>　　梧桐冷笑一声：“就凭你天师修为，你有什么资格这么说！”
<br/>　　“不信你就试试。”说完之后，叶少阳摆了摆手，“算了，别试了，我怕伤了你。”说着伸手掰开她按在自己额头上的手。
<br/>　　等到手放下来，梧桐才回过神来，自己为什么这么配合，把手放下？

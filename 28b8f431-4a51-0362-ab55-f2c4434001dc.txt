        &lt;h3&gt;第三卷第三百三十三章  柳暗花明又一村&lt;/h3&gt;
<br/>
<br/>        凌辰和董亚子起程飞往贵州，在机场上，他看到了杭州的新闻，他们手中的几个代理商被查了。
<br/>
<br/>        杭州某部门的大佬谷正雄在发声：“我们要严厉打击假冒伪劣产品，尤其是民生百货上的产品，这些东西是会喝死人的。”
<br/>
<br/>        谷正雄痛心疾首的说：“朗朗乾坤，青天白日，你们竟然如此的无法无天，一定要彻查这些货物的源头。”
<br/>
<br/>        凌辰对董亚子说：“亚子，你看有人替我们在行动了。”
<br/>
<br/>        “呵，真的被查了，你干的？”董亚子想不明白是谁做的。
<br/>
<br/>        “常在河边站哪有不湿鞋，他们以为他们身后有靠山，就不怕了，其实不然。”凌辰摇头晃脑的笑着说。
<br/>
<br/>        “不是，你那几天天天和宁雅谈情说爱，哪有时间去搞这些事情？”董亚子不明白了。
<br/>
<br/>        “我什么也没有搞啊，我就谈情说爱的。”凌辰卖关子。
<br/>
<br/>        “凌辰！”董亚子柳眉倒竖。
<br/>
<br/>        “其实吧，我就是怀疑有些代理商有问题，我将这些有问题的代理商的名单发给了宁少天，我是这样说的：山西凌家代理的产品有瑕疵，如果不想被牵连，可以试试看。”凌辰惬意的坐在贵宾室里喝着咖啡。
<br/>
<br/>        “宁少天直接将这个文件发给了谷正雄了？”
<br/>
<br/>        “聪明，我爸爸回家之后就没有去杭州，等于拒绝了和宁家的合作，那么宁家是不会顾忌和我们的关系了，便找到了谷正雄，让谷正雄动手了。”
<br/>
<br/>        “像宁少天这样的大企业，肯定和谷正雄的关系很好，有宁少天提议，谷正雄肯定会引起重视的。”
<br/>
<br/>        “聪明，但是不知道谷正雄能不能将这件事情继续下去。”
<br/>
<br/>        “谷正雄是甸城慈善家谷老大的兄弟，头顶上也罩着谷家慈善家的光环，你说他和何少华有没有关系？”
<br/>
<br/>        “每个地方关系都错综复杂，看似平静如水，谁知道下面的波涛汹涌呢？没有看见的事情谁也不知道真假，希望诚之在甸城有所发现吧。”
<br/>
<br/>        “我也要去甸城。”董亚子焦虑的说。
<br/>
<br/>        这些天她和刘诚之并没有联系，这让她心里隐隐不安。
<br/>
<br/>        如果所有的源头都真的从甸城出发，那么他们董家和甸城将会为了生存有一场生死较量，刘诚之会面临不可预估的危险。
<br/>
<br/>        她喜欢刘诚之，但是却不能让刘诚之为了他们家族而冒险。
<br/>
<br/>        “你去甸城只是给刘诚之拖后腿，让刘诚之举步维艰，说难听点，就是让刘诚之和何少华大打出手，正面起冲突，你不去，他装聋卖傻有可能还能调查到一点东西。”凌辰语带讽刺。
<br/>
<br/>        “你.......”
<br/>
<br/>        “我说的是真的，甸城何少华的心很大，他想娶的并不是你，而是你们家的产业，你到了甸城岂不是羊入虎口？”凌辰冷冷一笑。
<br/>
<br/>        “何家动的心思很早了，以前一直想要代理，我们推了一年又一年，想办法一次又一次让何家知难而退。去年已经是推不过去了。”董亚子看着人来人去的机场，挠了挠头。
<br/>
<br/>        她的父亲疼爱她，也算是一个开明的人，要不然按照有些家庭，估计会牺牲她的幸福和何家结亲。
<br/>
<br/>        这一次的西城之行，揭开了他们心里久存的疑团，何家想要的更多，或者是说何家其背后的人想要的更多。总代和代理确实有着本质上的区别，总代每年的分成会多，代理是有任务加提成的，但，何家的代理每年的销售额并不大，总代的位置是不能给他的。
<br/>
<br/>        何家实际上是几年前取代了甸城的一家代理商，单从完成销售额上来说，何家的销售额远远没有第一家代理商销售业绩好，但何家在当地有背景，如果想要在当地销售你的产品，还得有人支持。
<br/>
<br/>        他们这个老企业在社会的浪潮下已经今非昔比，销售业绩也是翻了几番，而且已经上市，目前形势一片大好。
<br/>
<br/>        作为董家的独生女来说，她的身价会跟着她家的产业水涨船高，盯着他们家的人不是一家，她的父亲因为在这个行业是大佬，也不肖用女儿的婚姻作为纽带，一次又一次的拒绝了很多人的提亲。
<br/>
<br/>        何家在去年让人来提亲了，不，不是让人来提亲，而是打了个电话，说是何家的小子人品不错，董亚子未婚，俩个人在一起很般配，不如让俩个年轻人相处一段时间，培养培养感情以后两家强强联合，潜力无限！
<br/>
<br/>        当董亚子的父亲给董亚子说的时候，董亚子那个时候心有所属，那就是凌辰，凌辰却是心仪宁雅，直到董亚子碰到了刘诚之。
<br/>
<br/>        憨憨的刘诚之不晓得第一次和董亚子巧合的碰触已经让董亚子动了心，董亚子又发现了刘诚之衣着的不同之处，悄悄的让她的父亲暗中调查了一下刘诚之的身份。
<br/>
<br/>        刘诚之不但是安林曦的得意门生，刘家还是举足轻重的大户人家，刘诚之的父亲性格憨直，对于不好的事情敢于发表意见，不怕得罪人，是一个正义勇敢的人。
<br/>
<br/>        这一调查让董亚子心生欢喜，最起码她喜欢刘诚之不会有太多的阻碍，她是一个敢爱敢恨的人，当时就联系了她的老师安林曦，告诉安林曦，她喜欢刘诚之，要让安林曦约上刘诚之他们一起见面。
<br/>
<br/>        当刘诚之想去甸城的时候，她极力反对，她的父亲说：“不经一番寒彻骨，哪得梅花扑鼻香？刘诚之想娶你为妻，必须得经得住考验，无论是心智还是谋略都得有过人之处，要不然我以后偌大的家产交给你们，岂不是会被你们坐吃山空？去甸城也算是对刘诚之的考验，他能搬到何家，他才有资格和你谈情说爱，要不然他有什么资本？就因为那一副好看的皮囊？好看的男子多得去了，何少华长得也不赖！”
<br/>
<br/>        董亚子无力反驳父亲的话，只好同意刘诚之去甸城，董亚子这一段时间总是在担惊受怕。
<br/>
<br/>        “你也不用担心诚之，那小子城府深着呢，在我身边好久我都不知道他的身份，这次宁少天算是帮了我们一个忙，能让诚之更能安逸点。”凌辰安慰董亚子。
<br/>
<br/>        “哈哈哈，我们在西城碰壁，没想到在杭州开花，真是柳暗花明又一村！我是对他有多么的不自信才担心他！”董亚子也觉得她的担心是多余的。
<br/>
<br/>        “不过，诚之那个憨憨可是经常撩拨我们公司的小姑娘和小年轻，这一点你得注意点。经常没事情了就给人家私信，聊得不亦乐乎。”凌辰又黑了刘诚之一把。
<br/>
<br/>        “你的话有几句可信度？”董亚子表示不信。
<br/>
<br/>        远在甸城的刘诚之身边确实是坐着美女。

　　第630章
<br/>　　程大雷待过的两个地方幽州和凉州，总的说来都不是什么好地方。凉州的气候要比幽州好一些，但也是地广人稀。简单来说，就是没有人，没有人就没有发展商业的基础。
<br/>　　而目前帝国的环境，其实也不适合发展商业。铁器、盐、铜、布匹、药材都算是硬通货。可这些货物有个共同点那就是运输不便，帝国的货币制度也发展不起来大规模商业。
<br/>　　帝国真正的好地方是江南，那里气候适宜，水道纵横又适合货物运输。戎族连年入侵，但从没有打到过江南。所以，那里的发展一直没被打断，曾经是很繁华的。
<br/>　　当然，覆巢之下岂有完卵，正义教在江南闹过一次。诸侯割据，互相间摩擦不断，现在江南也不太平了。
<br/>　　和珅的话打开了程大雷的思路。蛤蟆城的粮食不能动，那是保命用的。如果真的缺粮的话，未必不能向附近城池掠夺，来明的来暗的程大雷都擅长。
<br/>　　“车到山前必有路，事到临头我们也未必没有法子。”程大雷道：“咱们现在最该考虑的还是扩充朔方城的人口。”
<br/>　　和珅皱眉思索着这件事，因为程大雷名声的问题，这件事情的确有些难办。最后，他无奈的叹道：“大当家，你以后还是要稍微注意一下自己的名声。”
<br/>　　“现在说这个不觉得太晚么，关键是眼前该怎么办。”程大雷叹道：“名声呐名声……”
<br/>　　嘴里嘀嘀咕咕念叨着，程大雷心中忽然一动：出来混打什么旗很重要，可自己未必无旗可举呐。
<br/>　　事情还是要从林少羽说起，林少羽当初搅动风雨，也是因为他起事才造成目前帝国二帝十八王的局面。不得不说，林少羽一直记挂着程大雷，他知道自己没有帝王之才，从一开始就是想推程大雷上位，奉程大雷为圣主。
<br/>　　如今帝国有很多人揭竿而起，他们打的就是正义教旗号。正义教起义虽然被摁了下来，但正义教的信徒却无处不在。这大旗，由程大雷来扛比任何人来扛都合适，别人都能打着正义教的旗号招摇撞骗，自己为什么不可以。
<br/>　　将心意与和珅说来，和珅精神一振，双眼里目光闪烁。半晌，他道：“大当家，这件事大有可为呐，只要打出这旗号，便不用担心无人来投。”
<br/>　　“这件事还需从长计议，商量一下具体该如何操作。”
<br/>　　“操作上应该没有什么困难，不过……”和珅顿了顿：“大当家，这大旗如果扛起来，咱们就真成帝国之敌了？”
<br/>　　和珅颇有种幸灾乐祸的感觉，自己一再怂恿程大雷都不为所动。但到眼前这状况，程大雷已不得不展开行动。这便是人在江湖，身不由己的道理。
<br/>　　“咱们……好像一直是帝国之敌么，有什么区别么？”
<br/>　　“呃……”和珅忽然被噎了一下。
<br/>　　打什么旗号其实不重要，重要的是采取怎样的方阵。
<br/>　　和珅愣了愣道：“既然咱们要招揽正义教的旗号，必然需要一个口号，这口号于情于理都该由大当家来定，大当家看我们该喊什么口号。”
<br/>　　和珅看着程大雷，仿佛看到他一步步走上自己与他安排的道路上。
<br/>　　程大雷略沉思片刻，便道：“喊什么口号还用想么，就告诉天下人，我要与林少羽复仇，让天下正义教的信徒都来投奔我。”
<br/>　　在和珅的润色下，程大雷亲自定下了八个字的口号：光明普照，正义必胜。
<br/>　　和珅立刻着手开始安排，这次行动由阿喜率领的喜字队负责。阿喜撒出人手，进入各州各城，散步着一些事先编造好的消息。
<br/>　　他们大多乔装成说书先生，潜入各地的茶馆酒楼，说一些正义教的往事。只是说帝国如何如何黑暗，各路诸侯如何残暴，正义教当初就是为此才揭竿而起。另外程大雷与林少羽如何兄弟情深，当初如何拼命为他复仇。最后，才提到程大雷已在招兵买马，重新扛起正义教的大旗。
<br/>　　此外，还说天下人如何误解程大雷，程大雷如何的忍辱负重，藏书楼那把火并不是程大雷烧的呀。
<br/>　　这是一场明显的洗白计划，试图挽救程大雷已经糟糕到极点的名声。
<br/>　　这是计划中的事情，究竟今后会如何发展，程大雷还看不出来，目前也只能走一步看一步。
<br/>　　安排好一切后，程大雷忽然觉得有些无趣，本来还有许多事情要他处理，他却有些打不起精神来。起身迈步向城主府的后院走去，一个人形单影只，口中呐呐叹着：
<br/>　　“少羽呐少羽……”
<br/>　　……
<br/>　　“王上，喜事，喜事。”
<br/>　　陆洛大步走进宋伯康的书房，一边走口中一边道。
<br/>　　宋伯康皱起眉头，道：“什么喜事呐，会有什么喜事发生在我身上？”
<br/>　　“王上，这次真的是好事。程大雷已经入驻朔方城，开始处理朔方城的事情。”
<br/>　　“这件事值得这么高兴么，我们把一座城池送给他，他自然要着手经营，难道会放弃不管么。”
<br/>　　“王上如此想，那程大雷定然也是这么想的。”陆洛兴奋道：“属下刚刚得到的情报，程大雷的运粮队已源源不断将粮食运到朔方城。他想要控制朔方城，既要分兵也要分粮。那朔方城就是一个大坑，将会将他近几年攒下来的东西消耗一空。等到那时候，便是王上你拿下整座凉州，清除心腹大患的时机。”
<br/>　　宋伯康的眉头渐渐舒展开来：“一切不都和我们事先安排的一样么，贪心不足蛇吞象，程大雷目前根本没有吞下朔方城的势力，强撑着咽下去，会将他肚子撑破的。”
<br/>　　“已经占领的城池，谁会轻易让出来，他可没有王上的气魄。”
<br/>　　宋伯康扶须缓缓点头：“这件事你居功至伟，我是不会忘了的。目前朔方城有多少人？”
<br/>　　“大概有五千老弱妇孺，这五千张嘴怕会令程大雷很头疼。”
<br/>　　“五千人还是有些少，咱们该想想法子，送给程大雷一些人，让他们的消耗更大些。”
<br/>(本章完)

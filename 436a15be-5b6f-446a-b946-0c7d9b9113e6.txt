　　第368章
<br/>　　“崔弄海如此做事，便是王法不管，京城之大，目无天地的人是多的，难道就没人管么？”程大雷问出了心中的困惑。
<br/>　　“天子尚且避醉汉，我是什么身份，高高在上，坐不垂堂，岂能和一个疯子计较。”李行哉理了理衣冠。
<br/>　　程大雷看了李行哉一会，慢悠悠道：“因为你身份高贵，所以不与其计较，便看他虐杀那些身份卑贱的人。什么身份高贵，说白了不就是怕么。”
<br/>　　李行哉一怔，程大雷还是第一次用这种语气对自己说话。然而他不得不承认，程大雷这话有几分道理。对崔弄海的厌恶中，何尝没有藏着怕呢。就像京城提起程大雷，都嗤之以鼻，说他不过是个山贼而已。而这种鄙夷，何尝不是为了掩饰心中的畏惧。
<br/>　　李行哉倒颇有几分唾面自干的精神，这种臭不要脸的涵养，就连程大雷都没有。
<br/>　　“恶人自有恶人磨，如今他不是落到你手中了么。你刚才还说凉州城过来要人了，你会将人交给他们么？”
<br/>　　“呵，想什么呢，我若让他活着离开这里，我就不姓程。”
<br/>　　“可你要是不给，哪位京城的丞相怕不会善罢甘休，崔弄海虽然是个疯子，但却很得那位丞相大人的欢心。”
<br/>　　“是这样么，如此却有些麻烦了。”
<br/>　　程大雷低低笑了一声，笑得李行哉浑身冷汗。程大雷去而复返，又回到牢房中。李行哉隐隐觉察到气氛有些不对，并没有随着程大雷一起进去。
<br/>　　程大雷又差人将吴用叫来，崔弄海被吊在梁上，程大雷坐在太师椅上，面带微笑的看着他，崔弄海也回以微笑，二人就用这种姿态对峙。
<br/>　　时间一分一秒过去，吴用带着一队人进来，道：“大当家。”
<br/>　　“嗯，人都起了？”程大雷没有看吴用，视线依旧紧盯着崔弄海。
<br/>　　“都在这儿呢，全部是山寨的屠夫，刀很快。”吴用还有些糊涂：“大当家叫我来做什么？”
<br/>　　“你表现的机会到了。”程大雷抬手指着崔弄海：“你有八个时辰的时间，八个时辰后，我要让他成为一堆烂泥。”
<br/>　　吴用抬眼看看着崔弄海，双目烁烁放光。
<br/>　　“大当家，你要不要避一避，接下来发生的事情，怕是会有些血腥。”
<br/>　　“大当家还怕血腥么？”程大雷饶有兴趣道：“我就在这里看着，也向你们学习学习经验。”
<br/>　　“好。”吴用挽起袖子：“大家忙活起来。”
<br/>　　吴用带来七个人，都是满脸横肉的壮汉，在山寨干屠夫的事情。此刻在吴用的吩咐下，有条不紊的做事，烧火的烧火，忙碌的忙碌。这地下黑牢中，突然有了种热火朝天的架势。
<br/>　　吴用不愧有毒计的属性，『计』程大雷目前还没看出来，但毒可是真毒呐。用鞭子抽得崔弄海遍体鳞伤，铁梳子刮过他的身体这些事都根本是小儿科，甚至都不值一提。
<br/>　　他用利刃割下崔弄海胸口一块皮肤，逼其喝下滚烫的开水，用尖刀分开他一块块肉，再强迫崔弄海吃下去……
<br/>　　程大雷坐在太师椅上，从始至终保持着固定的姿势，一动不动。这倒不是说他有多镇定，甚至是相反，他静坐不动，是为了掩饰自己的慌乱。
<br/>　　某种程度上，这崔弄海的确给程大雷带来一些恐惧。杀死他是容易的，可杀死他未必能令程大雷克服恐惧，只有将同样的恐惧施加给对方，程大雷才能迈过这道坎儿。
<br/>　　吴用不停擦拭着额头的冷汗，便是程大雷也压抑不住心底的恐惧，更何况是他。
<br/>　　哪怕如今的崔弄海已经成了一个废人，但仍旧给几人带来巨大的压迫。刀子砍在他身上像砍在木头上，他大口大口咀嚼着自己身上的肉，脸上还带着笑容，像是吞噬着某种美味佳肴。
<br/>　　世间竟有这种人，世间竟有如此的妖孽。
<br/>　　一个操刀的屠夫突然弯下腰，不停的呕吐。这算是一个豪横的汉子，常对人吹嘘自己胆大如卵，敢在坟地里搂着白骨睡觉，可今天，他却是第一个败下阵来的。
<br/>　　或许，他今天才知道，这世间，人往往比恶鬼可怕。
<br/>　　程大雷挥挥手，令人将他拖下去，换一个人上去操刀。可这个人连一炷香都没有撑下去，便吓得两腿发软，面色苍白的昏厥过去。
<br/>　　然后是第三个，第四个……吴用一共带进来七个人，最长的撑了半个时辰。他们明明是施刑者，可被吓破了胆的却是他们，而不是崔弄海。
<br/>　　吴用越来越紧张，他来到山寨后，其实一直没得到重用。今天好不容易有个表现的机会，本来想好好露一下脸，可结果却败得一败涂地。
<br/>　　“大当家，我再叫人过来，今天非让他服了不可。”
<br/>　　“不必了。”程大雷终于站起身，道：“我自己来。”
<br/>　　“你……”吴用一怔。
<br/>　　程大雷握住那柄滴血的尖刀，一步步走到崔弄海面前，独自面对这妖孽。
<br/>　　“我刚才忽然想到一件事，他杀了很多人，种瓜得瓜种豆得豆，既然如此，何不让他把那些法子都尝一遍。”
<br/>　　崔弄海脸上表情突然凝固住，竟在一瞬间笑不出声来。他自然记得那些人是怎么死的，有人被拽出舌头，有人被挖出眼睛，更有人用一根木棍直接从头到尾串起来……这些法子，对于一个变态来说，总能刺激他的某根神经。可如果亲自经历，崔弄海终于觉察到恐惧。
<br/>　　程大雷看着他发呆的目光，抿嘴笑了笑，在崔弄海的视线中，程大雷笑得越来越残酷。
<br/>　　所以说，解铃换需系铃人，只有用妖孽的法子，才能攻破一个妖孽的意志。
<br/>　　程大雷离开牢房时，崔弄海双目涣散，像个苦苦蹂躏，生无可恋的少女。但他还没有死，不得不说他的生命力强悍。
<br/>　　而程大雷却是浑身浴血，走进他身周五米，就能嗅到一种血腥味。只有吴用从始至终看到了整个过程，他失魂落魄的跟在程大雷身后，一句话不说。
<br/>　　地牢内究竟发生了什么，他也再不曾对人说起过。
<br/>

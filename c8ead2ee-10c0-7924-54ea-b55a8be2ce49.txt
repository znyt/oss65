　　天墉城内稍微出色一些的精英弟子，谁不知道地灵丹的效果，七成的几率令筑基巅峰修士，迈入金丹期。
<br/>　　‘地灵丹’可以说是精英弟子们，梦寐以求的灵丹妙药，他们做梦都想得到一颗。
<br/>　　但是想想慕清霄的身份，送出一颗地灵丹并不奇怪，只是令他们更加的渴望，看向陵越的目光也变得不一样。
<br/>　　当然，羡慕可以，嫉妒就只能放在心里，因为陵越可是紫胤真人的首徒，未来成就必然不是他们能比的。
<br/>　　他们都只是普通或者精英弟子，没有靠山，得罪这样的人，绝对不是理智的行为。
<br/>　　将地灵丹递给陵越后，慕清霄便转身离去，唯独剩下陵越站在登记台前，望着手中丹药，愣愣出神。
<br/>　　他虽是紫胤真人的弟子，但资源也不是天上掉下来的，都需要自己去努力。
<br/>　　地灵丹对他来说，也是当下所需，心底顿时升起一股难以言喻的感激之情。
<br/>　　离开斩妖殿后，慕清霄便想着尽快前往自闲山庄，至于陵越，早就抛之脑后。
<br/>　　地灵丹虽珍贵，但也只是对于一般的弟子而言，对于他来说，顶多就是有点价值的普通丹药罢了。
<br/>　　在斩妖殿登记过信息后，慕清霄就前往凝丹堂，将近期需要炼制的丹药信息整理好，拿着灵药就返回住处。
<br/>　　半个月后，炼丹室的大门打开，一阵药香味弥漫而出，令人心旷神怡。
<br/>　　在这半个月的时间里，慕清霄都在炼丹室中，炼制丹药，总算是将后两个月规定的丹药，全部炼制了出来。
<br/>　　离开炼丹室后，慕清霄就来到大厅，将装有丹药的玉瓶交给侍女小兰后，便化作一道剑芒消失于天际。
<br/>　　片刻后，剑芒消散，慕清霄的身影就出现在天墉城主殿内。
<br/>　　“掌门真人，近两个月的丹药，都已经炼制完毕，在下需要下山一趟。”
<br/>　　“世俗中妖魔作祟，凝丹长老一路小心。”
<br/>　　涵素真人得知他要下山，满口答应，毕竟谁都有自己的私事要处理。
<br/>　　至于担忧，那更是无稽之谈，慕清霄可是练虚期修士，前往世俗根本不会有任何一丝危险。
<br/>　　“多谢掌门真人，在下告退。”慕清霄微微行礼，便离开中央主殿。
<br/>　　竖日清晨，旭日东升，慕清霄化作一道剑光，离开天墉城，向北边掠去，并没有引起任何人的注意。
<br/>　　...
<br/>　　...
<br/>　　离开天墉城后，慕清霄也不急着赶路，而是行走于山林，欣赏着景秀山河，修仙即修心。
<br/>　　古剑位面毕竟是高仙侠位面，哪怕在城镇中都不是百分百安全，更别说是荒郊野外的幽谷山林中。
<br/>　　慕清霄自然也会遇到这种情况，但好在他修为强横，一些杂鱼妖魔，皆是一道剑气就解决。
<br/>　　原本，慕清霄是打算直接前往自闲山庄，但想到某些事情，却在中途改变了注意。
<br/>　　常熟琴川是江南一座古老的城市，依山而立，运河流动，风景秀丽，城内相比别处，显得格外繁华。
<br/>　　琴川很有古韵，听起来就应该是古香古色的江南烟雨朦胧诗。
<br/>　　站在偌大的牌坊前，牌坊上刻着‘琴川’清晰的两个大字。
<br/>　　慕清霄一袭白衣，腰间挂着玉佩，星眸中光泽流转，望着近在咫尺，杨柳依依的琴川城，面露微笑。
<br/>　　半个月的路程，并非御剑飞行，而是游山玩水，慕清霄总算是来到的琴川城。
<br/>　　虽说已经不是第一次来琴川，但他依旧有些喜欢这里，走进琴川，望着道路两旁的杨柳与运河，柔和气息迎面而来。
<br/>　　街上挂着彩灯，杨柳随风摇摆，道路两旁摆着各种摊位，商贩的吆喝声，行人的欢声笑语......
<br/>　　收敛气息行走在街头，街上的行人根本注意不到他的存在，就在这时，慕清霄脚步一顿，停在了一个饰品摊位上。
<br/>　　摊位前正站着一名方年二十的女子，与一名十岁左右少年。
<br/>　　少年长相清秀，正盯着摊位上一块青色玉佩愣愣出神。
<br/>　　旋即，少年拉着女子的衣袖，抬起头来，奶声奶气的要求道：“姐，我想要这块玉佩。”
<br/>　　闻言，女子瞥了一眼玉佩，没看出何处吸引人，耐心道：“兰生，玉饰家中有许多，回家随你挑好吗？”
<br/>　　“我不，就要这块玉佩，我就要这块玉佩。”
<br/>　　吸引慕清霄视线的，并非执着的少年，也并非女子，而是摊位上那块青色的玉佩。
<br/>　　半个巴掌大小的玉佩，外形普通，与普通玉饰没有任何区别，但慕清霄却感觉到，其中有一股微弱的灵魂波动。
<br/>　　听到女子对少年的称呼，慕清霄思索片刻，低喃道：“难道......青玉司南佩？”
<br/>　　青玉司南佩，一魂一魄永相随。
<br/>　　若非与自闲山庄有关，慕清霄也不会在意，走上前去，道：“能将此玉饰给我看看吗？”
<br/>　　此时，少年与少女正在争执，店家见慕清霄容貌与着装，恐怕是那个世家公子出来游山玩水，连忙将玉佩递了过来。
<br/>　　“公子，您请随便看。”
<br/>　　接过玉佩，果然其中的微弱的灵魂波动了一下，慕清霄知道，肯定是青玉司南佩没错了。
<br/>　　这时，慕清霄抬起头来，只见刚才容貌温柔的女子正打量着他，至于少年则盯着他手中的玉饰。
<br/>　　视线跃过女子，打量了少年一眼，慕清霄面露微笑，扬了扬手中的玉佩，道：“你想要这块玉佩？”
<br/>　　少年眼中闪过一丝迷茫，微微点头。
<br/>　　“为何想要？”
<br/>　　“我......我不知道。”
<br/>　　闻言，少年欲言又止，旋即摇了摇，连他自己都不知道为何，看到玉佩的第一眼就想要买下来。
<br/>　　“兰生，别闹了。”回过神来，女子微微行礼，道：“公子，舍弟给您添麻烦了，请见谅。”
<br/>　　“无妨。”
<br/>　　说罢，慕清霄将玉佩递了过去，道：“这块玉佩不凡，恐怕与你有缘，想要就给你吧。”
<br/>　　接过青玉司南佩，方兰生幼嫩的脸颊上满是喜悦，女子见状叹了口气，道：“多谢公子。”
<br/>　　微微摇头，慕清霄便转身离去，青玉司南佩本就属于方兰生，对他来说并无作用。
<br/>　　在此处碰到方家姐弟，他并不意外。

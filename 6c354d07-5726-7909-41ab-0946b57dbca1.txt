　　第183章
<br/>　　枪名破浪，便有逆流而上的意思。
<br/>　　林少羽的枪法不如何，刚出世时，程大雷靠着三斧头就可以虐他。破浪枪讲究一力多劲，练到极致能够一枪九叠，劲力层层不穷。而林少羽学枪多载，如今也不过能做到一枪三叠的程度。
<br/>　　一个人天赋不如何，又不够努力，还想做大侠，事情就有些尴尬。
<br/>　　但今日握枪，感觉和以前不同。这柄陪伴自己许久的铁枪上面，除了湿寒似乎还有莫名的力量传递到自己体内。
<br/>　　心更稳，眼更利，力更沉。
<br/>　　一枪扎出，迎接他的是咚地一声。
<br/>　　对方用了盾牌，含义便很明显，不是要杀死自己，要的是活捉。
<br/>　　枪抵在藤盾上，陷入半寸，随之发力，一枪五叠。
<br/>　　林少羽爆发了他从未有过的实力，枪劲滔滔不绝，如大浪拍打礁石。
<br/>　　然后，洞穿盾牌，穿透敌人心脏。
<br/>　　是因为在做正确的事情么，所以能够发挥潜力，或者说隐藏属性生效。
<br/>　　夜，双方保持着默契，谁也没有做声，杀戮安静，只是偶尔有死亡前的惨叫发生。
<br/>　　通向峰顶的道路极其狭窄，所以每次林少羽面对的敌人并不多。而他的枪法又出奇的沉稳，例不虚发，很快山道上便挤满尸体。
<br/>　　但，人力有时而穷。
<br/>　　戎族前赴后继，踩着同伴的尸体前进，盾牌一层层压上来。
<br/>　　林少羽终于力竭，握枪的手轻轻颤抖。
<br/>　　大概，快结束了。
<br/>　　却没有那种死亡前的恐惧，他向山峰下看了一眼，黑不见底。
<br/>　　若自己一跃而下，会有人知道自己在这里做的事情么。或许自己也能像那些崇拜的前人一样，在说书先生口里流传。
<br/>　　某年某月某日，少侠林少羽力战戎族，不敌，力竭坠崖。
<br/>　　原来每个大侠都是属于山崖的。
<br/>　　如此或许还能换得几滴闺房小姐的眼泪。
<br/>　　或许也没人记得，似乎也不太重要。
<br/>　　林少羽放下枪，一跃而下。
<br/>　　帝国从不缺少贪生怕死的狗贼，也从不缺少视死如归的义士。
<br/>　　黑暗中，一根套索稳准狠的甩出，牢牢的套在林少羽身上，他没有坠崖，而是被吊在半空中。
<br/>　　林少羽顺着绳索往上看，一个身穿儒袍的男人抓着绳子，脸上挂着若有若无的笑意。
<br/>　　金问道。
<br/>　　北蛮部九王子，纵然也慕羡诗书乐理，但他毕竟是草原长大的男人，骨子里还是个戎族。
<br/>　　一手套索玩得干净漂亮。
<br/>　　“想死，可不容易喔。”
<br/>　　……
<br/>　　程大雷注意到了敌方的兵力增加，这让程大雷有些困惑。打不过就走，这该是戎族的行事风格才对。吃了这么大的败仗，戎族的兵力不减反增，这么想和我过不去么。
<br/>　　今日，戎族几百匹烈马出现在城下，却避在弓箭的射程范围外，很明显这不是攻城，而是要对话。
<br/>　　“请问是青牛山陷阵营游骑将军，帝国落叶城城主，幽州王印执掌者，仙人传艺的神斧将，守卫青牛山的蛤蟆大王么？”金问道。
<br/>　　程大雷心中一动：自己现在这么大名气么？
<br/>　　“不错，正是俺老程。”程大雷道：“你是那个，看你穿得这帮模样，莫非又是帝国的叛徒。”
<br/>　　“吾乃北蛮部九王子金问道，素闻程将军文武双全，义薄云天，今日一见方知闻名不若见面，见面更胜闻名。”
<br/>　　“行走江湖，最重要的就是个义字么，你这个娃娃不错，很得俺老程开心。速速退兵去吧，我今天饶你一命。”
<br/>　　“初次见面，有一份薄利相赠，待程将军收下这礼物后，或许就不让我退兵了。”
<br/>　　“呵，你这娃娃还很客气……”
<br/>　　程大雷脸上的笑容突然收敛起来，只见从戎族阵后推着一人出来。他被绑在木桩上，身上伤痕累累，手脚都被牢牢缚着。
<br/>　　林少羽。
<br/>　　“少羽！”
<br/>　　林冲已咆哮出声，手摁长枪就要冲下城去，程大雷让秦蛮摁住他。
<br/>　　“你不是说少羽不会有危险么，狗贼！”
<br/>　　程大雷脸色从未像今日一样难看，他开口，发现声音干涩得可怕。
<br/>　　“你要什么？”
<br/>　　“程将军义字当头，必然不忍心看兄弟受苦，还请程将军打开城门，放我等进去，把程将军的兄弟送还。”金问道依旧笑着，只是笑中有了寒意。
<br/>　　“谈判，是要提出对方能接受的条件，如果我放你们进来，是要让你们屠城么。”
<br/>　　“这不是谈判。”金问道摇摇头：“这是威胁。”
<br/>　　金问道看了孟子云一眼，孟子云会意，一刀狠狠的扎在林少羽腿上。
<br/>　　“三日之内，开城投降，不然我会在你面前将他千刀万剐。”
<br/>　　聚义厅，程大雷与各家头领。
<br/>　　“少羽与我们同生共死，绝不能让他落在敌人之手，定要救他出来。”
<br/>　　“可想要从戎族手中救人谈何容易。”
<br/>　　“难道自家兄弟便不救了么。”
<br/>　　“总不能向戎族投降，不然山寨所有弟兄都只有死路一条。”
<br/>　　各人发表着自己的意见，令程大雷没有想到的是，各家头领竟是主张救的居多，即便有一二犹豫者，担忧的也是救人的难度。这不得让程大雷重新评价这些山贼。
<br/>　　而程大雷心中其实明白，牺牲程少羽是最好的选择。
<br/>　　打仗总是要死人的，蛤蟆寨因为升级后提升了『治愈』技能，又一直避免短兵相接，所以保持着极低的伤亡率，可也总是有一些人死在敌人攻城时。
<br/>　　而想要救程少羽，必然需要和戎族展开白刃战，而这一直是程大雷避免的事情。毕竟攻城是戎族的短处，而野战则是他们所擅长的。一旦开战，必定要牺牲更多人。
<br/>　　一伙孩子在铁轨上玩耍，火车轰轰开来。只要你扳动岔道，火车就会驶向另一道铁轨，而另外的铁轨上，也有一个孩子。
<br/>　　为救一个，牺牲许多；为救许多，牺牲一个。
<br/>　　从数字讲是很好回答的选择题。可面对曾经并肩作战的兄弟，你真忍心看着他死去？
<br/>　　“大当家，咱们怎么办？”
<br/>　　大厅内已经安静下来，大家都将目光投在程大雷身上。
<br/>　　“人是必须救的。”程大雷斩钉截铁：“关于这个不用再谈，接下来要谈的，是如何救人。”
<br/>　　
<br/>(本章完)

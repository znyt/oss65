<br/>考核的日子很快就来临了，张大友这一队还是很受关注的。没办法，谁让十个人的队伍只有这么一队，而且还是站在了三年级升班考核的队列之中。<br/>　　<br/>　　不过张大友的队伍并没有被人关注太久，因为考核很快就开始了。<br/>　　<br/>　　考核的地方在学院的最深处，也是学院内最神秘的区域。这一区域常年处于封闭状态，任何人不得靠近，即便是院长金万年也不可以随意进出。<br/>　　<br/>　　这片区域是学院内唯一一个有官兵驻守的地方，而且官兵还是清一色的魔法师，传说这一区域是学院供奉们居住的地方。<br/>　　<br/>　　这片神秘的区域只有每年考核的时候会允许学员们来此，而且只有参加考核的学员才有资格进入。<br/>　　<br/>　　张大友等人跟随着带队老师走进了神秘区域，神秘区域并不像大家想想的那样，就是一片普通的树林，只不过在树林的深处有一座宫殿，宫殿的周围散布着几栋零星的房子，房子不大都是二层结构，并没有什么特殊。<br/>　　<br/>　　带队老师将众人领到了宫殿门前，停下了脚步，示意众人不要说话。<br/>　　<br/>　　这座宫殿给人一种神秘的感觉，从心底升起一丝敬畏。宫殿看起来已经相当有年头了，宫殿门前的台阶不知道是用什么材质制作的，明明看起来是石头，但是却散发着洁白的光芒。<br/>　　<br/>　　几分钟以后，有一名穿着古朴的中年人缓缓的从宫殿中走出。带队老师躬身施礼，并没有说话，而是转身一指一队考核队伍，示意他们跟随中年人进入宫殿。<br/>　　<br/>　　几分钟以后，中年人又出现在了门口，又有一队人跟随者中年人进入了宫殿。<br/>　　<br/>　　李振邦很好奇的是，这么短短的十几分钟的时间，已经进入了几批人，而且没有看到任何人出来，除了这名中年人。<br/>　　<br/>　　很快轮到了张大友的队伍，十个人浩浩荡荡的跟随着中年人走进了宫殿之中。<br/>　　<br/>　　宫殿之中很空旷，桌椅都没有，宫殿的最深处有五个立着的漩涡，分别是绿色、黄色、黑色、银色和金色。漩涡充满了神秘的气息，李振邦释放出精神力仔细观瞧，五个漩涡似乎能把自己的精神力吸扯进去似的，李振邦慌忙收回了自己探测的精神力。<br/>　　<br/>　　中年人诧异的看了一眼李振邦，好心提醒道：“这里面不要用精神力探查，否则有可能会引起反噬的，甚至会伤及精神力本源。”<br/>　　<br/>　　李振邦脸色微红，同时心中也有些许震惊，自己已经做的很隐蔽的，还是被发现了，中年人给他的感觉就和普通人一样，不过看样子他并不是普通人，应该是到了返璞归真一类的境界了吧！<br/>　　<br/>　　“你们应该是第一次来参加升班考核吧？”<br/>　　<br/>　　“是的，我们都是第一次来参加考核。”张大友恭敬的回答道。<br/>　　<br/>　　“难怪，你们的实力相对来说已经很不错了，不过十个人一起参加考核还是难度的。既然你们第一次来，我就简单的和你们介绍一下好了。”中年人平静的说道。<br/>　　<br/>　　“知道为什么学院不建立在帝都，而是建立在这边境龙城吗？”<br/>　　<br/>　　众人都一脸疑惑的看向中年人，显然中年人也没有准备从他们那里得到答案，所以并没有停顿继续说道。<br/>　　<br/>　　“就是因为这几个漩涡，其实你们看到的这几个漩涡是上古遗留下来的传送阵。这是整个大陆不算秘密的秘密，是属于我们卡罗帝国的未来！龙城之所以号称永不沦落的战争堡垒，就是因为这里拥有着这几个上古传送阵，所以帝国决不允许这里出现任何问题。”中年人一脸严肃的说道。<br/>　　<br/>　　“你们下面将会进入黄色的上古空间传送阵，黄色代表着的是三年级的升班考核。你们每个人一定要收好这枚玉佩，当遇到生命危险的时候，捏碎玉佩，你们就会从传送阵中传送出来。不过捏碎玉佩以后，也意味着你将在考核中失败。”中年人将玉佩分别交到这十个人的手中。<br/>　　<br/>　　“老师，那我们怎么才算是成功呢？成功以后又如何离开呢？”李振邦有些好奇的问道。<br/>　　<br/>　　“你们进去以后自然就知道了，不要以为你们从高年级的学员口中得到的信息是有用的，每一队人面对的情况都不是一样的，所以你们进去以后一切只能靠自己和队友！你们准备好了吗？”中年人脸色凝重，略带期待的眼神从十个人的脸上扫过。<br/>　　<br/>　　“老师，我们准备好了！”张大友看到大家都冲着他点了点头，沉声说道。<br/>　　<br/>　　“好吧，那你们就一个一个的从黄色传送门走进去就可以了。你们将要面对的是什么没有人知道，进入传送阵以后，一切只能靠你们自己了！期待你们胜利归来！”<br/>　　<br/>　　“谢谢老师的吉言！我们一定会胜利归来的！”张大友说完，毫不犹豫的当先走进了黄色传送阵，紧接着走进去的是李振邦，众人都走进了黄色的传送阵消失不见了。<br/>　　<br/>　　中年人眼神中充满了回忆，自言自语道：“有多久没有见到十人一起进入了，上一次见到好像还是那几个小家伙吧！好不容易见到了，竟然还是越级进入三年级升班考核！年轻真好！希望你们好运吧！”<br/>　　<br/>　　中年人说完，眼神恢复了清明，转身迈步走出了宫殿……<br/>　　<br/>　　进入上古传送阵以后，众人只觉得一阵天旋地转，然后就出现在了现在的地方。<br/>　　<br/>　　“大家都还好吗？”李振邦双脚感觉到坚实以后，将精神力瞬间外放，同时轻声呼唤道。<br/>　　<br/>　　这里似乎是一处洞穴，他们身处在洞穴的一处角落，背后是坚硬的石壁。<br/>　　<br/>　　虽然是洞穴，但是这里并不黑暗，洞穴的墙壁散发着柔和的光芒，使得洞穴亮如白昼。<br/>　　<br/>　　“我没事。”<br/>　　<br/>　　“我也没事。”<br/>　　<br/>　　……<br/>　　<br/>　　众人轻声回应着，很快聚集在了一起，十个人都完好无缺的，而且能力都没有受到任何限制。<br/>　　<br/>　　“大家小心一点儿，看看周围有没有什么有价值的信息，大家不要离开彼此的视线。”张大友简单的吩咐着，身为队长，他必须要肩负起相应的责任。<br/>　　<br/>　　众人散开，谨慎的搜寻着。<br/>　　<br/>　　“快过来，这里有个石碑！”保罗的惊呼声从通道的拐角处传了过来。<br/><br/><br/>

　　在场的武者，除了七大宗门的天才之外，也有着不少的强者，其中向着楚风眠轰击过来的力量，几乎都要不亚于剑祖荒等人。
<br/>　　这些隐藏的武者天才，如今看到了楚风眠的实力，一个个都是心中杀意暴起，要将楚风眠给轰杀。
<br/>　　漫天武技，在空中凝聚起来，化为了五彩霞光，向着楚风眠镇压而来。
<br/>　　这五彩霞光中蕴含的可是有着千百道武技，无上杀招，组成的这样一道五彩霞光。
<br/>　　“我的剑术已经大成，在这个九域中，没有任何人会是我的对手！你们还敢围攻我？正好，就以你们的精血，来为我增添力量吧！”
<br/>　　楚风眠面对这五彩霞光袭来，他的脸上甚至是突然间哈哈大笑起来，猛然之间他手中的剑锋便是出鞘了。
<br/>　　剑锋一闪，凌空斩去。
<br/>　　硬生生的斩向这五彩霞光之上。
<br/>　　只见这五彩霞光，这一道剑锋之下，突然之间从中心开始破碎，随后在下一刻之中，顿时破碎。
<br/>　　楚风眠的这一剑，居然是将这些武者一起联手之下的五杀杀招，全部都破解。
<br/>　　五彩霞光的力量，在被破开之后，全部都被剑锋剿灭，无一残留。
<br/>　　“怎么可能！单单一剑的力量，足以破开我们的联手？”
<br/>　　“他的剑术怎么会如此恐怖？这到底是一种什么剑术？”
<br/>　　“这小子到底是什么来历？这种剑术，简直是比起九域中任何剑术都要可怕。”
<br/>　　“云羽天宫！一定是云羽天宫中的剑术，现在被这小子学会了！”
<br/>　　一尊尊武者看着楚风眠纷纷惊叹一声，议论纷纷。
<br/>　　“哼！就算是你懂得了这云羽天宫的剑术又如何？今日你的灭亡是已经注定的了！我会杀了你，将你身上的一切机缘都给得到！”
<br/>　　荒圣君看着楚风眠，疯狂怒喝道，在他的面前突然一道古老的大门开启。
<br/>　　从这古老大门的背后，所出现的却是呼啸而来的远古气息，荒古大门。
<br/>　　这荒圣君打开了跟荒古联系的大门，源源不断的荒古神力，加持在了他的身躯之上。
<br/>　　“七大宗门！至高无上！楚风眠，我会让你知道，敢跟我们七大宗门抗衡的下场！”
<br/>　　荒圣君大手一挥，猛然从这荒古大门之中，一把青铜长戟，突然出现在他的手中。
<br/>　　这青铜长戟出现的一块，整个空间都是化为了一种古老的感觉，这青铜长戟上的花纹，也是古老无比，乃是荒古时代的象形文字。
<br/>　　“荒古大戟！”
<br/>　　“这是曾经，搅乱荒古的无上凶器！在荒古时代之中，不知道斩杀过多少强者的一尊凶器！”
<br/>　　荒圣君手持这荒古大戟，身上的力量都是暴涨，荒古门的力量本就是跟荒古时代沟通，利用荒古神力来。
<br/>　　这荒古大戟，便是对荒圣君来说最为完美的灵器，跟他的力量亲密无间的结合。
<br/>　　“荒古碎山海！”
<br/>　　荒圣君手握荒古大戟，猛然之间便是挥动起来，已经是向着楚风眠，猛然轰击过来。
<br/>　　这一戟轰击过来，山川腐朽，天地震动，曾经死在这荒古大戟之中的无数远古强者，他们的凶灵，都是缠绕在其中，纷纷怒喝这，加持力量，这一戟的力量，根本无法抵挡。
<br/>　　“荒古大戟？若是真正的荒古大戟在这里，也许我还会忌惮几分，但是你这不过只是凝聚出来的一个小小虚影罢了，也想要来跟我抗衡？”
<br/>　　楚风眠不屑冷哼一声，他手中祖龙至尊剑，似乎也感受到了这荒古大戟的气息，剑锋凝聚到了极致。
<br/>　　真正的荒古大戟，在传说之中，可是一件极道仙兵。
<br/>　　要是现在荒圣君手持真正的荒古大戟，那楚风眠的确是足以退避三分，楚风眠虽然有着人皇罐，但是却是破碎的。
<br/>　　远远无法跟真正的极道仙兵相比。
<br/>　　但是如今这荒圣君手中的荒古大戟，不过是以他的无上力量，凝聚远古神力而出的虚影。
<br/>　　仅仅是一个极道仙兵的虚影，楚风眠还不会惧怕什么。
<br/>　　“剑破天地！”
<br/>　　楚风眠凌空猛然一剑斩出。
<br/>　　这剑锋，跟那荒古大戟在空中猛然碰撞起来，这两股力量，一为龙族至宝，祖龙至尊剑，一为远古凶器，极道仙兵荒古大戟的虚影。
<br/>　　这二者的力量碰撞，周围不少没有来得及避开的武者，都是被卷入其中。
<br/>　　几乎是一个瞬间，恐怖的灵力便是撕裂了他们的身躯。
<br/>　　单单是这余波，让一尊碎命之境的武者来，只怕都无法全身而退，而这些生死境武者，卷入其中的结果，便是必死。
<br/>　　“给我破碎！”
<br/>　　楚风眠冷喝一声。
<br/>　　在空中猛然一声清脆的响声突然响起。
<br/>　　那荒古大戟的虚影，顿时一瞬之间，猛然破碎开来。
<br/>　　就算是极道仙兵的虚影，也不可能真正跟祖龙至尊剑的剑锋所抗衡。
<br/>　　“荒古大戟，就这样破碎了！”
<br/>　　“太可怕了！此人还是人吗？”
<br/>　　“生死境六重，足以跟碎命之境抗衡，甚至是占据优势！”
<br/>　　一些刚刚对楚风眠出手的武技，现在的心中已经都是萌生了退意，楚风眠的实力实在是太过可怕了。
<br/>　　但是更多的人，现在却是眼神中的杀意更浓。
<br/>　　楚风眠的实力越强，也就越是证明着云羽天宫传承耳朵珍贵，就算是知道现在楚风眠的实力。
<br/>　　为了得到这云羽天宫的传承，也足以是让大部分的武者，铤而走险。
<br/>　　突然之间，数十名武者同时出手，联手之下，再度轰击楚风眠。
<br/>　　这一次出手的武者，比起上一次还要多得多，那些什么玄天宗，烛炎门的弟子，也都是全部出手。
<br/>　　就算是鱼死网破，也要拼死楚风眠。
<br/>　　“既然都不知死活，那就别怪我了！”
<br/>　　楚风眠突然冷哼一声，他的剑锋再次一动，在空中足足是化为了数万道剑气，将这无数武技，全部剿灭。
<br/>　　那些出手的武者，一个个看到这一幕，纷纷想要退后。
<br/>　　可是他们还都没有来得及退后，无数剑气，已经是刺向了他们的胸口。
<br/>　　剑气横扫，一尊尊武者，便是当场陨落。
<br/>　　“碰！碰！碰！”

　　眼前的这一枚陨落心炎，只有拳头大小，其中所散发出的温度，比起他先前吞噬的那一枚，倒是远远不如。
<br/>　　如果说，慕清霄吞噬的是成熟期，那么，眼前的陨落心炎就相当于幼生期，其中所包含的能量，完全没有可比性。
<br/>　　当然，不管是幼生期，或者成熟期，毕竟都是异火，成型就相当困难，需要经历无数的岁月。
<br/>　　而眼前这朵并不算多猛烈的异火，所存在的时间，也绝对不会太短。
<br/>　　吞噬一枚陨落心炎后，他的任务也算是完成了一小部分。
<br/>　　至于幼生期的陨落心炎，对他来说，恐怕还不如储存空间中的能量珠，哪怕吞噬了，提升也没有多少。
<br/>　　不过，有了这枚异火，天焚炼气塔就能重新开启，而且效果，也不会比以前弱多少。
<br/>　　最重要的是，这枚幼生期的陨落心炎，能够逐步的变强，假以时日，绝对不会向先前吞噬的那枚一样，狂暴至极。
<br/>　　这也是先前，慕清霄说，吞噬陨落心炎，迦南学院会感谢他的原因。
<br/>　　“既然你召唤我来这里，难道不打算出来吗？”慕清霄望着骨骸，莫名其妙的说了一句。
<br/>　　话语刚落，一道璀璨的白光自骨骸内冉冉升起，旋即，白光蠕动，最后化作一道虚幻的苍老人影。
<br/>　　苍老人影一身白袍，须发皆是雪白一片，面容苍老，平淡的双眼中闪烁着精芒，且散发着一股极强的压迫感。
<br/>　　“小子，你是如何知道，是老夫在召唤你的？”
<br/>　　慕清霄道：“这枚陨落心炎暂未诞生灵智，怎么可能召唤我，先前我进入光罩时，也是你在窥视我吧？”
<br/>　　“你小子倒是聪明，不过，你明明炼化了陨落心炎，为何老夫在你身上，感觉不到一丝的气息？”
<br/>　　老者眼中满是困惑，开口问道。
<br/>　　先前他就察觉到，陨落心炎的气息消失，并且发出召唤，却没想到，来此的青年，身上没有一丝陨落心炎的气息。
<br/>　　“晚辈能够利用一些秘法，隐藏气息罢了。”
<br/>　　慕清霄微微一笑，并没有多解释什么，哪怕老者是陨落心炎上一届的主人，那又如何？
<br/>　　“......原来如此。”
<br/>　　老者闻言，微微点头，却是一脸鬼才信你的样子，当然，他现在和鬼也没什么区别，但也没有在追问下去。
<br/>　　他知道，许多人都有自己的秘密，再次追问下去，情况可能就有些尴尬了。
<br/>　　随后，白袍老者微微一笑，笑容中有着一份由骨子里散发而出的自傲，道：“老夫曜天火，别人有时也称我为天火尊者！”
<br/>　　说到‘尊者’两字的时候，老者声音特别提高了一些。
<br/>　　然而，慕清霄依旧面色平淡，微微点了点头，他早就知道老者的身份，又何来震惊一说。
<br/>　　更何况，尊者又怎么样？
<br/>　　望着慕清霄面不改色，仿佛没听见一样，自称曜天火的白发老者，神情一愣，目光有些古怪。
<br/>　　那表情就好似在问，你小子怎么一点都不震惊？
<br/>　　“咳咳，其实你也不用害怕，那只是以前的我，现在的我，只是一缕灵魂而已，以你的实力，足以抹杀我。”
<br/>　　说着，天火尊者又道：“你想要这枚陨落心炎，对吧？”
<br/>　　“不错，确实有些用处，但也不是非要不可。”慕清霄语气极其的平淡，脸上也没有任何的情绪。
<br/>　　他那平淡的话语，使得天火尊者眼角抽了抽，暗骂后者狡诈，一番话下来，居然让他进退不得。
<br/>　　“哎，小子，虽然老夫现在只是一缕灵魂，但除了这枚陨落心炎，好东西还是有不少的。”见慕清霄油盐不进，天火尊者也只能开始诱惑起他来。
<br/>　　慕清霄瞥了一眼这不要脸的老头子，功法斗技，对他来说都没用，但好歹要给人家一个台阶下不是？
<br/>　　“前辈有什么话就直说吧，不必拐弯抹角。”
<br/>　　“咳！”
<br/>　　话语刚落，天火尊者差点被呛死，干咳一声掩饰尴尬。
<br/>　　“当年，老夫寻至此处，首次发现了陨落心炎的踪迹，后花费几年的时间，方才侥幸将它收服，而我所收服的陨落心炎，正是被你炼化的那一枚。”
<br/>　　听得天火尊者说起往事，慕清霄洗耳恭听，并没有打断他。
<br/>　　“老夫仗着有几分本领，以往天下大可去得，因此在收服陨落心炎后，却并未就此离开，反倒是好奇的进入了这片岩浆世界中......”
<br/>　　天火尊者唏嘘不已，语气中包含着些许叹息，似乎是在为当年的鲁莽有些悔意。
<br/>　　“在深入岩浆世界后，我当时再度找到一枚陨落心炎的异火种子，这令得我相当欣喜。”说着，天火尊者再度叹息一声。
<br/>　　“也正是因此，由于老夫的贪念，想要将两枚异火融合与体内，不仅未能融合，反倒是落得个重伤的下场，”
<br/>　　闻言，慕清霄倒是理解，将两枚异火融与体内，不重伤才怪。
<br/>　　“而就在老夫重伤之际，忽然就遭到了袭击...”
<br/>　　“前辈是说外面的蜥蜴人？”
<br/>　　“看来，你也与它们接触过了。”闻言，天火尊者有些诧异，没想到面前的小子，碰到蜥蜴人，居然像没事人一样。
<br/>　　“那些蜥蜴人，似乎原本就存在于这里，数量极多，而且其中不乏足以匹敌斗尊的超级强者，那一次，老夫便不幸的遇上了一名蜥蜴族的斗尊强者。”
<br/>　　说着，天火尊者眼中略有些惊人色彩，似乎当年的那一场大战，令得他记忆犹新。
<br/>　　“若是巅峰状态，老夫自然不惧，但两枚异火，使得老夫重伤，以至于陨落，好在蜥蜴族强者不懂空间之力，因此也难以寻见老夫藏匿之所。”
<br/>　　原来是这样，慕清霄瞬间明白，难怪斗宗级别的强者，站在不远处，都难以找到这里，原来是不懂空间之力。
<br/>　　“在老夫陨落不久，肉身也随之崩溃，只能将陨落心炎释放，灵魂则附于这枚异火种子上，想要借此保存，但多年下来，灵魂也渐渐支撑不住了。”
<br/>　　天火尊者看了眼慕清霄，接着说道：“若是两年内，没有人进入这里，或许老夫将真正的陨落。”
<br/>　　慕清霄自然懂他的意思，实力到了这种层次后，哪怕肉身崩溃，灵魂也能借助媒介存活下来，比如异火。

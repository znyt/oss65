　　诛仙剑阵随着阴阳老祖的自爆被攻破，到了此时，这场战争就这剩下了鸿钧老祖与一个神秘的混沌道君。
<br/>　　只见鸿钧道人身形狼狈地手持先天至宝盘古幡从诛仙阵门之中走了出来，善恶两尸已经被他收了起来。现在已经用不到他的善恶两尸了，先不说罗睺经过阴阳老祖的自爆绝不好受，况且此处还有一个更加神秘的混沌道君。
<br/>　　混沌道君脸色不变，依旧是来时那副模样，只是眼睛紧紧盯着魔祖罗睺。
<br/>　　“噗！”
<br/>　　罗喉此时的样子极为凄惨，十二品灭世黑莲已经被打落在地，他双手拄着弑神枪，单膝跪地，口中一口黑血喷出。
<br/>　　他知道，自己已经败了，败在眼前这个突然出现的神秘混沌道君身上。
<br/>　　他做梦都不会想到，洪荒除了他知道的几个大能外，竟然还会有他不知道的存在，而且这人还会来帮助鸿钧一方，真是人算不如天算！
<br/>　　而且，他也没有想到阴阳老祖能够如此决绝，竟然拼了命的出手，就算自爆，也要让自己重创。
<br/>　　阴阳老祖虽然实力不如鸿钧道人以及混沌道君，但好歹也是堪比准圣后期的存在，这一自爆，发挥出来的实力简直远超准圣巅峰的力量，这种力量远不是准圣圆满的罗睺所能抵挡的！
<br/>　　“哈哈哈！”
<br/>　　突然间，魔祖罗睺大笑而起，抬头仰天，大叫道：
<br/>　　“吾，罗睺，一生无愧于心，今朝败于尔等之手，吾心不甘！但，力不如人，吾无怨无悔，只叹天道不公！”
<br/>　　接着，他冷冷地看了一眼鸿钧道人，道：
<br/>　　“鸿钧！今日就算我死，也绝不会让你好过。大道在上，今我罗睺，愿以身补道，立魔道，从今往后，魔涨道消，道消魔涨！”
<br/>　　“轰隆隆！”
<br/>　　话音刚落，一阵巨大的轰鸣声响起，紧跟着阴阳老祖后，魔祖罗睺也自爆了，他的自爆，可不是阴阳老祖可以比的。不说他的修为比阴阳老祖高了许多，就是他如今自爆也没有开始时的诛仙剑阵的抵挡，其破坏力可想而知。
<br/>　　罗睺的自爆，以庞大恐怖的力量，更是直接摧毁了洪荒西方的大部分灵脉与山川。
<br/>　　龙汉大劫之中，洪荒西部的根基就被毁了许多，到如今还没有被修复，现在更是加上罗睺的自爆，从今往后，这西方大地怕是再也难以恢复了。
<br/>　　“罗睺！”
<br/>　　鸿钧道人恨恨地看着眼前破碎的西方大地，现如今罗睺死去了，可是这西方的因果却要落在他的身上啊，如此庞大的因果，这是要到何时才能还清啊！
<br/>　　而且，那罗睺自爆之前的起誓，让天地之间的魔道大涨，今后修士的修炼以及渡劫，可是少不了心魔等魔道的影响啊，这对于他想要执掌天道来说，更难了一步。
<br/>　　唯一的好消息就是，大劫总算落幕了！而且，他鸿钧算是最后的赢家。
<br/>　　哎！罢了，罢了，这因果今后再想办法了结吧，如今还是先回玉京山融合三尸证道成圣再说。
<br/>　　鸿钧道人心里默默道，显然，经过此番大战，鸿钧道人的道行大有突破，对于如何融合三尸成圣，他已经有一定把握了。不过，在这之前，他还要解决眼前的麻烦，要知道，眼前不只有他一人，还有一个神秘的混沌道君。
<br/>　　鸿钧道人仔细打量着眼前的青年道人，心中异常吃惊，以自己如今的修为，竟然看不透此人的修为，算不出他的根基，这还如何了得？
<br/>　　而且，感受到对方体内透出的混沌气息，强悍无比。他隐隐有些感受到，眼前此人可能踏入了第三步，成为混元大罗金仙了。
<br/>　　“此人实力强横，绝对是隐修的大能人物，不好与之交恶。”
<br/>　　鸿钧老道一边打量着对方，一边心下暗道。
<br/>　　同时，心中吃惊不已，洪荒，真是卧虎藏龙啊！
<br/>　　鸿钧在打量混沌道君，而混沌道君则看了一眼鸿钧之后，就不再看他，而是大手一挥，却是将罗睺的十二品灭世黑莲与乾坤老祖的乾坤鼎收起。他此番来此的目的就是为了这两件先天灵宝。
<br/>　　元始本尊看上的不是两件灵宝的威力，这两件先天灵宝都没有达到先天至宝级别，对于元始来说，却是作用不大，他看上的是两件灵宝的作用。
<br/>　　十二品灭世黑莲，与元始的十二品净世白莲正好相反，一黑一白，一灭世一净世，合该为他所用。而乾坤鼎的作用返后天为先天，对于元始的炼器一道，很有帮助。
<br/>　　“嗯？”
<br/>　　看了看眼前的灵宝，混沌道君想了想，再次将弑神枪也收了起来。
<br/>　　五件先天至宝，鸿钧道人一人就占了盘古幡、诛仙剑阵、太极阴阳图三件，自己拿一件弑神枪也不算为过。况且，混沌道君表示，剩下的灵宝自己一件都没有拿，这些足够鸿钧老祖用的了。
<br/>　　要不是看在这些灵宝以后会是他们三兄弟的法宝，他说不定还会再取走几件。
<br/>　　“道友，这三件灵宝从今往后就属于本道君了，你没意见吧？”
<br/>　　混沌道君收了三件灵宝后，对着鸿钧道人淡淡道。
<br/>　　“道友说笑了，今日还要多谢道友前来相救，要不然贫道就麻烦了。此三件灵宝合该为道友所有。”
<br/>　　鸿钧道人虽然有些心疼，眼前的灵宝之中，那被取走的三件都是极好的，他当然不想让眼前这人全都取走。但，看了看对方的实力，再看了看其余的灵宝，自己也算不亏，还是同意了对方取走三件灵宝。
<br/>　　“如此甚好！”
<br/>　　混沌道君看了一眼鸿钧道人，心道，算你识趣，要不然还得麻烦本道君出手。
<br/>　　“此番多谢道友相助，贫道鸿钧，不知道友如何称呼？”
<br/>　　鸿钧伸手一抓，将剩下的灵宝诛仙剑阵、太极阴阳图、乾坤尺、灵剑、擎天棍等灵宝全都收走，对着混沌道君微微一笑，客气道。
<br/>　　“本道君号混沌道君，一直在混沌之中修行。”混沌道君淡淡道。
<br/>　　“原来是混沌道友，幸会！”
<br/>　　“鸿钧道友，贫道此番大战小有所得，还要回去整理一番，就此告辞。”混沌道君道。
<br/>　　“大善！”鸿钧自然不会反对，他自己巴不得眼前此人赶快离开，自己好回去参悟大道。但脸上却是笑着说道：
<br/>　　“贫道亦要回山，闭关领悟大道，道友后会有期。”
<br/>　　说着，架起一朵祥云，往东方飞去。
<br/>　　混沌道君，则往东海之滨飞去，在去混沌之前，他需要将三件先天灵宝交给本尊。

　　居然还有一只鬼！！
<br/>　　叶少阳心头震骇，万万没有想到会是这个结果，然而后面发生的事情更是让他震惊：又有一只鬼魂从婴儿体内飞了出去，两只鬼魂，在空中幻出两个婴儿的模样，咧开大嘴，露出一口尖牙，朝着叶少阳扑了过来。
<br/>　　叶少阳心下一惊，急忙后退避开，右手摸出了一把铜豆子，砸了过去。
<br/>　　他这么一动手，之前结的印无以为继，那一只被铸母大钱困住的鬼魂也冲破了束缚，飞了出来。
<br/>　　三个婴儿，长的一模一样，身体黝黑发亮，穿着红肚兜，绕着叶少阳转圈，最后停下来，一个趴在床上，一个落在窗户上，另一个落在墙上的神龛上。
<br/>　　三个婴儿，望着被围困在中间的叶少阳，一个个发出了咯咯的笑声，笑得十分难听，带着一种嘲讽。
<br/>　　刘四夫妻，还有刘老头，肉眼看不见鬼，只能听到层层叠叠的鬼笑，骇然至极，吓的三个人都缩在墙角，一动不敢动，傻傻地看着叶少阳。
<br/>　　叶少阳表情没有任何变化，虽然他看到的场面比他们几个要恐怖的多：三个小鬼，各自占据了一个地方，将自己围在中间，咧着大嘴笑着，每一个都是两眼发白，没有黑眼珠，两排尖尖的小牙齿，长在胖嘟嘟的脸上，更是充满了一种强烈的对比。
<br/>　　这些真实的恶鬼，模样远远超过恐怖片导演的想象。叶少阳相信，这种婴孩形态的恶鬼，要是出现在恐怖电影里，什么都不用干，只要对着镜头这么一笑，足以吓坏一帮人。
<br/>　　叶少阳倒是不怕，他左手捏着一把灵符，右手插进腰带里，目光在三只恶鬼身上流转，心中分析着局势，思索着应对。
<br/>　　这三只鬼魂虽然看上去恐怖，不过身上隐约泛着绿光，说明等级是恶鬼，按照法术界定的等级，恶鬼属于第三等级，在幽魂和怨灵之上，在厉鬼之下。相比一般的鬼魂，能在日头之下出现的时间更长，且能喷吐鬼气。
<br/>　　若在以前，自己法力全盛时期，别说三只恶鬼，就算是三十只，对他来说也只如蝼蚁一般。不过自己的法力只相当于一个方士。对于一个方士来说，对付三只恶鬼，还是比较危险的。
<br/>　　在之前的较量之中，那三个恶鬼也算是摸到了叶少阳的实力，因此脱困之后，并不离去，而是想要杀了这个多管闲事的法师，之所以没有动手，他们也是在寻找叶少阳的破绽。
<br/>　　叶少阳心中清楚，邪物就像野兽一样，如果数量上占据优势，绝不会贸然进攻，而是用各种恐怖的形象迷惑和恐吓对方，一旦发现破绽，会立刻群起而攻。
<br/>　　三只鬼魂围着叶少阳缓缓旋转，同时口中不断发出爹爹怪笑，时高时低，意在分散叶少阳的注意。
<br/>　　当有一只鬼魂移动到身后的时候，叶少阳立刻紧张地转身，避免有鬼魂在自己身后。然而那三个恶鬼转动的速度也是时快时慢，几圈下来，叶少阳立刻有点狼狈，没有跟上它们转动的节奏，有一只鬼魂绕到了他的背面。
<br/>　　“哇”的一声怪叫，身后那只恶鬼扑了上来，叶少阳急忙转身，从背包里拔出了桃木剑，口中念咒，对准刺了过去。
<br/>　　那小鬼看到桃木剑，嘴角露出一丝诡笑，扑上去，双手合起来，一下子抓住了剑锋，张口吐出了一口鬼气。
<br/>　　叶少阳左手一弹，凌空打出了几张灵符。都是普通的黄符，现在的他，连代表天师的紫符都不能用了，更不要说暗金神符。
<br/>　　灵符燃烧，释放出的灵力，暂时抵挡住了鬼气，对方那小鬼却没有进一步的行动，叶少阳听见背后有动静，转头一看，是另一个小鬼张牙舞爪地扑了上来，一前一后夹攻自己。还有一只，飞到半空中，张着双手，十根手指好像利剑一般，朝自己头顶刺了下来。
<br/>　　这三个小鬼虽然实力有限，但是聪明绝顶，十分懂得配合，他们等到了一个机会，从三个不同的角度对叶少阳展开猛攻。
<br/>　　三只厉鬼，围攻一个方士，还是这种完全占优的局面，叶少阳看上去已经完全陷入了死局，三个小鬼也露出了得意的微笑，在他们眼中，叶少阳已经是一个死人了，他们想不到叶少阳有什么办法可以从这种情况下逃生。
<br/>　　“想杀我啊！”
<br/>　　叶少阳冷笑一声，右手突然松开了桃木剑，插进腰带，把阴阳镜摸了出来，咬破舌尖，喷了一口血在镜面上。
<br/>　　“阴阳相生，循环天理，天理不休，万劫不服！急急如律令！”
<br/>　　叶少阳左手在镜面上用力一拍，镜面一闪，一道灵光溢出，照向对面那个小鬼。对于身后和头顶上空那两只鬼魂，他完全不管，也用不着管。
<br/>　　这就是他的绝地反击的计划：全力进攻面前的小鬼，一招祭出，要几乎完成瞬杀，但又不能杀死，他早已看出这三个小鬼似乎同根而生，关系自然不一般，如果其中一个遇险，在情急之中，另外两个小鬼一定会放弃杀害自己，而去保护他们的伙伴，帮他们抵挡阴阳镜的灵力。
<br/>　　然而，叶少阳自己虽然没什么法力了，实力只相当于一个方式，但他手中的阴阳镜却是九段光的法器，只要一击击中，叶少阳相信自己能够完成瞬杀，甚至一下杀死三个，那可真是彻底的反败为胜。
<br/>　　绝地反击，极限而生，一向是叶少阳的特长，越是危险关头，他表现得越是冷静，越是能够想出各种奇诡妙计。就算阴阳镜一击杀不死三个小鬼，也能够重伤他们，至少自己是化险为夷了。
<br/>　　他预判地没有错，他身后和额头顶上空的那两个小鬼，一眼看到阴阳镜的灵光，立刻敏锐地察觉到了危机，确切说就算没要察觉到危机，他们也一定会放弃进攻叶少阳，去保护自己的同伴。
<br/>　　因为鬼怪也是有智商的，面对这种绝境，被困在中间的人，必定会是想方设法逃生，像叶少阳这样非但不跑，反而全力进攻，其中必然有猫腻。他们也不会冒险。

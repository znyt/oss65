　　“我还有一个问题。”忽的，顾凉晨问道：“没有修罗战斧在手的话，哪怕是已经被认主的我们老大楚涵，也无法伤到任何神秘家族的人对吧？”
<br/>　　“当然。”叶永阳严肃道：“不仅是伤不到，连碰到衣角都不可能，星河法则是绝对的，在那些宇宙贵族身上，任何诡异的状况都有可能发生。”
<br/>　　“衣角也碰不到……”顾凉晨愣愣的重复着这句话，而后下意识反问：“更不可能踹到屁·股？也不可能被揪起衣领什么的？”
<br/>　　叶永阳皱起眉，一脸的‘你在说废话’的表情，更搞不懂这个小子问这些奇怪问题干什么。
<br/>　　看到叶永阳这幅表情，陆玟丞也傻了一般的喃喃出声：“碰不到……踹不到……”
<br/>　　“你俩到底在瞎想什么？”叶永阳有些不耐烦。
<br/>　　他忽然觉得狼牙的这帮年轻人，是不是都脑子不大正常？
<br/>　　神秘家族的人，当然不可能被如此对待，这个星球上唯一能做到的只有制裁者，但制裁一门也是依靠了特殊武器，并且还是回炉一门赐予他们的，一代又一代的传下来，经历了千年之久，其过程里也有过无数次与神秘家族之间的明争暗斗。
<br/>　　那四大神秘家族，当然不肯被制裁者一直压制，用过无数种办法想要将这件武器夺走，只是每一次都失败不说，还连那件武器到底是什么，长什么样都不知道。
<br/>　　除此之外，如果还有人可以对神秘家族的人做出这种事，并且不依靠修罗战斧，那这个人本身就一定是高等生命的存在！
<br/>　　所以说，这根本就是不存在的……
<br/>　　此时的顾凉晨和陆玟丞对望了一眼，彼此眼中都透露着各种复杂和惊喜的情绪，很多事情不能说破，但也不言而喻！
<br/>　　“他什么时候醒？我还有很多话想问他。”叶永阳有些急切的看着楚涵，眼中的神色带着亲切，在他看来楚涵就是叶默的传人，也是他们叶家认定的人。
<br/>　　顾凉晨和陆玟丞都没有说话，这个问题他们回答不了，也下意识的想要逃避。
<br/>　　只是在这三人的谈话间，他们并不知道躺在他们面前的楚涵，虽然紧闭着双眼也无法动弹，思维却一直在活跃，更将他们的对话听的一清二楚。
<br/>　　“握草？原来老子的修罗战斧这么厉害！难怪当初叶默欲言又止，还一副不想给的样子，不过这个老头是谁，妈的谁允许把他带进来看老子昏迷的挫样的？”
<br/>　　“不过这货好像知道挺多啊，不会是叶家人吧？顾凉晨把人找来了？”
<br/>　　“他们刚才是在说星河法则？神秘家族之人衣角都碰不到？”
<br/>　　“那高少辉被老子踹了那么多次屁·股是什么鬼……”
<br/>　　眼前是一片狭窄黑暗的空间，很压抑，也安静的可怕，楚涵就这么叨叨絮絮的说着，却谁也听不见。
<br/>　　自从昏迷之后的不久，楚涵就感觉自己一直待在这片空间，怎么都出不去，他可以听到周围的谈话声，但却没有任何感觉，更无法感受到身边有没有人，唯一的感官就只剩下了声音。
<br/>　　这与第一次被精神攻击时的状态完全不一样，让楚涵在这片黑暗空间里暴躁异常，更压抑的快要疯掉，恨不得冲出去杀个痛快。
<br/>　　可随着封闭的时间越来越长，楚涵倒也习以为常，开始自娱自乐起来，暴戾的本性渐渐变得有些会苦中取乐，长期黑暗的心理在极度崩溃之后，反而有了对生命的另一种看法。
<br/>　　他知道白允儿来过，也知道尚九娣每日都来，就是不知道外面怎么样了，也没人跟他说。
<br/>　　楚涵并不知道自己这是怎么回事，但却可以推测一二，加之白允儿来时与尚九娣的对话，他便清楚自己能做的只有等，等到某一天重获自由。
<br/>　　屋内的谈话声不在，在听到房门被关上后楚涵就知道那些人走了，接下来的便又是长久的寂静，他能听到远处鸟鸣从木质的小屋细缝里穿透而来，从一开始的迷糊到眼下，楚涵已经能分辨出十几种不同类型的鸟类叫声，并从它们的声音强度里分辨距离这里多远，体型多大。
<br/>　　“老子的听力，都快赶上十阶巅峰了。”楚涵自嘲起来，闲来无事他也只能将唯一的感知锻炼起来。
<br/>　　望着眼前的一片漆黑，楚涵伸出手，触摸·到的是一片冰冷的平面，这片空间小的可怜，只够他一个转身和坐下，估计也就一平米的面积，高度大概两米左右，像个棺材，封印着他的灵魂。
<br/>　　但在每日一次又一次的触碰之中，楚涵也隐隐感觉到了什么。
<br/>　　“这像是某种精神类的能量实质化。”他的声音只有自己能听到，却说的一本正经：“带着让人厌恶的黑暗属性，应该不属于人类，难道是来自于丧尸么？”
<br/>　　“丧尸强大到这个地步，攻击到我的时候我一点没能察觉到就中了招，难道是丧尸王啊？”楚涵声音带笑，好似开玩笑一般，但在这片漆黑之中如果有人能看到他的眼睛，便会发现他在说出这番话的时候，眼神里的唯一情绪便是认真，并且还是认真异常。
<br/>　　“不对。”忽的楚涵再次出声，又一次伸出手触碰着那面平滑冰冷，严肃道：“排斥的感觉来自于我本身，但……”
<br/>　　但是他那个无法打开，却一直没有失联过的回炉系统，却对这股能量发出了隐隐的探索之意，这样的感觉楚涵并不是第一次感受到。
<br/>　　第一次寻找回炉碎片时最为强烈，而后每当找到回炉碎片的时候，探索便会转变为吞噬的欲望，这种感觉也让他熟悉至极。
<br/>　　而除了回炉碎片，回炉系统也对水族的圣物海蓝之心，发出过同样但却淡了许多的探索之意，带着好奇和一丝探究感，楚涵不知道该如何形容那种感觉，不强烈，也不如想要得到回炉碎片时的霸道，还有点看不起和勉强接受的样子？
<br/>　　“难道？”
<br/>　　楚涵神情复杂又严肃，收回手不再多说一句话，大脑却高速运转起来，不懂闪动的双眸冒着光，没有人会知道他此刻在想什么。

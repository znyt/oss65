　　　　《峡谷之巅，十王者的巅峰对决》——时代周刊
<br/>　　《史诗级对决，极限与极限的交锋》——一周电竞报
<br/>　　《大魔王极限闪现，带队赢下王者战》——每日电竞报
<br/>　　《颠覆英雄联盟比赛方式的教练战》——电竞周刊
<br/>　　几乎一瞬间，这些仿若早就已经拟好的头条便是全部在各个网站上表表了出来，几乎只有能够数的上名的网站，在他们的头条位置上，都有着类似的报道。
<br/>　　场上，当红队的比赛室打开之后，当看到叶帝一行人从比赛室内走出来的时候，现场的欢呼声越发的激烈了起来，到了这个时候已经不知道有多少人因为嘶吼而使得嗓子变的嘶哑了起来。
<br/>　　赢下比赛，叶帝六人自然是要去蓝队握手的，当然了就是互相调侃一番，都已经相处这么久了，虽然他们输了比赛，心里可能会有一些不舒服，但是肯定不会因此而嫉恨自己等人的。
<br/>　　而且就算是输了，他们肯定也会电竞史上留名的，而且还不是不好的名声，能够打出这么精彩的比赛，场上其实已经是没有输的人了，大家都是胜利者。
<br/>　　进入蓝队比赛室的时候，金相赫几人早就已经收拾好了情绪，笑着和叶帝他们打着招呼。
<br/>　　“卧槽啊，你这小子最后那一波是什么反应啊，心态崩了，夏季赛没法打了！”进入比赛室的一刹那，蒋念便是怪叫的喊道。
<br/>　　“我从那个比赛室特意观察你的表情来着，你表情一动，我就立刻闪现了！”叶帝调侃着说道。
<br/>　　几人笑谈了一会，便是向着舞台上走去，今天虽然没有奖杯的发布，但是他们还是要感谢现场这些观众的热烈支持的。
<br/>　　“哇，今天这一场比赛真的是精彩，大招流对阵全球流，最后一场更是极限和极限的对决，今天的比赛双方都是胜利者。”此刻米粒和林雅楠两人已经是拿着话筒来到了舞台中央。
<br/>　　看到叶帝他们出现的一刹那，米粒便是略带感叹的说道。
<br/>　　“没错，这一场称之为世纪对决一点都不为过，而且到了现在我终于明白大魔王给我们带来的惊喜了，这个教练战果然名不虚传，第一场比赛那一波教练指挥两个打野在上路草丛同步回城，这个真的是太惊艳了。”林雅楠跟着开口说道。
<br/>　　一旁的李景禹和金在贤两人听着林雅楠的话，眉头都是一挑，随后一脸便是一脸笑意的看着对方，当然了金在贤这边还是叶帝给他翻译的。
<br/>　　之前比赛的时候李景禹和金在贤可都是以为自己把对手给阴到了，让自家的打野先回城一步，然后好让对方的打野吃线上经验呢。
<br/>　　经过林雅楠这么一说，他们才知道两人的动作几乎是同步进行的，这倒是让他们有种惺惺相惜的感觉。
<br/>　　接着林雅楠和米粒两人又对着几人采访了一番，临近着采访就要结束的时候，林雅楠两人带着的耳返却是突然传来后台导播的一道话语，听到导播的这个话，两人都楞了。
<br/>　　“大家不要走，我们得到最新消息，华夏火神总部负责人已经感到现场，接下来要有重大新闻发布。”看着现场的观众已经有要退场的趋势，林雅楠连忙开口喊道。
<br/>　　听着林雅楠的话，无数准备退场还有退出直播间的观众这一刻都是一愣，不过听到华夏火神总部负责人的时候，大家心里都是一动，这个时候对方过来，大家心中都已经有了一丝猜测。
<br/>　　就是连场上的叶帝等人听到林雅楠的话，也都是有些发愣。
<br/>　　“华夏总部负责人，那不就是你二叔么？你通知你二叔来的？”蒋念有些好奇的问道，参加过叶帝的婚礼，他自然是知道叶帝的身份的。
<br/>　　“没有啊，不过估计是教练战那个事有着落了。”叶帝笑着说道。
<br/>　　听着叶帝的话，一旁的几人眼睛都是一亮，既然叶帝没有通知，那么对方这个时候过来，那也就只能是因为这个事了。
<br/>　　片刻后一席西装的叶天江便是笑着来到了舞台上方，那样子看起来到真的是有些英姿飒爽的感觉，就是连叶帝心中也都是暗赞一声。
<br/>　　不过这种公开的场合，他可不会上去主动叫二叔的，他这身份能隐藏一段时间就多隐藏一段时间，他可不想因为这件事而整天被那些媒体争相报道，来消费他。
<br/>　　那样的话不仅自己烦，就是连他家里说不定也会受一些流言蜚语什么的，总之叶帝很讨厌麻烦。
<br/>　　“经过火神总部讨论，正式决定下个赛季将增加教练战，同时由衷感谢十王者为英雄联盟做出的巨大贡献。”叶天江拿到话筒后，便是开门见山的说道。
<br/>　　虽然众人早就已经有所预料，但是当真听到这句话的时候，还是有些激动。
<br/>　　现场再一次响起了激动的欢呼声，直播间里弹幕也是被瞬间刷了起来。
<br/>　　“6666666666666，火神这个反应速度真的没毛病，这比赛才刚打完，那边就已经讨论完了，给点个赞。”
<br/>　　“明年的比赛可是真的有意思了，以后可不是队员操作牛逼就能够夺冠了，没有教练那可是不行的。”
<br/>　　“如果荣耀蓝白两队明年再战，白队这边还没有教练的话，我敢打赌蓝队绝对是要血虐白队的。”
<br/>　　“哈哈，这下大魔王估计心里要郁闷死了，早知道就不给蓝队这个教练了。”
<br/>　　听着叶天江的话，直播间里更多的人则是在调侃起了叶帝，当然了大家也并不是对叶帝有敌意，他们只不过喜欢用这种弹幕来表达自己对叶帝的喜爱。
<br/>　　听着自己二叔的话，虽然心里有所准备，但是叶帝也是被火神这反应速度给吓了一跳，没想到火神会这么关注自己等人的比赛。
<br/>　　不过细想一下也对，毕竟这是十王者战，就是连官方都给下了一个这是当今世界最巅峰的一场王者战，也是唯一的一场十王者战的定义，可想而知它的轰动程度了。

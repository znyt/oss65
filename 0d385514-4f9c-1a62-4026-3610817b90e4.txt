　　闻言，慕清霄微微一愣，没想到还是一只见多识广的梦魔，他布下的虽不是结界，但也与结界相似。
<br/>　　领域！
<br/>　　在这片领域中，他就是主宰，除非梦魔能一击轰碎他布下的领域，，否则绝无逃出去的可能性。
<br/>　　若梦魔真有如此实力，怎会躲在这乱葬岗中苟且偷生。
<br/>　　古剑奇谭可是高级仙侠位面，这里宗门何其之多，强大的修士数不胜数，弱如蝼蚁，强如剑仙，更有虚无缥缈的天界。
<br/>　　梦魔虽是可怕的存在，但这也是在成长起来的前提下。
<br/>　　否则，它的消息若是传出去，保不准当年的宗门会对它下达追杀令。
<br/>　　到时候，哪怕它在强，也难逃陨落的下场。
<br/>　　望着笼罩在黑雾下，赤红色瞳孔中的嗜血与疯狂，慕清霄自言自语，道：“天墉城的剑诀果真不凡，也是时候让你消失了。”
<br/>　　说罢，慕清霄眼中浮现三颗勾玉，勾玉旋转间已经化作风车图案。
<br/>　　“天照!”
<br/>　　陡然，九霄琉璃焱乍然而现，不知何时已经将梦魔笼罩在内，甚至连黑雾与周围的空间都燃烧起来。
<br/>　　“啊啊啊...”
<br/>　　察觉到九霄琉璃焱致命的威胁，甚至魂体都一丝丝的溃散，黑雾中顿时响起竭斯底里的惨叫声。
<br/>　　立于空中，听着耳中传来的惨叫声，慕清霄丝毫不为所动，万花筒写轮眼依旧锁定着百米外的梦魔。
<br/>　　在九霄琉璃焱的燃烧下，周围空间都扭曲起来，而笼罩着梦魔的黑雾也早已不见，露出一个魁梧却狰狞的姿态。
<br/>　　魁梧的身躯，苍白的面孔，赤红如野兽一般的瞳孔。
<br/>　　看到梦魔的瞬间，慕清霄首先想到的是人类，但他却没有停手，人形姿态不过是梦魔迷惑世间的手段罢了。
<br/>　　这具身躯不过是由极度的阴气凝聚而成，在九霄琉璃焱的焚烧下，渐渐的变得虚弱透明起来。
<br/>　　“放......放过本座，本座可以给你任何想要的东西。”
<br/>　　梦魔虚弱的声音响起，赤红色瞳孔中的疯狂与嗜血已经消失，取而代之的是无尽的恐惧。
<br/>　　慕清霄冷笑道：“我想要的东西，自己会取，况且，你有跟我谈条件的资格吗？”
<br/>　　“既然你不肯放过本座，那就与本座一起去死吧！”话语刚落，梦魔虚弱透明的身躯里却爆发出无与伦比的恐怖气息。
<br/>　　“打算自爆内丹吗？”
<br/>　　望着向自己急速掠来的梦魔，慕清霄低喃一声。
<br/>　　正所谓兔子急了也咬人，更何况梦魔天性残忍，智商完全不低于普通人的妖魔鬼怪。
<br/>　　“去死吧，桀桀桀......！”
<br/>　　伴随着一阵诡异的笑声，只见梦魔魁梧的身躯，掠向慕清霄的同时，也在不断的缩小，最后化作一颗纯白色的玉珠。
<br/>　　慕清霄剑眉一挑，暗骂一声，早知道就应该以雷霆手段将梦魔击杀，现在倒是给敌人一个临死前反扑的机会。
<br/>　　不愧是天性残忍的梦魔，练虚三重的梦魔自爆，若非提前布下领域，恐怕整个碧山都会被夷为平地。
<br/>　　顷刻间，以玉珠为中心，一股毁灭性的力量，瞬间向四周席卷而出。
<br/>　　“轰轰——”
<br/>　　恐怖的爆炸声，犹如晴天霹雳，虽有领域的阻挡，依旧是响彻在碧山方圆数十里范围的上空。
<br/>　　良久，恐怖的风暴消失，狂风拂过，弥漫的尘埃随风而散，碧山再次归为平静。
<br/>　　立于高空，慕清霄脸色苍白，嘴角溢出一丝血痕，望着百米外，泯灭在风暴中的梦魔，久久无言。
<br/>　　梦魔的死，给他上了一课，对于强大的敌人，应当以雷霆手段击杀，否则，后果不堪设想。
<br/>　　若非来的是他，哪怕是天墉城除掌门和执剑长老外的任何一位，都没有留下梦魔的可能性。
<br/>　　甚至，会陨落。
<br/>　　虽然用领域挡住了梦魔自爆内丹的恐怖力量，他自己本身也在恐怖的风暴下，受到了不小的反噬。
<br/>　　当然，只要不是致命的伤势，对于他恐怖的恢复力来说，完全不算事。
<br/>　　收敛心神后，慕清霄便撤去领域，周围空间细微的动荡一下，宛如透明玻璃一样的领域结界就消失不见。
<br/>　　向下望去，自闲山庄与碧山丝毫不损，只是后院的通道处却早已满目疮痍，望着角落处的叶沉香，身形一闪就来到金属大门处。
<br/>　　梦魔死亡，玉衡并没有被它带在身上，感受着周围汇聚过来的灵气，慕清霄毫不犹豫的向仓库中走去。
<br/>　　仓库并不打，三十平方左右的地方，墙角堆满了布满尘埃的铁箱，墙壁上也架着各种武器。
<br/>　　瞥了一眼箱子，无非就是些金银珠宝，至于墙壁上的武器，虽不是凡品，但也尽是如此罢了，对他来说与破铜烂铁无异。
<br/>　　整个仓库中，唯一能吸引他的，只有视线尽头的石台上，那散发着淡淡荧光，类似于玉玺一样的东西。
<br/>　　玉衡！
<br/>　　随手一卷，石台上的玉衡仿佛受到吸力一般，下一秒就被慕清霄攥在手掌中。
<br/>　　“叮，掠夺任务③，掠夺玉衡任务完成，奖励200000掠夺积分。”
<br/>　　脑海中响起久违的系统提示声，慕清霄嘴角终于牵起一丝笑意，不虚此行。
<br/>　　“铮铮——”
<br/>　　就在这时，清脆的剑鸣声响起，只见魂剑自阴阳玉中掠出，环绕在慕清霄身旁，似乎在催促他一般。
<br/>　　望着魂剑激动的模样，慕清霄轻轻将手松开，手中的一般玉衡就与魂剑一同消失。
<br/>　　进入阴阳玉后，魂剑就开始炼化玉衡，慕清霄心中难免有些激动，这可是仙器，不知道炼化后，自己能增强多少实力。
<br/>　　不知何时，他的手中已经攥着另一半玉衡。
<br/>　　看现在的情况，魂剑似乎只需要炼化一半即可，至于如何使用另一半，他心中已经有了了一个大致想法。
<br/>　　玉衡虽只有一般，但也是仙器，虽然只有一半，但仅是汇聚天地灵气的效果，就丝毫不弱于天墉城中的顶级聚灵阵法。
<br/>　　若是拿出去，不知道多少修真门派会不择手段的得到它。
<br/>　　除了玉衡，本次最大的收获，便是任务奖励的二十万掠夺积分了。
<br/>　　加上以前积累，天墉城所获得的掠夺积分，与现在任务完成后获得的掠夺积分，总共有八十八万，距离目标也越来越近。

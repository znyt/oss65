　　噗噗噗！
<br/>　　剑气所过之处，冲在最前面的十几人，顿时便化成了一团团血雾。
<br/>　　“啊！！！”

<br/>　　叶尘脚下步子不停，淡淡开口道。

<br/>　　“人质？”
<br/>　　叶尘不由得眉头一皱，同时神念一扫，顿时便发觉在这艘游轮的下方，还有上百道普通人的气息，显然应该就是黑人中年男子口中的人质了。
<br/>　　‘看来眼前这些人，还真是海盗，而船上的这些人质，应该都是华夏人！’
<br/>　　‘我好歹也是华夏国的元帅，此事可不能不管！’
<br/>　　一念及此，叶尘脸色一寒，
<br/>　　“既然如此，还不敢赶紧把人给我放了！”
<br/>　　中年黑人男子连忙点头称是，冲着一旁的手下叽里咕噜说了一通。

<br/>　　“小伙子，你自己都自身难保了，又如何救我们？”
<br/>　　“这些人可都是杀人不眨眼的魔鬼，你小心说话！否则他们会杀了你的！”
<br/>　　……
<br/>　　叶尘微微一笑，正要开口解释，这时人群之中，忽然传出一道充满欣喜的清脆声音，
<br/>　　“叶尘？真的是你吗？”
<br/>　　叶尘顿时一愣，当即循着声音，朝那人望去，
<br/>　　只见，这是一名身材有些臃肿的女子，而且头发凌乱，满脸的污秽，看不清楚真实面容。

<br/>　　“庄菲菲？你怎么也在这儿？”

<br/>　　很显然，她那臃肿的身材，和满脸的污秽，应该是刻意为之，否则以她的绝美容貌，恐怕早就被这些海盗玷污了。
<br/>　　庄菲菲见叶尘认出了自己，顿时更加欣喜不已，当即从人群中冲了出来，直接扑进了他的怀里，
<br/>　　“真是太好了！你来了，那我们就有救了！”
<br/>　　庄菲菲曾经见识过叶尘的手段，自然知道眼前这些海盗，根本不可能是叶尘的对手。
<br/>　　“菲菲！快回来！他们会杀了你的！”
<br/>　　人群中，一名中年女子看到庄菲菲冲了出去，顿时急的不行，忍不住大声吼了起来。
<br/>　　原来此人是庄菲菲的经纪人。
<br/>　　庄菲菲却回头展颜一笑，
<br/>　　“红姐，你放心好了，有叶尘在，我们都不会有事的！”
<br/>　　庄菲菲说完这话，直接将身上那件破旧的大衣脱了下来，露出一袭红色的束腰长裙，将她那完美的身材，尤其是那一对雪白、修长的玉腿，彻底展露了出来。
<br/>　　继而又擦去脸色的污秽，整理了一下乱糟糟的头发，露出一张绝美的容颜。

<br/>　　人群中那位红姐见状，顿时都快要急疯了，
<br/>　　“菲菲！你是疯了吗？快！快穿上衣服！”
<br/>　　其余众人质，也顿时纷纷摇头叹息，
<br/>　　“唉！这姑娘怕是精神有问题，就怕这些海盗把她给……”
<br/>　　“她看上有些眼熟，好像是国民女神庄菲菲？”
<br/>　　“可惜了！她根本不知道，自己面对是一群什么样的人！”
<br/>　　……
<br/>　　在众人的议论声中，那名中年黑人男子，战战兢兢的朝叶尘走了过来。
<br/>　　因为是背对着众人，红姐等人无法看到，黑人中年男子此时，脸上是何等的恐惧和敬畏。

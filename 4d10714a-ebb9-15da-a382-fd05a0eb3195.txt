　　岁月无声无息，让人害怕。洪荒之中，最不值钱的就是时间。在元始斩三尸之后三千年过去了。
<br/>　　这一天，帝俊、太一降临不周山，兄弟二人与女娲、伏羲一起，整合妖族势力，准备祭天立族。
<br/>　　又一日，帝俊麾下的妖圣白泽，偶然间发现不周山上方，有着浩瀚的一界，三十三重天衍生而出。在他禀告帝俊之后，帝俊大喜，带着太一、女娲、伏羲四人，急忙登上不周之巅。
<br/>　　当四位妖族准圣发现这一方世界，并踏入正在孕育的天界时，望着那广阔无垠，浩瀚无边的三十三天，四人狂喜，帝俊忍不住仰天大笑：
<br/>　　“天界，这三十三天上接太古星空，可接引星辰之力修行，下凌驾不周神山。
<br/>　　哈哈！哈！这正是我妖族崛起，确立根基之所。白泽，你功劳甚大，他日我妖族立下天庭之时，你当为第一妖神。哈！哈！哈！”
<br/>　　帝俊的大笑声，如同雷霆，带着一股迫不及待要爆发的野心一般，看到如此广阔、浩瀚的三十三天，哪怕是他，也激动不已。
<br/>　　“谢过妖帝，属下立刻就去通知众多妖族妖神。”白泽听到帝俊的话后，立即兴奋拜倒，整个人变得兴奋不已，恭敬地退下去。
<br/>　　“大哥，有如此根基，我妖族大兴，指日可待啊！”太一脸色狂喜，望着帝俊，大笑道。
<br/>　　女娲与伏羲也点头称是。于是，在四人谋划之下，开始指挥大批妖族，迁徙到天界之中居住。
<br/>　　岁月流转，眨眼间，又是一千年过去了。这天，整个洪荒天地，一片仙音飘扬，响彻云霄，一股股玄奥之音弥漫八方，天界三十三天，彻底成型了。
<br/>　　“嘶！原来天道之下，洪荒竟然还有如此世界出世，天界吗？居太古星空之下，接引星辰之力，果然是好地方，贫道去看看。”一瞬间，如此惊天异象显化而出，洪荒各地，许多大神通者纷纷惊醒，神色一动道。
<br/>　　昆仑山玉虚宫中，悟道古茶树下，元始双目睁开，抬头看向苍穹，沉声道：“三十三天出世了吗？如此，巫妖之争也即将开启了。哎！”
<br/>　　这般说着，元始的目光刹那间穿透过虚无之空，落在了太古星空下面，那浩大的三十三天之中，他顿时看到了，那不周之巅，有着三十三个小世界，世界之中，金碧辉煌，雕栏玉砌，灵气纵横，仙鹤飞舞，正是天界。
<br/>　　“帝俊与太一，果然不愧是量劫主角，竟然抢先一步，将那天界三十三天占据了。呵呵，可惜，这世界哪有永恒的主角啊，就算是日后的人族，也不敢说自己是永恒的主角。你妖族如今看似声势浩大，但在这大劫之下，谁又能永存呢？
<br/>　　妖族大战一起，洪荒的平静就要被打破了。吾有两个选择，一是坐守昆仑山，不理两族，争取早日证圣。至于第二点嘛，则是相助一方，参与到大劫之中。
<br/>　　妖族我早已得罪了，却是不能选择，况且，说不定妖族哪天还会来找我麻烦呢。那就只剩下巫族了。”
<br/>　　说道这里，元始眼中精光一闪，道：“巫族吗，倒也不错。至少我三清与巫族之间还有盘古这一层关系，比起妖族要亲近许多。”
<br/>　　“量劫么？《万劫魔典》也该出世了。”
<br/>　　“不过，我三清的身份不能明着来。得想一个办法。”
<br/>　　元始眉头一皱，思索了一会儿，从他的身体之中，走出一个身穿黑袍，头戴魔冠，手持十二品黑莲的道人，他满脸张狂，眼神犀利，全身散发着异样的魔性。他与元始没有一点相像，两人站在一起，绝对没人能够看出。
<br/>　　这正是他的恶尸。
<br/>　　“吾为万劫魔尊，见过道友。”
<br/>　　“吾还需继续参悟大道，你就替吾走一趟吧。”元始本尊淡淡道。
<br/>　　万劫魔尊毫不犹豫地转身，走出玉虚宫，向着巫族所在的方向而去。
<br/>　　万劫魔尊走后，老子与通天来元始住所，问道：“大兄，这妖族要立天庭，吾等又该如何？”
<br/>　　“哼，妖族立天庭乃天数，吾等不该阻拦。况且妖族要立天庭，该着急的又不是我们。”元始淡淡道。
<br/>　　老子眼睛一缩，道：“大哥是说——”
<br/>　　“不错，不是还有巫族吗？我等只需参悟道法，早日成圣便可。”
<br/>　　“是，大兄。不过，大兄，你可有何办法助我等斩尸。”两人想起之前元始连斩三尸之事，顿时向元始请教道。
<br/>　　“斩尸是自己的事，每个人的经历都有所不同，这种事我也帮不了你们。不过，我也有一句忠告，听不听在你们。”元始看了两人一眼，淡淡道。
<br/>　　“大兄但说无妨。”
<br/>　　“不要太过依靠外物。成圣之前，斩尸越多，积累也就越多，成圣后进步也就越快。不要急着成圣。”
<br/>　　“大兄此话何意？”
<br/>　　“就是这个意思。”元始没有再多说，听不听是他们的事。他只能隐晦地说不要太过依靠外物，难道他还能明说鸿蒙紫气可能有限制吗，那不是找死吗。
<br/>　　老子若有所思，通天则是完全没有在意。两人无奈，只能回去继续参悟大道。
<br/>　　而与此同时，那三十三天之端，帝俊、太一、伏羲、女娲、鲲鹏五大妖族准圣强者，还有遍布三十三天的亿万妖族，其内大罗金仙强者，被帝俊太一等人聚集起来，竟有数以百万计，简直太多了。
<br/>　　“嘶！帝俊、太一、女娲、伏羲、鲲鹏五人，这是要干什么？统合整个洪荒之内的妖族，聚集在三十三天中？居然有如此多大罗金仙，这妖族，真是可怕！”各路准圣大能者，望着帝俊五人的动作，看着妖族的底蕴，眼皮狂跳。
<br/>　　即便是大罗金仙强者，在准圣眼中，犹如蝼蚁，但猛然间见到如此浩大的队伍，各路大神，还是震惊不已的，哪怕是元始，他也不得不承认，他小看了帝俊四人了。他并非是忌惮妖族的势力，对于他来说，大罗金仙再多，也不能伤他分毫，大罗金仙对于他来说，都是蝼蚁。
<br/>　　他小看的是，帝俊与太一等几人的野心，汇聚千万妖族大罗金仙，无数小妖，这还要干什么？而今更是占据在三十三天，这是要争霸洪荒的节奏啊。
<br/>　　而众多大神的担心，不是没有道理，众目睽睽之下，却见帝俊五人在无数妖族的恭敬，狂热眼神之中，踏步冲天，仿佛俯视苍生。
<br/>　　之前，帝俊等人便去紫霄宫亲自请示鸿钧道祖，得到了道祖的同意。
<br/>　　于是，妖族要立天庭了。

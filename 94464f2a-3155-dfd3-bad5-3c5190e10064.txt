　　接下来的这段时间里，刘哔大部分时间都待在基地，偶尔会和什造一起外出，带她熟悉这边的世界。
<br/>　　封印卷轴只有一张，他执行任务的时候，没办法将什造带在身边。
<br/>　　更何况下一个位面的危险程度如何，他也无法确定，相对来说，什造留在这边还是比较安全的。
<br/>　　趁着还有时间，他用上一次收集来的那些晶核，在平民区置办了一套房屋，带着什造和刘莽搬了过去。
<br/>　　事实上住在哪里对他来说都没有太大关系，之所以购置这套房屋，也是为了刘莽什造两人着想，
<br/>　　毕竟贫民区那种地方，实在是太过于混乱，住久了也难免会引出一些事情。
<br/>　　一个月后，系统提示的声音姗姗来迟，和什造两人告别之后，刘哔独自一人出了城。
<br/>　　依旧是选择了那片丧尸横行的建筑，刘哔整理了一下物品，默默等待着传送的时间。
<br/>　　和前两次相比，他这次传送做的准备更加充足，首先是武器装备上升了一个档次，其次他也花费积分，兑换了一些常用药品，以备不时之需。
<br/>　　等待了大概一个多小时候，刘哔四周的空间一阵扭动，一股眩晕感传上脑海。
<br/>　　清醒过来的时候，周围的场景已经变换成了另外一种样子，这是一条残破的街道，无人管理的车辆密密麻麻的停放在道路上，将马路塞的如同沙丁鱼罐头一般拥挤。
<br/>　　“吼啊！”
<br/>　　一声嘶哑的尸吼声从耳边传来，刘哔心中一惊，扭头看去，发出声响的，是一个浑身挂满烂肉的怪物。
<br/>　　这种怪物和丧尸十分相似，看起来熟悉至极，要不是其身上一些细微的不同之处，刘哔还以为传送出了什么问题。
<br/>　　是生化位面么……只是不知道是哪一个世界，自己看过的生化片子并不算多，只要给他点时间，还是很容易能推算出来的。
<br/>　　现在要搞明白的是，自己的体质在这个世界还适不适用，以及，这个世界的基本信息。
<br/>　　这样的环境对他来说，简直再也适合不过了，这个位面的危险程度，无形之间也降低了许多。
<br/>　　“吼啊！”
<br/>　　丧尸的尸吼声继续响起，拖着有些残破的身躯，蜗牛一般的来到了刘哔面前。
<br/>　　当然，这种慢只是对刘哔来说，因为他的神经反应力太过于强大，故此才会感觉到这只丧尸行动缓慢。
<br/>　　实际上，它的速度也和正常人类差不许多。
<br/>　　刘哔没有动，他只是静静的等待着这只丧尸，他想试试，这只丧尸会不会对他进行攻击。
<br/>　　“嗖！”
<br/>　　就在丧尸好不容易挤到他面前时，一柄木质的箭矢射穿了丧尸的头颅，瞬间，一股腥臭的液体溅了出来，洒在刘哔面前的地面上。
<br/>　　“快跑啊，愣着干嘛，有一大群尸兄在后面！”
<br/>　　一道陌生的声音响起，刘哔皱了皱眉头，转身看去，不远处一个戴着奇怪口罩的胖子，正被一大群丧尸追赶着。
<br/>　　口罩胖子虽然身体肥胖，但速度却是不慢，至少还能在尸群的追赶下，用手中的自制长弓‘帮’刘哔拦下了丧尸。
<br/>　　“不想死就快跑啊，前面我知道一个胡同，我们翻过去，这些尸兄追不上来。”
<br/>　　胖子用不符合他体型的速度冲到刘哔面前，气喘吁吁的喊了一声，动作却是没有丝毫减慢的向前跑去。
<br/>　　“啊，哦，等等我啊，我不认识这边的路！”
<br/>　　看了一眼身后的尸群，刘哔脸色煞白，眼中流露出一抹惊慌，极为费力的跟了上去，紧紧掉在眼睛胖子身后。
<br/>　　往后看了一眼，注意到刘哔随时要被甩开的速度，胖子皱了下眉头，伸手拉住他的胳膊，继续向前逃跑。
<br/>　　“吃……吃……好吃。”
<br/>　　“吼啊……人……好吃。”
<br/>　　后面的尸群追赶着，口中发出含糊不清的嘶吼声，因为这边混乱的声音，附近街道上的丧尸也被吸引，汇聚到尸群当中。
<br/>　　这便是尸群可怕的地方，你完全无法和它们开战，如果一被拖住，那就很难从尸群中逃离出来了。
<br/>　　胖子拉着刘哔的手臂，脸上的表情愈加焦急，本来他自己逃出去的话还比较容易，但多带上一个人，就略微有些艰难了。
<br/>　　不过从小培养的观念，根本做不出见死不救这样的事情。
<br/>　　“我不行了，跑不动了，你扔下我吧，带着我们谁都跑不掉！”
<br/>　　大口大口的喘着粗气，刘哔脸上的表情十分痛苦，双腿如同筛糠一般，小幅度抖动着。
<br/>　　常年修习武术的胖子知道，这是即将脱力的表现。
<br/>　　“妈的，就在前面了，拐过这个街道，我们就能顺利逃走了，你可别在这个时候掉链子啊。”
<br/>　　死命拽着刘哔，胖子在街道中灵巧的穿梭着，躲开了所有向他扑来的丧尸，终于，两人经过前面的街道，拐入了一间死胡同当中。
<br/>　　“快，我先上去，然后再把你拉上来，速度麻利一点，后面的尸兄马上就要追过来了。”
<br/>　　叮嘱了一声，胖子翻身上墙，冲刘哔伸出了右手。
<br/>　　似乎是由于脱力的原因，刘哔第一时间并没有上去，眼看即将被尸群淹没的时候，突然爆发出了一股力量，支撑着他跃了上去。
<br/>　　一时没有防备，胖子整个人向后仰倒，摔在了墙壁后面。
<br/>　　“哎呦我去，要不要这么惊险啊，不过还好，勉强算是逃出来了。”
<br/>　　揉了揉摔的生疼的屁股，胖子站起身来，看了一眼坐在原地，仿佛被吓懵了的刘哔。
<br/>　　唉，也是一个可怜人啊，这世界突然变成了这幅样子，又有几个人能缓过神来呢。
<br/>　　叹了口气，胖子拍了拍刘哔的肩膀，安慰道：“走吧，能活命就好，你有地方去吗，要不就先跟我回家，我那里相对来说也算是比较安全。”
<br/>　　被胖子轻轻一拍，刘哔哆嗦了一下，缓过神来，呆愣愣的看了一眼胖子。
<br/>　　“谢谢，谢谢，您叫什么名字，这次我能活下来，真是多亏了您了。”
<br/>　　站起身来，刘哔语气中充满感激。
<br/>　　“谢啥，这世界变成这样，大家就应该互相照应，你叫我小胖就行，天色不早了，先回去吧。”
<br/>　　摆了摆手，小胖唏嘘了一声，虽说是互相照样，不过看着情况，他也没指望这人能帮上自己什么忙。
<br/>　　太弱了。

　　凭心而论，李承乾对于长孙无忌其实并没有什么特别的看法，从某些方面来说，如果不影响大局，他并不介意给长孙家一些照顾。
<br/>　　可是如果说整整一座矿山送给长孙家，李承乾自认做不到这一点。
<br/>　　毕竟大唐并不是只有长孙一家，而且长孙无忌是才头子的嫡系，并不是他的嫡系，把矿山给了才长孙，最后也不会得到他的效忠，反而会因此让自己的嫡系有想法，完全得不偿失。
<br/>　　所以李承乾并没有满足长孙无忌的所有要求，只是保证了可以不让长孙氏饿死，又吃不饱的一个状态。
<br/>　　……
<br/>　　抛开长孙老狐狸与李承乾之间的勾心斗角不谈，却说辽东乃至于东北。
<br/>　　转过除夕之后，还没有等到上元节，一群穿着厚厚冬装的矿工便来到了矿上，炮声隆隆中开始了一年的工作。
<br/>　　煤矿如此，铜矿、铁矿亦是如此。
<br/>　　不过相比于煤矿，铜矿石和铁矿石在被开采出来之后需要进行冶炼加工，将矿石提炼成金属原料，而后为数不多的铜则要运往长安，优先保证军方使用，而那些生铁则是分配到辽东的各个世家的小型钢铁厂。
<br/>　　而此时的大安城，则正在进行着新的改造，原本的城墙全部被破坏掉，成为大唐唯一一座没有城墙的城市。
<br/>　　本着废物利用的原则，那些拆下来的碎砖烂瓦基本上全都被拿来修路，一些木质结构的东西则要看其木料的稀有程度再决定其处理方式。
<br/>　　但是显然，这样的分类方式并没有什么大用，因为没有人会在城墙上使用什么上好的木料，所以城头上拆下来的木头大多数都被用来烧制水泥，一些特殊粗大的，则被送给了一些从大唐迁居到此的百姓，留着给他们建房时充当大梁。
<br/>　　最重要的是，大安城的外围正有十余台推土机在不停工作着，整片整片原本的荒地在推土机经过之后变的平整，而当地面被堆平之后，冻土层也随之被推走，留下的是已经化开的土地。
<br/>　　这样一来就方便了施工，大量工人追在推土机的后面，对地面进行夯实工作，留待日后建厂。
<br/>　　刘弘基很不理解李承乾拆掉城墙的做，毕竟在大唐这个时期，城墙什么的依旧是对敌人最好的防御手段，如果没有了城墙，整个城市立刻就会变的如同一个手无寸铁外加身无寸缕人女人，而且这个女人还是行走于土匪的老巢之中。
<br/>　　可是李承乾给他的回复是就这么干，城墙必须拆除，而且不但但是大安城，以后如果有机会，整个辽东，乃至于整个大唐所有的城墙也都要拆除。
<br/>　　至于原因，以后有机会再说吧。
<br/>　　得到这样的答复之后，刘弘基也没有什么好办法，只能有些抱怨的执行李承乾的命令，将大安城的城墙彻底摧毁。
<br/>　　……
<br/>　　“老李，你说，这不是扯蛋么，也不是东宫那位是怎么想的，这城墙怎么能说拆就拆呢。”在李绩的大都督府中，刘弘基把近来发生的事情都说完之后，狠狠给自己灌了一大口酒，然后有些抱怨的说道。
<br/>　　李绩在听完刘弘基的诉苦之后，微微一笑，陪着他干了杯中酒，然后笑呵呵的说道：“这也不是什么不可理解的事情，如果弘基你关心一下军方的事情就会知道这样的做其实也是有好处的。”
<br/>　　“怎么！拆了城墙还有道理了？”刘弘基翻着白眼反问道。
<br/>　　“有没有道理先不说，我来给你一点提示吧。”李绩见刘弘基依旧执着于城墙，笑着摇了摇头，然后问道：“你知道贞观炮吧？”
<br/>　　“知道，不就是那个能打出五百多步，和投石机差不多的东西么。”刘弘基往嘴里塞了一口菜，有些不在乎的说道。
<br/>　　但李绩却出乎他预料的叹了口气，然后缓缓说道：“能打五百步的贞观炮那是多少年前的事情了？没想到你的消息竟然落后了这么多。”
<br/>　　“那现在能打多远？八百步？”刘弘基反问道。
<br/>　　“你在前面再加个二还差不多。”看着刘弘基倔驴一样的嘴脸，李绩哼了一声道。
<br/>　　“噗……”端起杯子正在往嘴里灌酒的刘弘基一口老酒全都从鼻孔里喷了出来，其中还夹着两根香菜。
<br/>　　不过他现在已经顾不上这些了，瞪着两只灯炮一样大的眼睛说道：“你再说一次，多少？”
<br/>　　李绩伸出三根手指：“标准射程，三千步。当然，如果用说明书上的说法，那叫三千米。”
<br/>　　“三，三千？”刘弘基的眼中带着不可置信，无论如何也理解不了三千步或者说三千米的概念是什么，要知道，三千步之外，那可是连人都看不到了。
<br/>　　“而且射程达到三千步的贞观炮还是十六卫的标准配备，一些精锐军团，比如说玄甲军和飞凤军，我听说他们装备的贞观炮已经可以打到四千到五千步。”李绩生怕吓不死刘弘基，继续给他加码：“而且我还听说，科学院那帮家伙正在研究可以打到六千步的贞观炮。”
<br/>　　老实说，能打三千米的贞观炮刘弘基已经理解不上去了，至于李绩后来说的六千，其实对于他来说和三千没有什么区别，左右不过就是一个数字而已，该理解不上去还是理解不上去。
<br/>　　李绩看着哑色的刘弘基，嘴角微微一挑：“弘基，你不要以为这是我在吹牛，事实上现在咱们辽东军已经全部换装了新式的贞观炮，而且火箭筒之类的也全都换了新的，这种新式的火箭筒已经不是竹节的了，而是钢制的，其射程也达到了一千五百步。”
<br/>　　“所以你现在还觉得城墙有用么？真正的炮火覆盖之下，只怕你连敌人在哪里都没看到，城墙上的人就已经死光了。”
<br/>　　“而且那么大的射程，你觉得一面城墙能起到什么作用？除了能让自己的百姓逃的慢一些，你觉得它能挡得住那种火炮的进攻？”
<br/>　　挡的住么？当然挡不住，当年的贞观炮他曾经看过，那东西威力极大，就算是你有城墙在，人总是会死的，所以当城墙起不到保护作用的时候，拆了它似乎成了必然。

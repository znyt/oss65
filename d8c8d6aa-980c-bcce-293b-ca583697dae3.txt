        “但也不是这么个奢侈法呀？我就开你这台备用车就行了。”童筱颖食指上勾着一把车钥匙，在傅夜沉的眼前晃了晃。
<br/>
<br/>        他那台备用车应该价值上千万了吧！他还给她买车，实在是太浪费钱了。
<br/>
<br/>        而且，好像苏漫雪跟他在一起的时候，也就只开着百万以下的豪车。
<br/>
<br/>        殊不知，苏漫雪当时挥霍傅夜沉的钱，若不是有刘管家管着，只怕早就把上千万的顶级奢侈跑车排着队买回家了。
<br/>
<br/>        “宝贝，你真有眼光，挑了一辆，最贵的备用车。”傅夜沉将童筱颖拉到自己跟前，大手揽住她纤细的腰肢，会心一笑。
<br/>
<br/>        “多少钱？”童筱颖忍不住好奇地问。
<br/>
<br/>        傅夜沉淡定地回答道：“全球限量只卖六台，每台价值一个亿，待会我让郑助理去办理手术过户到你的名下。”
<br/>
<br/>        “车钥匙是我随手拿的……”童筱颖将车钥匙握在手心，嘴角微微抽了抽，“过户就不用了。”
<br/>
<br/>        她就说她开着那辆跑车一路过来的时候，怎么被那么多路过的司机落下车窗盯上了，还差点造成他们的车辆连环追尾现场，原来……
<br/>
<br/>        “不许拒绝我。”他很霸道地说。
<br/>
<br/>        童筱颖尴尬道：“那个，我跟你换吧！你把你今天开出来的车给我开，你开我开出来的那台备用车回去。”
<br/>
<br/>        “开跑车不适合我出门谈生意。”
<br/>
<br/>        “好吧！”童筱颖妥协了。
<br/>
<br/>        刚刚在楼下大厅里吃过她做的饭菜后，他又拉着她来总裁办，明明说是给她看婚戒款式，结果莫名其妙地扯到了车子的话题上。
<br/>
<br/>        “把手机给我。”傅夜沉一边说着，一边往沙发上坐下去的时候，顺手搂着童筱颖，让她一屁股跌坐在了自己的大腿上。
<br/>
<br/>        童筱颖想起身，却被他紧紧地按住，双腿悬在了半空中：“大哥，别这样……这里是办公室。”
<br/>
<br/>        “手机给我。”傅夜沉扬了扬眉梢。
<br/>
<br/>        童筱颖不得不从裙兜里掏出手机递给傅夜沉。
<br/>
<br/>        傅夜沉接过童筱颖的手机后，将自己的私人号码存了进去，并设置了拨号键“1”为快捷键。
<br/>
<br/>        “这是我的私号，二十四小时为宝贝服务。”傅夜沉勾唇一笑，将手机放回了童筱颖的手中。
<br/>
<br/>        不得不说，她今天的行为有点反常。
<br/>
<br/>        她先是主动为他下厨，还主动送到公司里来，甚至不顾形象地愿意坐在他公司楼下大门外的台阶上等他，现在又对他几乎是百依百顺。
<br/>
<br/>        这小女人脑子里不会又想到了什么离开他的法子吧？
<br/>
<br/>        “那我先回去了。”童筱颖说着，欲要起身，结果傅夜沉仍旧环抱着她的腰肢，不肯松手。
<br/>
<br/>        她抬眸，与他的目光在半空中相遇。
<br/>
<br/>        他用他那双墨黑如曜玉、深邃如银河的眼眸，意味深长地瞧着她。
<br/>
<br/>        “亲我一下。”傅夜沉嘴角微扬，用手指捏住童筱颖尖巧的下颔，面向自己，眉眼间透着坏坏的笑意。
<br/>
<br/>        童筱颖下意识地悄悄吞咽口水，身子顺应自己的本能，忍不住往后躲了躲。
<br/>
<br/>        她的抗拒，再次让他心口隐隐作痛。
<br/>
<br/>        好，没关系，他等，慢慢等……
<br/>
<br/>        傅夜沉刚一松手，童筱颖便趁机“噌”地一下，直接从他的大长腿上跳了下来。
<br/>
<br/>        显而易见，她有意与他保持距离。
<br/>
<br/>        “那我先走啦！晚上见！”童筱颖抿嘴浅笑，然后提起茶几上的那个空饭盒，转过身去。
<br/>
<br/>        傅夜沉随之从沙发上起身，一个跨步上前，伸手直接捉住了童筱颖的臂弯，带动着她的身子往自己跟前拉动。
<br/>
<br/>        童筱颖被这股突如其来的力道，转出一个华丽的旋身，杏色的纱裙裙摆在半空中散开优美的弧度，她随之撞上了一个结实的胸膛。
<br/>
<br/>        “晚上，我会早点回来。”他微微俯身，凑到她耳边呢喃，一股灼热的气息拂上她的脸颊。
<br/>
<br/>        他搂着她，温热的掌心贴着她的后背，隔着衣服，来自于他的体温慢慢地渗入了她的身体里。
<br/>
<br/>        “嗯。”童筱颖不敢直视傅夜沉的眼睛，聋拉着脑袋，垂着眸子，木讷地点了点头。
<br/>
<br/>        他握住她手臂的大拇指缓慢而轻柔地在她肌肤上滑动，仿佛极为留恋着与她每一寸的触碰。
<br/>
<br/>        傅夜沉身上透着一股灼热的气息，将她暖暖地包裹。
<br/>
<br/>        他看着面前的童筱颖，抬起手来，温柔地将她颊边的发丝撩到耳后。
<br/>
<br/>        童筱颖有些不知所措地抬眸，不经意间地迎向傅夜沉的眼。
<br/>
<br/>        她怔愕地看到他眸里的温柔再自然不过，仿佛她真是他眼中的宝贝，正细细地呵护着。

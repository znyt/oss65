　　比赛结束后，国内的媒体自然是一片欢腾，所有的新闻都在讲诉着今天的这场对决。
<br/>　　当然了叶帝使用的龙王这个英雄自然是被大写特写，除此之外金相赫所用的中单圣枪游侠也是被各大媒体好一顿宣扬，可以说这一场小组赛的落幕之战，真的是让大家一饱眼福。
<br/>　　接下来的几天无论是叶帝他们还是蒋念这群人，都在基地内抓紧训练着，三天后就要开始下一轮的比赛了，接下来的比赛将会更加的激烈，所以由不得他们轻松下来。
<br/>　　当然了他们这样积极，苦了的自然就是皇族战队了，不过同样的在和荣耀蓝白两支战队还有雷霆战队的对决之中，皇族的实力也是在迅猛的增长着。
<br/>　　同样的雷霆战队的实力也是在那里增长着，进入世界赛其实对于他们也是一场洗礼，所以此刻皇族和雷霆两支战队在对决的时候依旧是55开。
<br/>　　一转眼的时间三天时间便是过去，在无数观众期待的目光中，12进8的比赛终于是来临。
<br/>　　“之前已经和大家说过了，12进8是bo5单循环赛制，每胜一小局加一分，输一局则扣一分，也就是说战队很可能会出现负分的局面的，而之前在各小组获得第一的战队则是会额外获得两个积分，第二名则是会加一个积分，当然了能够进入到这里的都是第一第二名，也就是说初始几分大家只相差一分而已。”解说席上米粒开口在那里说着赛制规则。
<br/>　　“也就是说上下半区会各自进行15场bo5的比赛，加起来也就是30场bo5，我们接下来每天会进行三场比赛，所以这次的12进8的比赛中，预计会在十天完成，当然如果出现特殊情况那可能就要延迟了。”林雅楠跟着开口。
<br/>　　“今天我们的第一场比赛是ST战队对阵SKY战队，让我们拭目以待。”随着米粒的一声开口，场上的比赛终于是正式开始。
<br/>　　而这个时候叶帝一行人也都是在基地内大厅中围坐在一起，等待着这场比赛的到来。
<br/>　　“SKY战队这边有着long这边十王者的辅助选手，加上几名队友也都十分的强力，今天这场比赛倒是也挺有意思的。”蒋念开口说道。
<br/>　　“是啊，不过我还是更加期待明天你们和eP战队的比赛，能够以一个ad位置一己之力冲上十王者，emp的实力绝对要超过绝大部分选手，就是不知道你们明天会不会翻车了。”秦霄笑着说道。
<br/>　　今天他们两支战队都没有比赛，两队的比赛都排到了明天。
<br/>　　“明天你们对战的是雷霆战队吧，雷霆战队的实力同样不可小觑，你们也加油吧。”叶帝笑着说道。
<br/>　　如今就剩下12支战队了，而且蓝队和雷霆战队都在下半区，他们自然是会相遇的，虽然蓝队的实力要强一些，但是在赛场上什么局面都可能发生的，所以也保不准蓝队就一定不会输，同样的对于白队也是如此。
<br/>　　在几人议论着的时候，场上ST和SKY战队的比赛已经开始，值得一说的是这一局金相赫选择了龙王这个英雄。
<br/>　　“看来相赫回去应该没少练习这个英雄，今天有的看了。”李景禹笑着说道。
<br/>　　场上的时间飞快的走着，虽然SKY战队对于龙王这个英雄已经有了很深的了解，但是这个英雄毕竟是游走支援类型的英雄，即使SKY战队了解，但是面对着经常找机会gank的金相赫，也是终有疏漏的时候。
<br/>　　最终第一局比赛在时间到达31分钟的时候，便是被ST这边早早的赢了下来。
<br/>　　接下来第二局虽然SKY战队这边将龙王这个英雄ban掉了，只不过在ST这边的中野联动下，经济还是被拉开了，最终这一局比赛在时间到达41分钟的时候结束。
<br/>　　到了第三局SKY战队这边long终于开始发力，拿到了锤石这个拿手的英雄，配合着自家的打野游走，在15分钟的时候便是已经击杀了金相赫两次，让他们这边的中路起来了。
<br/>　　虽然ST一直在拖延着，只不过在经济劣势已经拉开的情况下，而且金相赫还被压制的时候，最终他们还是无力回天。
<br/>　　好在第四局他们再次火力全开，最终以3:1的战绩成功击败了对手。
<br/>　　“好的，让我们恭喜ST战队取得这场比赛的胜利，因为他们是3比1击败了对手，所以这一场比赛ST积2分，而SKY战队则是要减掉两分，因为SKY战队是第三组第一名的身份出线，积分有两分，所以这场比赛结束之后，SKY战队的积分清零，他们拥有零分。”比赛结束后，米粒便是开口分析了起来。
<br/>　　“不过这只是刚刚开始，能够在ST战队的手里拿下一分这对于SKY战队已经是非常的有利了，接下来他们还是有机会的。”林雅楠跟着开口说道。
<br/>　　“接下来让我们休息20分钟，稍等片刻我们将会迎来下半区雷霆战队和BK战队的比赛，希望伟神他们多多加油。”米粒继续开口说道。
<br/>　　听到米粒这话场下众多观众自然又是一阵欢呼，因为是在华夏主场作战，所以这里的观众绝大多数都是华夏的观众，身为一个华夏人，虽然他们可能有着自己喜欢的国外选手，但是真到了这种世界级别的比赛的时候，他们的心里自然是向着国内的战队多一些的。
<br/>　　而且今天能够来到现场观看比赛的观众，绝大多数还都是雷霆战队的粉丝，所以听米粒提及雷霆战队马上就要出场的时候，场下才会爆发出这么大的热情来。
<br/>　　“今天小伟他们和BK战队打比赛，也不知道会怎么样？”林萧有些担心的说着。
<br/>　　“赢下来问题应该不大，BK战队只有一个kaka在，而且kaka也只是前十大王者选手，现在估计小刀的实力都能和他有的一比，更别提雷霆战队还有着小伟这样的十王者选手呢。”秦霄笑着说道。
<br/>　　他自然是能够理解林萧的心情，毕竟林萧曾是雷霆战队的一员，和陆伟还有雷霆战队的感情自然是十分的深厚，此刻关心则乱也是可以理解的。

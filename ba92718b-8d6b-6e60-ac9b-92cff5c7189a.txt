
<br/>　　<br/>　　    王风饶走了，王惊蛰忽然看到了他在那个年纪并未看到的一幕，王仙芝的眼圈红了。<br/>　　<br/>　　    很红的那种红，好像要哭了一样。<br/>　　<br/>　　    男儿有泪不轻弹，更何况是王仙芝这样的男人？<br/>　　<br/>　　    父，子，王风饶和王惊蛰，可能是王仙芝这辈子心里最大的痛了，要说他是这世上最可怜的男人，似乎也不为过。<br/>　　<br/>　　    当年，王风饶用一句出门办事告辞了家人后就再也未归，王惊蛰和他再见已是经年，那个时候他真真的是以为爷爷是去出门办事了，此时看见红了眼圈的王仙芝，他明白了，王仙芝也知道王风饶这一走就不可能再回来了。<br/>　　<br/>　　    画面如梭，不知多久以后，王家父子女三人月下长谈，随后王仙芝走了，然后王冬至也走了，再往后朝歌王村的王家老屋就空无一人了，王惊蛰也走了。<br/>　　<br/>　　    那些年里，王惊蛰去了很多地方，他听了很多民谣然后就去了那些地方，他就像是个漫无目的的旅行者，身背一包菜刀走到哪里是哪里，其实说来也不是没有什么目的，他行走江湖一直想着要解开身上的天谴，王风饶是这样，王仙芝和王冬至也是如此，王家三代人齐出朝歌，为的就是他而立那一年。<br/>　　<br/>　　    再往后王惊蛰看到自己去了曹妃甸，又进了京城，等他离京的时候，手里多了一把玉簪子，那是他去陇西李氏大宅提亲的信物。<br/>　　<br/>　　    渭河上泛舟，偶遇一钓鱼女子，王惊蛰的脑袋里便想起了一段话。<br/>　　<br/>　　    “年少不应该碰见这么惊艳的女子，否则余生该都是她的影子了……”<br/>　　<br/>　　    画面如梭，从良水城到黔西南的深山一直都有着那个女子的身影，从八拐里坡到骊山后的秦村是王风饶，王仙芝和王冬至的身影，他们就像是一幅幅挥之不散的图画，深深的印在了王惊蛰的脑袋里，就好像是烙印一样，永远永远都抹不下去了。<br/>　　<br/>　　    画面嘎然而止，王惊蛰悄然醒转，当他醒来之后已然满面泪痕，其实他也不知道，自己的眼眶里怎么就忽然流出了泪水。<br/>　　<br/>　　    陈三岁眨了眨眼睛，盯着他问道：“进沙子了？”<br/>　　<br/>　　    “嗯”王惊蛰点头。<br/>　　<br/>　　    陈三岁叹了口气：“这个借口太拙劣了，你好歹用手揉揉眼眶在回答我啊，这么掩饰，你不尴尬嘛？”<br/>　　<br/>　　    王敬之直接掠过了他的嘲讽，问道：“有多久？”<br/>　　<br/>　　    陈三岁看了眼石门外面，说道：“虽然看不见外面是什么样，我估计这时候肯定已经天黑了，也就是说刚才你过去了很久很久”<br/>　　<br/>　　    王惊蛰“哦”了一声，然后双手合十朝着扎西降泽敬礼道：“大师，我应该是没过去吧？”<br/>　　<br/>　　    扎西降泽说道：“你心中牵挂太多，红尘事不了，自然无法承接大圆满传承，我佛与你无缘”<br/>　　<br/>　　    在王惊蛰醒来的时候，他眼角的余光曾经看到扎西降泽的眉头有一瞬间，轻挑了一下，虽然幅度很小很细微，但他绝对看清楚了，王惊蛰估计是自己的醒来，让对方有点吃惊。<br/>　　<br/>　　    也许，有太多人一直沉浸在自己的记忆里，而一直都没有再醒过来。<br/>　　<br/>　　    “我是不是很容易走不出自己脑海里的回忆？如果走不出来，要么疯了，要么就是崩溃了？或者干脆就永远都无法醒过来，就像，就像……”王惊蛰搜肠刮肚，歪着脑袋想了半天才找到个合适的形容词：“就像变成个植物人？”<br/>　　<br/>　　    扎西降泽居然毫不避讳的直接点头说道：“是的，你有很大的可能是崩溃或者是根本醒不过来”<br/>　　<br/>　　    王惊蛰不解的问道：“大师，请问我为何能醒的过来呢？”<br/>　　<br/>　　    “你脑海里的东西我是看不见的，我唯一能解释的就是，你的执念太重牵绊太多，导致不管是什么因素都难以将你留在识海里，你的欲望可能已经超脱了你的生死”<br/>　　<br/>　　    王惊蛰张了张嘴，苦笑着点了点头，随即指着还盘坐在地上，但明显已经醒过来，却还在发呆的三井美奈说道：“她也醒了，这是怎么回事？”<br/>　　<br/>　　    王惊蛰醒来之后，就发现三井美奈也睁开了眼睛，明显对方也没落下挺惨的下场，似乎只是虚惊了一场，正在慢慢的恢复着。<br/>　　<br/>　　    扎西降泽皱眉说道：“她……的灵魂应该受过什么加持”<br/>　　<br/>　　    三井美奈忽然开口了，她淡淡的说道：“我在神社里被天照大神赐佑过”<br/>　　<br/>　　    王惊蛰深深的看了这女人一眼，但没心思跟她闲聊，就再次抬头问道：“大师，你就没有什么想要对我说的了嘛？我相信你应该看到了很多的东西，我恳请你能为我解惑，或者为我指点一二”<br/>　　<br/>　　    王惊蛰在说这番话的时候，很礼敬的朝着扎西降泽弯下腰，其实他现在已经知道，自己这一趟藏区十有八九是要白奔波了，但他不甘心。<br/>　　<br/>　　    扎西降泽摇了摇头，平淡的说道：“我知道你的命格里有裂痕，想要补上这道裂痕非人力难为”<br/>　　<br/>　　    王惊蛰凄然的叹了口气，心道一声果不其然。<br/>　　<br/>　　    扎西降泽这时看了陈三岁一眼，却又接着说道：“大圆满象雄耳传经中，就像我之前念的那些，可以洗涤人的灵魂，虽然最后没办法补上你命格中的裂痕，但长久的洗涤是可以为你起到加持的作用的……”<br/>　　<br/>　　    王惊蛰顿时动容，这时他也想起来了，去年王冬至曾经去过一趟小昭寺在桑木仁切法王哪里为他求换了一本摩诃瑜经文，这就是用来洗涤和稳固人三魂七魄的，可惜这卷摩诃瑜经文他读了几遍都没办法领悟，最后不得不放弃了。<br/>　　<br/>　　    现在从扎西降泽这段话中，他明白了，想来藏传佛教对人的灵魂都有一种难以想象的认知，比中土的佛门道派都要往前走的太远了。<br/>　　<br/>　　    “只是可惜，我已经无法为你长久的洗涤和加持了，我要圆寂了”扎西降泽摇头说道。<br/>　　<br/>　　    王惊蛰也看向了陈三岁，眼神里含情脉脉。<br/>　　<br/>　　    陈三岁被他看的一激灵，无语的说道：“你看的我浑身上下都是鸡皮疙瘩了”<br/>　　<br/>　　    王惊蛰很认真的说道：“我推演过，你和孔良还有我要九死一生过后可能会遇到个机缘，如今我和老孔都已经被淘汰了，剩下你是什么结果自然不言而喻了”<br/>　　<br/>　　    陈三岁顿时一愣，不解的说道：“你哪来的那么大信心？”<br/>　　<br/>　　    “你有这个气质……”

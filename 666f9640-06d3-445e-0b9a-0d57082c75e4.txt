　　听到这话，叶尘愣了片刻之后，忽然嘴角微微一扬，
<br/>　　“通天路？你说的是珠峰脚下的那虚空通道吧？”
<br/>　　“什么！你怎么……这不可能！”
<br/>　　云千秋脸色狂变，显然没有想到，叶尘居然也知道通天路，而且还直接说出了准确位置！
<br/>　　这可是蓬莱仙界最大的秘密！
<br/>　　叶尘看到云千秋这幅表情，知道自己所猜不错，不由得冷冷一笑，
<br/>　　“现在看来，你似乎已经没有存在的价值了！至于你们药王宗的那些先辈们，他们若是敢回来，那我等着便是！”
<br/>　　说着这话，叶尘缓缓抬起脚，准备一脚踏下。
<br/>　　云千秋这才顿时慌了，
<br/>　　“叶仙人！叶前辈！只要您饶我一命，我愿意将药王宗千年的宝物，全部奉献给您，并且我愿意成为您最忠诚的奴仆，求您，求您不要杀我……”
<br/>　　此时云千秋，哪里还有之前太上长老的威风，简直犹如一条丧家之犬。
<br/>　　然而，
<br/>　　对于云千秋的话，叶尘却犹若未闻，这一脚最终还是狠狠踩了下去。
<br/>　　“彭！”
<br/>　　没有任何悬念，元婴境巅峰境界的云千秋，在叶尘的脚下，脆弱如同西瓜，一脚便被他生生踩爆。
<br/>　　“嘶！！”
<br/>　　其他四大宗门的宗主看到这一幕，顿时纷纷倒吸了一口凉气。
<br/>　　堂堂药王宗太上长老，天榜前五的强者，居然被一个世俗少年一脚踩死了？
<br/>　　若非亲眼所见，恐怕谁也不敢相信，这竟然是真的！
<br/>　　“这少年，怕是已经具备化神天君的实力了！”
<br/>　　四人不由得对视了一眼，在心中暗暗惊叹。
<br/>　　“恭喜叶前辈大仇得报！”
<br/>　　太乙门宗主倒是反应极快，忽然朗声开口道。
<br/>　　其余三人心中，顿时暗骂此人卑鄙无耻，不过也当即纷纷见风使舵，
<br/>　　“是呀！药王宗平日里作恶多端，叶前辈今日为我蓬莱仙界，除掉了这颗毒瘤，真是可喜可贺！”
<br/>　　“云千秋这个老匹夫，死有余辜！”
<br/>　　几人嘴上这么说着，心里却都在暗暗思量，
<br/>　　‘先混过眼前这一关，等回头禀明宗门前辈，再慢慢找你算账！’
<br/>　　然而，这几人的心思，又岂能瞒得过叶尘？
<br/>　　“我说过，今天你们这些人全都要死！你们以为我是在说笑？”
<br/>　　叶尘转头望着天空上的几人，一脸冰冷。
<br/>　　四人顿时全都瞳孔一缩，显然没有想到，叶尘竟然这么狠辣，他们都已经放下身段主动求和，竟然被他直接拒绝！
<br/>　　“叶前辈！做人还是留一线的好，药王宗在我们五大宗门之中，只能算是末流，倘若你执意要和我们斗下去，我们四大宗门，也未必会怕了你！”
<br/>　　“不错！阁下的修为再强，难道真的要和整个蓬莱仙界为敌不成？”
<br/>　　几人一边说着，一边开始缓缓朝后方倒退。
<br/>　　“说够了吗？那就给我，统统去死吧！”
<br/>　　声音落下的同时，叶尘暴然出手，
<br/>　　轰！
<br/>　　一道紫色剑芒在空中划过，速度快到了极致，直奔五人中，实力最弱的天道宗宗主狠狠劈落。
<br/>　　天道宗宗主根本没有想到，叶尘会突然对他出手，等到他反应过来，已经为时晚矣。
<br/>　　“不！”
<br/>　　伴随着一道凄厉不甘的吼叫，天道宗宗主，被叶尘一剑斩为两段。
<br/>　　不远处的太乙门宗主和逍遥派宗主，看到这突如其来的一幕，顿时吓得朝后方暴退数千米，方才站稳身形。
<br/>　　“叶狂仙！我们好话说尽，没想到你竟然如此不知好歹，既然如此，那就等着承受我们整个蓬莱仙界的怒火吧！”
<br/>　　逍遥派宗主显然已经有些气急败坏，不由得怒吼了一声，转身便逃。
<br/>　　太乙门宗主和天雷宗宗主，同样分别朝另外两个方向远遁而去。
<br/>　　这三人全都是元婴境巅峰的高手，速度并不比叶尘慢多少，如今分别三个方向逃跑，叶尘自然也分身乏术，只能从中选择一个去追。
<br/>　　“哼！”
<br/>　　叶尘在电光火石之间，已经有了决断，直接冷哼一声，最终朝天雷宗宗主上官云阙追了过去。
<br/>　　此人在三人中实力最高，而且天雷宗又和药王宗交好，自然成为了叶尘的首要目标。
<br/>　　刷刷！
<br/>　　叶尘将速度催动到极致后，在九天之上留下道道残影，瞬间便已经追出数十里开外！
<br/>　　然而就在这时，叶尘体内忽然传来一道虚弱之感！
<br/>　　原来，他之前强行催动逆生百转，后来又强行突破金丹境，体内早已经千穿百孔，后继无力了……
<br/>　　不过，既然已经到了这一步，叶尘又岂能轻易放弃？
<br/>　　倘若他突然收手，说不定反而会引起这几个老狐狸的疑心，到时候反将他一军！
<br/>　　一念及此，叶尘又强行提起一道真元，将那股反噬之力压制下去，然后狠狠一剑斩出，
<br/>　　“天地一剑！”
<br/>　　轰！
<br/>　　璀璨夺目的一剑，长达千余米的剑芒，在空中划过一道优美的弧线。
<br/>　　“不！”
<br/>　　上官云阙顿时大吼了一声，他的速度再快，也不可能快过叶尘的紫琼剑，只好祭出法宝迎了上去。
<br/>　　可惜，两人实力相差实在悬殊，几乎没有任何悬念。
<br/>　　咔嚓！
<br/>　　上官云阙所使用的那把权杖，竟然被叶尘的紫琼剑直接斩断，而且去势不减，劈在了上官云阙的身上。
<br/>　　“啊！！”
<br/>　　伴随着一道凄厉的惨叫声响起，上官云阙终究还是死在了叶尘的剑下。
<br/>　　此时已经奔出数十里外的另外两名宗主，听到上官云阙的凄厉惨叫，哪里还敢有丝毫停留，发疯似的狂奔而去。
<br/>　　然而，
<br/>　　他们又哪里知道，叶尘施展出这最后一击后，整个人已经如同虚脱，直接一头栽倒了下去，倘若他们此刻回过头来，叶尘恐怕也只有束手待毙了。
<br/>　　可惜，他们早已经被叶尘吓破了胆，哪里还敢回头，这才让叶尘逃过了一劫。
<br/>　　叶尘半跪在地上，喘息了好一会，这才渐渐恢复了几分力气，不过却依旧头晕目眩。
<br/>　　没办法，他如今体内的经脉破损严重，而且加上之前强行动用逆生百转，巨大的反噬力，更是让他痛苦不堪。
<br/>　　就在这时，
<br/>　　刷！
<br/>　　远处传来一阵躁动！
<br/>　　叶尘还以为是那两个家伙去而又返，顿时霍然站起身来！
<br/>　　不过等他看清楚来人，这才松了一口长气。
<br/>　　原来却是祝明飞，带着祝小白和苏蔓两人赶了过来。
<br/>　　看到这三人，叶尘顿时再也支撑不住，顿时两眼一黑，直接倒了下去……

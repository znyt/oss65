　　第299章
<br/>　　程大雷仔细打量着樊梨花，发现这姑娘不仅有点儿虎，还有点二，又二又虎。
<br/>　　这就是家庭教育缺失造成的后果呐，自己责无旁贷要担起这个教育工作。
<br/>　　程大雷不能再盯着樊梨花看了，再看下去，这姑娘就怯了。
<br/>　　他一振双臂，道：“来，喝！”
<br/>　　酒封拍开，清香阵阵，酒倒入两口粗瓷碗内，程大雷举起碗，将杯中酒一饮而尽。
<br/>　　咚！
<br/>　　程大雷脑袋重重嗑在桌子上，手一松，酒碗啪的一声落地，摔得粉粉碎。
<br/>　　樊梨花手中的酒刚喝了一半，看到这一幕愣了愣，呆呆的看着醉得人事不省的程大雷。
<br/>　　“不会……这么没量吧？”
<br/>　　樊梨花呐呐道了一声，看程大雷生得五大三粗，也不是风一吹就倒的弱鸡仔。可怎么酒量这么小，一碗酒就给放倒了。
<br/>　　如此，倒是令樊梨花有些不知该如何是好。今夜来闯程大雷屋子，本就存了羊入虎口，半卖半送的意思。如此，把身子交给程大雷，为自己，也为山寨老弱找个依靠。
<br/>　　可没想到，程大雷酒量这么浅，自己还想着陪君醉笑三千场呢，结果程大雷一碗就倒。
<br/>　　樊梨花把酒碗搁在桌上，想着又拿起来嘬了一口解渴。看程大雷一眼，见他趴在桌上打鼾，樊梨花越看越气，狠狠在其身上踹了一脚，程大雷仰面倒地，依旧没醒。
<br/>　　樊梨花坐在那里长吁短叹，月色撩人，脸上有了几分女儿愁绪。不知坐了多久，想着程大雷躺在地上不是个事儿，终究有几分不忍，又将他抱在床上，靴子脱了，盖好被子。
<br/>　　盯着睡熟的程大雷，樊梨花握着刀真想一刀捅了他，想了想，还是无可奈何的叹口气，关上门离开。
<br/>　　当天夜里，天还没亮的时候，程大雷就和徐神机一起撤了。
<br/>　　“大当家，咱们现在走没事么？”
<br/>　　“没什么大事，有秦蛮照应着，出不了事。”程大雷骑在马上：“咱们回去照应好山寨，真有什么事，也随时可以联系么。”
<br/>　　“那么有必要夜里走么？”
<br/>　　“呃……这个，哈哈……工作要紧么。”
<br/>　　“我今天好像看到樊寨主去你屋了，她找你干嘛？”
<br/>　　“工作，聊的可不都是工作的事情。”
<br/>　　“我看这樊梨花不错，虽然长得壮点，但人不丑么。大当家你也老大不小，该成个家了，等你成了亲，我去老寨主那边也能有个交待……”
<br/>　　徐神机絮絮叨叨说着，程大雷也不知道他是觉察到什么，还是怎地，只能竖起耳朵听着。随着徐神机啰里啰嗦的声音，两匹马消失在东方的鱼肚白中。
<br/>　　……
<br/>　　牧长庚最近一直在铁匠铺中抡大锤，不得不说，他是一个很用心的人。即便是抡大锤，也能抡出成绩来，很快他就成为铁匠铺一个小小的管事，主要工作是带着大家一起抡大锤。每当看到同事钦佩的目光，听到上司夸奖的话时，牧长庚就觉得浑身上下有用不完的力气，抡起大锤更有劲儿了。
<br/>　　往往这时候，牧长庚都会心中一警：自己来这里的目的，可不是成为一个出色的体力工作者。
<br/>　　他的目的是刺探蛤蟆寨情报，或者刺杀程大雷。关于后者，他暂时没有实施的可能，不过关于前者，他已经找到了情报目标。
<br/>　　目标就是徐神机肚子里的计划。
<br/>　　当然，他一直没有刺探的机会，而对于徐神机这种足智多谋的高人，他也不敢轻易行动，以免引起对方警觉。
<br/>　　不过，这一日，蛤蟆寨倾巢而动，山寨只剩下些文职工作者。这当然是进攻的最好机会，牧长庚还要考虑，自己要不要去琴川关传讯。还没等他考虑清楚，赵子龙几个就又都回来了，所以，这天赐良机也就错过了。
<br/>　　然后牧长庚一直等着，等待徐神机回来，自己继续从他口中打探消息。
<br/>　　到这一日，徐神机终于回来了，牧长庚蓄势良久，当天夜里，备了一坛好酒，让厨房炒了两个菜。
<br/>　　“徐军师，这一路奔波可是辛苦，我特意准备的，好好为您接风。”牧长庚将酒给徐神机添满：“我不是也想着求徐军师指点指点，跟着徐军师多学习学习。”
<br/>　　“嗨，想学我还能不教你么。”徐神机叹了一声：“当然学得好学不好，就要看你天分了。”
<br/>　　“我也知道，我和徐军师的天赋差着十万八千里，不过能从徐军师身上学一丁点，也就够我一辈子受用。”
<br/>　　酒过三巡，菜过五味，徐神机渐渐有些醉意，他的酒量比程大雷也强不了多少。牧长庚一直说着奉承的话，只将徐神机说得天上有，地上无。
<br/>　　徐神机是吃这一套的，被牧长庚一捧，精气神就来了，裂开大嘴，开始胡喷醉侃。
<br/>　　“你可知道，当初戎族十万铁骑，是如何破的……那多亏大当家请来天河水，水淹十万戎族兵呐。”
<br/>　　牧长庚被吓住了：酒后吐真言呐。
<br/>　　牧长庚喝的也不少，看上去也有七分醉，但是，他脑袋里却清醒得很。此刻见徐神机酒已经喝得差不多，他大着胆子开口，双目之中爆射出寒光。
<br/>　　“军师，你上次和我说的事，要想攻破山寨，该从何处下手呐？”
<br/>　　“什么，你说什么？”徐神机舌头已经有些大。
<br/>　　“该如何攻破山寨？”牧长庚半站起来，咄咄逼人的问：“咱们山寨最怕的是什么？”
<br/>　　“最怕什么……”徐神机已经有些醉了，半倒在床上，口里含糊不清道：“怕火啊……”
<br/>　　“怕火？”牧长庚眼睛一亮。
<br/>　　“欲破山寨，当用火攻。”徐神机醉醺醺倒在床上，翻个身呼噜噜睡着了。
<br/>　　可牧长庚却毫无醉意，眼里像是着了火一般，脸上神色变幻。
<br/>　　欲破山寨，当用火攻。
<br/>　　是呐，自己等这么久，吃这么多哭，求的不就是这八个字么。为了得到这八个字，自己受了多少折磨，想想就是一肚子委屈。
<br/>　　今天谜底已经揭晓，自己再无留在这里的必要。牧长庚悄悄潜出房间，一直被人看瘪的铁匠，露出不符合他身份的身手，路上为引起任何人注意，泅水离开山寨。
<br/>

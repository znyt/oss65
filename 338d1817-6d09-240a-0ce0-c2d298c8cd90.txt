　　砰！
<br/>　　一声闷响，地面弹起一震灰尘。
<br/>　　身穿黑衣武士服的影武者，狼狈不堪的躺在地上。他身旁，同时躺着十多个痛苦呻吟的同伴。
<br/>　　苏重深吸一口气平复呼吸，转头看向目瞪口呆的亨利·杜卡：“可以教我下一套格斗术了吗？”
<br/>　　“当……当然可以。”杜克咽了一口唾沫，强自镇定。
<br/>　　数月以来，苏重以让人瞠目结舌的速度，迅速完成影武者训练。
<br/>　　每一项训练内容，最多不超过三天，苏重就能完全掌握。并运用纯熟，能以之击败众多影武者。
<br/>　　过去数月，像刚才这种躺一地影武者的现象，已不止一次上演。
<br/>　　饶是亨利·杜卡见多识广，也被苏重吓得够呛。
<br/>　　他甚至已经有些怕训练苏重。好学生固然让老师高兴，但太妖孽，老师变得教无可教，那就不妙了。杜卡现在就有一种被掏空的感觉。
<br/>　　“你刚完成训练，先回房间休息一下，我们下午继续。”杜卡干咳一声，赶紧把苏重支走。他得好好想想，下午到底该教什么。
<br/>　　苏重笑了笑，微微躬身行礼，从容转身走向房间。
<br/>　　回到房间，关上门，苏重闭上眼睛，唤出技能面板。
<br/>　　暗杀刀术：圆满；暗器投掷：圆满；桑博格斗术：圆满……
<br/>　　一连串圆满格斗技能，充斥苏重整个视线。
<br/>　　“有技能面板固化技能，根本不用担心技能退化。减少大量重复修炼，只要能抓住要点，只要有一次练到圆满，就能永远圆满！”
<br/>　　苏重武道修为，让他迅速掌握一项技能。技能面板固化，保证效果永不消退，节约大量时间。
<br/>　　这就是苏重不到半年，就把忍者大师掏空的原因。
<br/>　　以苏重眼光，能被他看入眼的，也只有杜卡掌握的各种格斗术。
<br/>　　“这些格斗术不涉及超凡力量，纯粹利用身体力量。在本源世界，覅阿尔比较有用。”
<br/>　　苏重虽经历武侠世界，但那里的核心是内力，是体内能量。即便外功，也是铁布衫一类硬功。所有招式，都有其他力量加成。
<br/>　　纯粹依靠常人身体力量格斗，反不如杜卡的格斗术实用。
<br/>　　“可惜，依照杜卡表现，他大概已教无可教。即便有也是杜卡看家本领，不太可能教给自己。”苏重明显感觉到杜卡的戒备。
<br/>　　随着苏重能力展现，杜卡对苏重态度，也从一开始的欣赏，逐渐过渡到警惕戒备。
<br/>　　苏重的进步太快，只要时间足够，很可能会超越杜卡。而苏卡心思难测，杜卡难以掌控，这威胁到了他的地位。
<br/>　　这个世界的忍者大师，不管如何优雅强大，本质上不过是个佣兵头子。是影武者联盟，这个黑暗组织的首脑。并不是dc漫画里，充满传奇色才的千年怪物。
<br/>　　感到苏重的威胁，杜卡立刻改变教导策略。这几天已经在有意识的保留后手。
<br/>　　“不能在刺激这老家伙了，万一触及底线，说不定会下毒手。明天去练一练枪械好了。”
<br/>　　“说起来，穿越这么多个世界，好似还从未练过枪械。”
<br/>　　第二天，基地室内靶场。
<br/>　　砰砰砰……
<br/>　　一连串枪声不绝于耳，苏重一口气打光弹匣十三颗子弹，然后从容不迫的更换弹夹。
<br/>　　亨利·杜卡双手抱胸，保持着和蔼微笑，嘴角抽了抽，眼中满是不可思议。
<br/>　　“50环！”旁边一人高声报靶，想笑又不敢笑，憋的脸一片通红。
<br/>　　杜卡走到苏重身边，拍了拍苏重肩膀：“苏，没事的。枪法可以慢慢练。每个人都有自己的擅长和不擅长的东西，不要太在意。”
<br/>　　苏重面无表情。50环？十三发子弹只有50环，甚至还有一发脱了靶！
<br/>　　即便是苏重自己也没料到，天才妖孽的前身，竟有这么一手渣枪法！自己以前又没练过，两个渣渣合在一起，能打50环已是老天开眼啦！
<br/>　　和之前妖孽表现相比，现在的苏重，笨拙的好像白痴！
<br/>　　“我知道，老师您可以去忙了，我再练一会儿。”苏重阴着脸。
<br/>　　“好好，你练，你练……”杜卡憋着笑，带着一群同样憋红脸的手下，快速离开靶场。
<br/>　　砰砰砰……一连串枪击声再次响起。
<br/>　　之后几天，苏重停止所有格斗训练。有技能面板，他不怕倒退。所有精力，全都放在了枪械射击上。
<br/>　　“他还在练枪？”杜卡一脸轻松。
<br/>　　“是的，已经消耗了五个月的份额了。”黑衣属下欲言又止。这打的不是子弹，可都是钱啊！
<br/>　　杜卡不以为意摆摆手：“我让你查的事情呢？”
<br/>　　“进入佣兵界，经过短期培训后。苏就开始截取刺杀人物。而且喜欢单人行动。每次都能以不可思议的手段，近身格杀目标，并从容退走。至今没有一次任务失败记录。正因此，才被我们外围势力看中吸收。”
<br/>　　“也就是说，他对枪械是真的不精通？”杜卡感觉难以置信。那么天才的人，怎么偏偏枪法不行。
<br/>　　“实际上，他的佣兵训练并未完成，半途就被佣兵组织排除。没人组队，这也是他选择单干的原因之一。”黑衣属下也觉得不可思议。
<br/>　　“因为枪械不过关？”杜卡感觉真是世事弄人。
<br/>　　“是的，枪械掌握不过关，被佣兵组织认为没有培养价值。”
<br/>　　“谁能想到，一直天才妖孽的苏，竟然无法掌握枪械。”杜卡摇头失笑。
<br/>　　现代佣兵不精通枪械，几乎就等于丢了半条命。好在这里是影武者联盟，枪械并不占绝对主导位置。不然，苏重还真没法进影武者基地。
<br/>　　“让他练去吧，那点份额不算什么。”杜卡沉吟半晌，眼神闪动：“准备一下，给他安排一次测试。”
<br/>　　“是！”得到指示，黑衣武士迅速离开房间。
<br/>　　傍晚，苏重坐在床上，仔细回想着练枪新的。可能是这个身体真的不太擅长枪械，即便有技能面板固化技能，苏重进步仍旧如龟爬。
<br/>　　好在，有海量子弹喂养，经验值提升虽慢，却实实在在提升。只要时间足够，弹药管够，他就能把枪械练上来。
<br/>　　“苏，有空吗？”杜卡敲了敲门框，温和对苏重道。
<br/>　　“老师请进。”苏重起身，让杜卡进门入座。
<br/>　　“明天还去练枪？”杜卡有些玩味道。之前苏重表现妖孽，不到一天就掌握一种格斗术，这速度让众人狠受刺激。现在见苏重受打击，他表示喜闻乐见。
<br/>　　“是。”苏重板着脸。
<br/>　　“苏，你要知道，我们的训练资源，全都是有份额的。即便我可以多给你一些，但以你的消耗速度，还是远远不够的。基地的资金并不宽裕。”杜卡开门见山。
<br/>　　“是需要我完成任务，赚取佣金吗？”苏重淡然道。份额？资金不宽裕？影武者联盟会资金不宽裕？开什么玩笑！是要变相支走自己吗？
<br/>　　“不，你误会了苏。基地目前正缺少一名格斗训练助教，你看？”杜卡一脸征询问道。
<br/>　　“助教？”不是把自己赶走？苏重了然，杜卡拿不住自己，所以要把用拴在基地。一个格斗训练助教的身份，苏重短时间内别想离开。
<br/>　　“可以。”苏重没有犹豫。他还想多练练枪呢。至于助教，每天把那些人揍一顿而已，废不了多大功夫。
<br/>　　“好！明天，拉斯·阿古大师会给你安排考核。只要通过，立刻就任助教。到时候，你的各项用度份额，将会大大提高。足够你尽情练习枪械。”杜卡笑眯眯，智珠在握。
<br/>　　考核？这是要用致幻剂考验自己吗？以助教考核名义，试探自己的底线？
<br/>　　我说怎么那么好说话，原来这这里等着我呢！
<br/>　　苏重心底冷笑，一旦自己超出杜卡掌控，明天就不是对他的考核，而是绞杀！

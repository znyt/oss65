        &lt;h3&gt;第三卷第三百一十九章  世间千万事，不及你一笑&lt;/h3&gt;
<br/>
<br/>        陈子昂看完了第二天所有的工作流程和计划确保无一疏漏，抬头外面已经的灯火辉煌。
<br/>
<br/>        站在窗前看着街道上的人来车往，陈子昂微微叹了口气，奋斗了这么多年，他们的公司已经有很多了项目和研发技术都一项项的落地了。
<br/>
<br/>        这些项目和技术都凝集了大家的心血，如今耗材也正式推向市场，这将是一件划时代的事情，看着自己办公室隐于无形中的现代化设备，陈子昂的心头划过一丝骄傲，虽然她是一介平民，虽然她的学历也不高，但是她做到了别人没有做到的，这前半生也不算白活。
<br/>
<br/>        赵绾绾一定要和她一起去甸城，她的心里还是有些犹豫，如果将赵绾绾和呼延寰宇都带走了，这边的人事又要重新做调整。
<br/>
<br/>        赵绾绾和呼延寰宇这一段时间的功劳简直是大大的，将甸城的角角落落各种情况都汇聚起来了，这当然是少不了守中给她的人的功劳。
<br/>
<br/>        他们的绝密铁三角已经发挥了强大的作用。
<br/>
<br/>        明天耗材上市，江俞轩和寒伯安他们都会来，有些事情需要提醒他们注意。
<br/>
<br/>        陈子昂准备离开办公室回家的时候，江俞轩到公司了。
<br/>
<br/>        江俞轩站在陈子昂的办公室门口就是进不去，以前想怎么进就怎么进，现在看着陈子昂站在窗口就是走不进去，玻璃门上没有任何防护，以前刷指纹，现在是什么都没有。
<br/>
<br/>        江俞轩推了推门，纹丝不动，他敲了敲门，陈子昂转过身，看了他一眼，手一挥，江俞轩面前的玻璃门自动开了。
<br/>
<br/>        江俞轩想起来郝景文说陈子昂最近做了一个防护系统，难道这就是防护系统？什么都看不见的防护系统？
<br/>
<br/>        “你不是和大哥他们在一起吗？怎么回来了？”陈子昂知道江俞轩是回到北京，下午和寒伯安他们在一起，这个时候应该是还没有吃晚饭吧。
<br/>
<br/>        “我回来看看你有没有需要帮忙的。”江俞轩拖着行李箱。
<br/>
<br/>        “你还没有回家？”陈子昂看了一眼江俞轩手中的行李箱。
<br/>
<br/>        “没有，下飞机就去找文文了。”江俞轩放下行李箱，直接就到了洗手间洗手去了，边洗手边伸着头看着陈子昂。
<br/>
<br/>        陈子昂还是站在窗前，外面阑珊的灯光透过窗户打在她的身后，显得有些迷离，她的瓜子脸又小一些了，尖尖的下巴微微上翘着，嘴巴倔强的紧紧的抿着，长长的秀发扎成了马尾，静静的垂在肩膀一直延伸到身前，v字形的t恤松散的套在身上，露出了深深的锁骨和白皙的皮肤。
<br/>
<br/>        “晚饭吃了吗？”陈子昂看着探出头的江俞轩问，她发现她和江俞轩陷入了一种奇怪的境地。
<br/>
<br/>        他们在一起的时候太过于亲密和自然，她想疏离都疏离不了，一切都是下意识的。
<br/>
<br/>        陈子昂深深的叹了口气，时间真是个磨人的小妖精，总是在不经意间让你无法自拔。
<br/>
<br/>        “最近又瘦了？”江俞轩走出洗手间站在陈子昂的面前。
<br/>
<br/>        “哪有，一直都是这样的。”陈子昂偏了一下头躲过了江俞轩的视线。
<br/>
<br/>        “我还没有吃晚饭，你也没有吃，我们一起！”江俞轩的口气不容置疑，今天就是要和你一起吃饭。
<br/>
<br/>        陈子昂张了张嘴，什么也没有说，她走到办公桌前，低头将资料整理了一下，然后才对江俞轩说：“好吧，吃完饭送你回家。”
<br/>
<br/>        俩个人走出办公室，迎面碰到了赵绾绾和呼延寰宇。
<br/>
<br/>        “哇，老江，你回来啦？”赵绾绾夸张的和江俞轩打招呼。
<br/>
<br/>        “江总，你回来了？”呼延寰宇微微一笑，这俩个人站在一起一般高，看起来很养眼。
<br/>
<br/>        “我要不是不回来你们岂不是将我忘记了？”江俞轩笑着说。
<br/>
<br/>        “想忘记你不容易啊，喜糖还没有吃呢。”赵绾绾笑着捶了一下江俞轩的胳膊。
<br/>
<br/>        “就不想拥抱一下？”江俞轩张开怀抱。
<br/>
<br/>        “不，你已经抱了别人，不香了。”赵绾绾撇了撇嘴，这家伙之前隐藏的够深的，不但是江家大少，还有未婚妻。
<br/>
<br/>        大家一直以为他和陈子昂是一对，简直刷新了大家的三观，一点也不好玩，白白浪费他们的感情。
<br/>
<br/>        “我谁也不抱，就想抱抱你，都好久没有见面了，没爱了没爱了。”江俞轩逗着赵绾绾，他们在一起打拼了很多年，彼此很熟悉。
<br/>
<br/>        “没爱了就没爱的，反正我不稀罕你。”赵绾绾走过去挽着陈子昂的胳膊。
<br/>
<br/>        “我们要去吃饭，你挽着子昂的胳膊干啥？这么晚了，改干啥干啥去。”江俞轩开始赶赵绾绾了。
<br/>
<br/>        “吃饭啊，好啊，我们也没有吃，大家一起，你请客，当给我们送行。”赵绾绾已经磨叽着让陈子昂答应她一起去甸城，心情特别的好。
<br/>
<br/>        “送行？你要去哪里？”江俞轩问赵绾绾。
<br/>
<br/>        “我和子昂一起，子昂到哪里我就到哪里，嗯哼！”赵绾绾得意的一笑，陈子昂走了大本营交给江俞轩了，江俞轩是不能去甸城的。
<br/>
<br/>        “子昂，她也跟你一起去甸城？”江俞轩看着陈子昂，为什么你答应赵绾绾去都不让我去？
<br/>
<br/>        “绾绾和寰宇都是信息上的好手，过去也很不错，她要求过去呢，我还正准备和你商量。”陈子昂看了一眼挽着她胳膊的赵绾绾微笑着说。
<br/>
<br/>        “也好，多一个人多一个帮手，反正你们的软件我也不懂，你们就一起走吧，留下我孤家寡人在这里守着吧。”江俞轩期期艾艾。
<br/>
<br/>        “什么叫孤家寡人啊，公司这么多人任你调遣，你还想咋地？”赵绾绾不满的瞪了一眼江俞轩。
<br/>
<br/>        四个人说说笑笑的去了他们经常去的火锅店，陈子昂和呼延寰宇要了一瓶白酒，赵绾绾和江俞轩不喝酒，要了一点葡萄酒。
<br/>
<br/>        四个人边吃边聊，聊着聊着就说到了去甸城的事情。
<br/>
<br/>        “原本我不想你过去，太危险，我们只是一个生意人，但是你想过去，我们就陪着你了，多一个人多一份力量。”赵绾绾喝了一点葡萄酒，脸上浮起两朵红云，显得非常可爱。
<br/>
<br/>        “太危险？怎么回事？”江俞轩抓住了赵绾绾话里的重点。
<br/>
<br/>        “嗯，嗯。这里就不要多说了，明天我会告诉你的，对了，绾绾将软件已经测试完成了，你们的租赁业务可以开展了，运行起来还不错，我还体验了几回。”陈子昂岔开了话题。
<br/>
<br/>        江俞轩有些不满陈子昂岔开了话题，正准备开口，呼延寰宇笑着拿起酒杯：“江总，我们此次征战前方，后背就叫给你了，来，我敬你一杯，喝酒！”
<br/>
<br/>        江俞轩拿起酒杯傲娇的说：“不喝，合着你们都知道，就我什么都不知道。”
<br/>
<br/>        “俞轩，该让你知道的自然会让你知道，你何必计较呢？有些事情一两句话说不完，我此次去甸城你早都知道是为何，寰宇都说了，我的后背都交给你了。”陈子昂拿起酒杯，冲着江俞轩笑了一下。
<br/>
<br/>        这一笑，让江俞轩似乎回到了初见之时，涌起甜甜的暖意，世间千万事，不及你一笑。
<br/>
<br/>        陈子昂看着江俞轩心里默默的想着：我与你一起在商场上征战了多年，是你陪着我走过人生的艰难，多希望在春暖花开的时候，我还能与你一起肩并肩，看流萤飞舞，赏春花满山，多希望晨起携手烹清粥，晚来夕阳落田园，此生有你心易安，此生有你心难安！

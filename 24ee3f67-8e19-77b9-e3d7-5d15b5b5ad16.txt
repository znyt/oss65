说到这个话题，苏希手里的红酒杯倏然一顿，俊美的眉心落了一层冷鹜阴霾。
<br/>
<br/>        作为海城四大豪门之一，苏希当然是知道陆家的，但是很可惜，他知道的并不比外人多。
<br/>
<br/>        陆家少主陆寒霆过于低调神秘，从来没有出现在大众的视野里，苏希曾经派人去打探过，只得知陆寒霆是这几年才来到海城的，他真正的归属是…帝都城。
<br/>
<br/>        帝都城是最繁华的金融都市，大街上随便遇到一个人都可能是富二代出身，一般的豪门根本触不到金融中心。
<br/>
<br/>        当然，帝都城也有自己的顶级豪门，那种顶级豪门是真正的商界巨子，掌握着全亚洲的经济命脉。
<br/>
<br/>        苏希曾经听闻过，帝都城第一商界贵胄之家，也姓陆。
<br/>
<br/>        这一切，都很巧，是不是？
<br/>
<br/>        其实海城根深蒂固的权阀世家还是顾家，顾家少主顾夜瑾，又是跟陆寒霆穿一条开裆裤长大的。
<br/>
<br/>        夏小蝶已经感觉到了苏希情绪的变化，她继续努力道，“苏希哥哥，难道你忘了夏夕绾在乡下跟一个男人在一起，两个人还在山洞里睡了一夜的事情了么，当时夏夕绾才多大啊，就跟野男人乱搞了。”
<br/>
<br/>        苏希将杯子里的红酒一饮而尽，然后伸手一推，直接将夏小蝶给推倒了床上。
<br/>
<br/>        苏希的动作很粗鲁，夏小蝶的脑袋直接“咚”一声撞在了床柜上，她疼的都要流眼泪了。
<br/>
<br/>        这时苏希压了上来，他双目赤红的盯着夏小蝶看。
<br/>
<br/>        夏小蝶被苏希的样子吓到了，每一次她说夏夕绾跟别的男人在一起后，他都会这样失控，变得很可怕。
<br/>
<br/>        但是夏小蝶真的很喜欢苏希，所以她伸手搂住了苏希的脖子，“苏希哥哥，我爱你，你是我唯一的男人，我的第一次都是给你的。”
<br/>
<br/>        苏希俊脸狰狞，没错，夏夕绾背叛了他，就是一个水性杨花的女人，根本就不值得。
<br/>
<br/>        他要忘记夏夕绾！
<br/>
<br/>        苏希伸手去扯夏小蝶身上的睡衣，夏小蝶开心的抱着他，“苏希哥哥，我要永远跟你在一起。”
<br/>
<br/>        苏希闭上眼，脑海里挥之不去的还是夏夕绾的容颜，那一年她还很小，夏夕绾的妈咪带着她来苏家做客，夏夕绾妈咪问他，苏希，以后绾绾给你做老婆好不好？
<br/>
<br/>        当时已经是大男孩的他脸都红了，害羞的一句话都没说，直接进了房间。
<br/>
<br/>        那一天，两个人娃娃亲就这样定了下来。
<br/>
<br/>        他还记得夏夕绾小时候乖软聪颖的模样，仰着盈亮黑漉的眸子看着他，对他笑苏希哥哥。
<br/>
<br/>        苏希将俊脸埋在夏小蝶的发里，嗓音低哑的呢喃出一个人的名字绾绾…
<br/>
<br/>        夏小蝶听到了这一声“绾绾”，仿佛有一盆冷水从她的头顶直接扑了下来，淋灭了她所有关于爱情与幸福的幻想。
<br/>
<br/>        ……
<br/>
<br/>        夏小蝶已经睡着了，黑暗里苏希倚靠在床头，伸手点燃了一根烟，幽幽的抽着。
<br/>
<br/>        等一根烟快抽灭的时候，苏希翻出了手机，拨去了一个电话号码。

　　嗖…
<br/>　　萧逸的身影，直朝远处幽光而去。
<br/>　　还未靠近，一头庞然大物，已横亘而来。
<br/>　　“九寒幽蟒？”
<br/>　　几乎是庞然大物横亘而来的瞬间，萧逸已认出了这头妖兽。
<br/>　　庞然大物身上那惊人无比的气息，更是让萧逸确定了自己没有看错。
<br/>　　九寒幽蟒，绝世妖兽，这可是有名的凶兽，但却极其稀有。
<br/>　　若是萧逸全盛之时，对付这头九寒幽蟒，并非难事。
<br/>　　可他现在这般状态，如何对付？
<br/>　　“不管了。”萧逸心头一狠，瞬间而出。
<br/>　　吼…
<br/>　　九寒幽蟒，怒吼一声。
<br/>　　作为绝世凶兽，它即便在一些闻名中域的大型险地，都是霸主级别的妖兽。
<br/>　　这区区一个小湖，一个生机淡弱得如若将死的人类武者，竟敢挑衅它？
<br/>　　吼…
<br/>　　九寒幽蟒的吼声之下，道道白色寒光冲击而来。
<br/>　　沿途所过，湖水悉数冰封。
<br/>　　萧逸不为所动，他现在最需要的，便是这些寒冷气息。
<br/>　　湖水，一路冰封而至。
<br/>　　以九寒幽蟒的实力，在这湖底之下，即便是一个绝世强者，都会在它的吐下之下冰封结冰。
<br/>　　可，当它的吐息完全靠近萧逸时，却是瞬间被快速融化。
<br/>　　“舒服。”萧逸不禁长长呼出一口气。
<br/>　　不愧是九寒幽蟒，绝世妖兽，吐出的寒冷气息，果真冰冷异常。
<br/>　　只是，这些寒冰气息，仍旧未能替萧逸完全抵消掉剧烈的灼热感。
<br/>　　嗖…
<br/>　　萧逸的身影，疾速而前。
<br/>　　在湖水之内，宛若一道一闪而过的‘水箭’。
<br/>　　萧逸的手，瞬间搭在了九寒幽蟒巨大而狰狞的头颅之上。
<br/>　　嘶…
<br/>　　同样只是一瞬间。
<br/>　　整头九寒幽蟒，瞬间被融化成灰烬。
<br/>　　“好猛。”萧逸吓了一跳。
<br/>　　堂堂绝世妖兽，竟是触碰一下，便被秒杀。
<br/>　　九寒幽蟒的身死，同样再度让得萧逸身上灼热感减缓了不少。
<br/>　　九寒幽蟒，可是冰属性妖兽中的佼佼者。
<br/>　　它的妖兽血液、内丹，则是寒冰属性中的重宝。
<br/>　　嗖…
<br/>　　萧逸身影一闪，瞬间落入湖底幽光之内。
<br/>　　九寒幽蟒，虽是绝世妖兽，但要彻底缓解萧逸身上灼热感，还是远远不够。
<br/>　　“嘶。”萧逸看着湖底这大堆寒晶石，不禁倒吸一口凉气。
<br/>　　寒晶石，有石之名，却非石状，而是通体晶莹剔透，宛如宝石，散发着的幽光，如若白色幻光，美丽而绚烂。
<br/>　　寒晶石，如若灵石。
<br/>　　但灵石内，蕴含的是精纯的灵气。
<br/>　　而寒晶石，蕴含的则是精纯的冰属性力量。
<br/>　　“吸。”萧逸几乎没有半分犹豫，瞬间吸收起来。
<br/>　　难怪这小小的妖兽之湖内，连九寒幽蟒这等绝世妖兽都存在。
<br/>　　如此大堆的寒晶石，对妖兽而言，不亚于一份天大的宝藏。
<br/>　　若萧逸能悉数吸收，即便是以他的庞大气泉，怕是都足以连跳数重修为。
<br/>　　可惜，现在他身处龙炎反噬。
<br/>　　这些寒晶石，只能用于抵消龙炎炙热，并无法用于吸收化作元力和修为。
<br/>　　……
<br/>　　半月时间，转瞬便过。
<br/>　　六龙城，某密室内。
<br/>　　毒袍看向三长老，淡漠地问了一句，“如何？”
<br/>　　三长老，点了点头。
<br/>　　“桀桀。”毒袍，霎时桀桀冷笑。
<br/>　　……
<br/>　　六龙地域之外，某大湖之内。
<br/>　　湖底。
<br/>　　“呼。”萧逸重重呼出一口浊气。
<br/>　　湖底之下，原本大片的寒晶石，已悉数融化殆尽。
<br/>　　但同时，他体内原本剧烈的炙热感，被抵消大半。
<br/>　　体内原本冲撞不堪的元力，已然被稳下。
<br/>　　但，他的身体，仍旧通红炙热。
<br/>　　他体内的龙炎反噬，仍旧未能悉数全消。
<br/>　　归根结底，龙炎反噬，着实可怕。
<br/>　　本身，龙炎就非人类武者所能驾驭。
<br/>　　当年，萧逸也借用过龙炎，以过空间乱流。
<br/>　　但，那仅仅是借用罢了，乃是通过八龙焚火炉第一道封印的解开，而借用些许。
<br/>　　可半月前与龙啸一战，萧逸可是完全驾驭。
<br/>　　自然的，妄图驾驭龙炎，其后果严重如斯。
<br/>　　本来，仅仅一丝龙炎，不可能爆发如此可怕的反噬。
<br/>　　但，萧逸以九阳轮回诀的功法，强行将龙炎作为第七种强悍火焰，融入到自身。
<br/>　　要知道，九阳轮回诀，本身就是驾驭火焰之法。
<br/>　　他的紫晶灵炎、十界灭生火等等世间强悍火焰，皆是先驾驭之后，方能自成轮回，凝聚成圆环于萧逸手腕间。
<br/>　　故，龙炎融入轮回圆环中，本身就等同萧逸掌控了的火焰。
<br/>　　也是在那时，原本一丝龙炎，已飙升至不亚于其它六种强悍火焰的澎湃度。
<br/>　　那股滔天龙炎，赋予了萧逸与龙啸一战之力。
<br/>　　可之后的反噬，却几乎要将萧逸吞噬焚烧殆尽。
<br/>　　“呼。”萧逸不禁再次呼出一口浊气。
<br/>　　他体内的龙炎反噬，仍旧未完全除去。
<br/>　　但，萧逸已不必担心。
<br/>　　锵…湖底之下，一道惊天剑鸣，陡然响起。
<br/>　　一把锋利之剑，散发着绚烂蓝芒，陡然照耀整个湖底。
<br/>　　正是冰鸾剑。
<br/>　　之前，他意识模糊，又一时极度着急下，这才忘了冰鸾剑。
<br/>　　论及世间冰冷之物，萧逸还未见过有什么东西比得过冰鸾剑。
<br/>　　事实上，他一时急了罢了，若他真的接近身死的话，体内冰鸾剑武魂，会自主触发的。
<br/>　　武者的武魂，与武者乃是生死一体，自觉醒后，便伴随武者一生。
<br/>　　故武魂，会自动触发。
<br/>　　以冰鸾剑的寒冰力量，以及萧逸自己本身体内的庞大元力，要抵御龙炎反噬，应当不成问题。
<br/>　　不过，他在这湖底之下，发现如此大堆的寒晶石，倒是给他省了不少功夫。
<br/>　　若没了这大堆寒晶石，萧逸也不知道自己要耗费多少元力还有天材地宝，方能不断激发冰鸾剑的寒冰气息。
<br/>　　此时，冰鸾剑凝聚而出。
<br/>　　当然，并非调动剑内力量，仅仅凝聚出剑身罢了。
<br/>　　萧逸体内元力源源不断输入剑内，剑身之上，极致冰冷汹涌而出。
<br/>　　边激发着剑上寒冰气息，萧逸边大口吞服着丹药。
<br/>　　要完全抵消龙炎反噬，所需寒冰气息，绝非少数；他也不知晓自己体内元力是否吃得消，自然边吞服大量丹药，以补充元力。
<br/>　　嘶…嘶…嘶…
<br/>　　丝丝寒冰气息，不断涌入萧逸体内。
<br/>　　那抹极致冰冷，以极快的速度，抵消着萧逸身上的灼热感。
<br/>　　其速度与效果，远远超过了寒晶石的效果。

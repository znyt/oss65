        郭裕一听苏漫雪这话，顿时十分感兴趣地起身坐到了苏漫雪的身边。
<br/>
<br/>        “傅大少奶奶，说来听听？”
<br/>
<br/>        “台上那个女孩，一看就知道她是还没从学校毕业的，不谙世事。你唐突地让她陪酒，她肯定不买你的账。所以，你得来点软的！”苏漫雪咧嘴一笑。
<br/>
<br/>        就连一旁的陈思伊顿时也感兴趣起来，不禁插嘴问道：“漫雪，快点说来听听！”
<br/>
<br/>        苏漫雪会心一笑，朝郭裕和陈思伊勾了勾手指，示意他们凑过来侧耳倾听。
<br/>
<br/>        而这边诺筱颖唱完从台上下来后，收到了服务员送来的一束鲜艳的玫瑰花。
<br/>
<br/>        玫瑰花上还挂着一张卡片：小谢，你的歌声真好听！加油！
<br/>
<br/>        诺筱颖看着卡片上的字，不由地嘴角微扬，在外能收到别人的鼓励，她真的很欣慰。
<br/>
<br/>        此时，郭裕端着两杯酒朝诺筱颖走了过去。
<br/>
<br/>        同时就在这个时候，傅昼景和自己的几个朋友也进了夜魅娱乐城。
<br/>
<br/>        当傅昼景看到诺筱颖，准备走过去跟诺筱颖打声招呼的时候，只见有个彬彬有礼的男人已经出现在她身边了，便停下了脚步。
<br/>
<br/>        “昼景，看什么呢？走，我们为了庆祝你成为地利置业的总经理，早就订好包厢了！”好友钱承俊拍着傅昼景的肩头说道。
<br/>
<br/>        傅昼景回过神来，点了点头。
<br/>
<br/>        钱承俊又接着感慨道：“刚刚台上那女人唱得不错！以前没见过，应该是新来的！”
<br/>
<br/>        “嗯！”傅昼景淡淡地应了声，若有所思地和钱承俊并排走在了一起。
<br/>
<br/>        余光里，他看到诺筱颖正和那男人有说有笑。心里惆怅，她为什么要来这种地方卖唱？
<br/>
<br/>        她的脚不是受伤了吗？白天还在他面前装出一副楚楚可怜的样子，到了晚上，就来娱乐会所勾搭上了市长的儿子。
<br/>
<br/>        “郭先生，很高兴你能喜欢我的歌声，但是这玫瑰花，我不能收。”得知是郭裕送给自己的这束玫瑰花后，诺筱颖委婉地谢绝了。
<br/>
<br/>        苏漫雪说，这个女人对那种非常有绅士风度的男人比较有好感。
<br/>
<br/>        于是郭裕决定先装成一副温文儒雅的贵公子，跟诺筱颖套近乎。
<br/>
<br/>        “小谢，我没别的意思。就觉得你歌声好听，送束花鼓励鼓励你！”郭裕接着说道。
<br/>
<br/>        诺筱颖看着郭裕，只见他身着浅蓝色西装礼服，剪着平头，衣冠楚楚，相貌堂堂的样子，便接受了郭裕的这份鼓励。
<br/>
<br/>        “谢谢你。”诺筱颖微微一笑，单纯地收下了这束玫瑰花。
<br/>
<br/>        郭裕接着将酒杯递给了诺筱颖：“来，我们干一杯！”
<br/>
<br/>        “不了，我不会喝酒。”诺筱颖接着拒绝道。
<br/>
<br/>        郭裕咧嘴一笑：“这不是酒，是饮料，你在台上唱了那么久，也口渴了吧？来，尝尝，很好喝的。”
<br/>
<br/>        说到口渴，诺筱颖还真感觉有点口渴了。她完全没察觉到郭裕是只笑面虎，在郭裕的连哄带骗下，她毫无防备地喝下了郭裕递过来的这杯饮料。
<br/>
<br/>        诺筱颖喝完这杯饮料，被动地跟郭裕有话没话地聊了一会儿，才感觉头有点儿晕，甚至有些站不稳脚。
<br/>
<br/>        郭裕连忙搀扶着住她，微笑着说：“你一定是唱累了，我带你去楼上休息一下。”
<br/>
<br/>        “谢谢，不了。我得去找我……”嫂嫂。
<br/>
<br/>        诺筱颖话还未说完，觉得头越晕得更厉害了，有种天旋地转的错觉。
<br/>
<br/>        “我先扶你上楼去休息吧！我在楼上有VIP包厢，你先去休息休息。”郭裕一脸淫笑。
<br/>
<br/>        不知不觉，诺筱颖被郭裕搀扶着带到了楼上私人休息区，这里是VIP客房包厢。
<br/>
<br/>        苏漫雪说诺筱颖可是非常“纯”的，还未被任何男人碰过，这让郭裕对诺筱颖的占有欲越来越大。
<br/>
<br/>        郭裕将诺筱颖带进客房后，便粗鲁地将她推倒在了大床上。
<br/>
<br/>        他已经迫不及待地想要尝尝这个含苞待放的女孩是什么滋味了！
<br/>
<br/>        雪纺质地的裙摆，抬至了她的大腿处，白皙细腻的皮肤，姣好的身材，别是另一番少女般新鲜的味道。
<br/>
<br/>        郭裕看得热血沸腾，如狼似虎地脱着衣服。
<br/>
<br/>        诺筱颖迷迷糊糊地睁开眼睛，看到郭裕露出禽兽的面目，不由地一阵心悸：“你……”
<br/>
<br/>        郭裕十分饥渴地往诺筱颖身上扑去，想要对诺筱颖上下其手。
<br/>
<br/>        诺筱颖使出浑身解数推了推郭裕，心里虽乱成了一团，但头脑却是清醒的，故作娇嗔地说：“别急嘛！”
<br/>
<br/>        听到诺筱颖这一声柔到骨子里的声音，郭裕倏地停了下来。
<br/>
<br/>        “你先去洗澡……”诺筱颖勾唇一笑，吃力地抬起手来，摸了摸郭裕的脸颊，“你一身汗臭味……”
<br/>
<br/>        郭裕听着顿时喜上眉梢：“你想跟了我？”
<br/>
<br/>        “能进这里来消费的人，非富即贵。我跟着你，有肉吃！”诺筱颖故意让自己表现得跟拜金女一样。
<br/>
<br/>        郭裕心里乐开了花，狠狠地在诺筱颖的脸蛋上亲了一口：“亲爱的，你伺候我洗澡！”
<br/>
<br/>        “人家第一次，会害羞，你先去洗。我等你……”诺筱颖努力让自己扯出一抹微笑。
<br/>
<br/>        郭裕似乎是信了，于是转身去了卫生间。
<br/>
<br/>        诺筱颖随即起身，跌跌撞撞地朝门口跑去。
<br/>
<br/>        她才跑出门口不到几步，便被郭裕给抓住了手腕。
<br/>
<br/>        “我就知道你这女人不是省油的灯！想骗我，门都没有！”郭裕恶狠狠地说，为了不引起骚动，他又假惺惺地接着说道，“亲爱的，别走，我错了还不行吗？不要跟我闹了，快跟我回去！”
<br/>
<br/>        “你放开我！”诺筱颖挣扎着，但是头晕得厉害，浑身渐渐地感觉到了无力。
<br/>
<br/>        “我送给你的花，你是不是不喜欢？”郭裕又问道，他说着，一边拉着诺筱颖往自己的VIP客房包厢走去。
<br/>
<br/>        诺筱颖本身脚上还带伤，刚刚又急于逃跑，脚上的伤似乎又加重了。她一个踉跄，跌坐在地上。
<br/>
<br/>        郭裕忙蹲下身去，搂住诺筱颖的腰肢，衣冠禽兽地问：“亲爱的，有没有伤到哪里？”
<br/>
<br/>        就在这时，对面VIP客房包厢里走出来几个男人。
<br/>
<br/>        诺筱颖抬眼望去，看到他们当中的某个人时，顿时眼前一亮。

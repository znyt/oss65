　　“捉鬼联盟里都是我最好的朋友，都是人类，阴阳司都是我的门人，和他们收的小弟。我的门人，有鬼、妖、邪灵，大家都是兄弟姐妹，你也不用叫我主人，叫我老大就行。我们的规矩……只有一条，不管对手是谁，共同进退，一个都不能丢下。不存在什么为了我牺牲自己之类的，你记住了？”
<br/>　　花童怔怔地看着他，道：“为主人牺牲自己，不是理所当然的么？”
<br/>　　“在我这，没有这种事。反正你现在也不认主，你先在我这呆着，我正好考察你一段时间。”
<br/>　　“好吧，我记得了。主……老大。嘿嘿，你刚才说，你有很多门人，那我排第几呢？别人都是谁，跟我说说呗。”
<br/>　　“你排……第十个，关于他们的情况，以后总有机会说的。我现在没心情。”
<br/>　　“哦哦好吧。”花童冲美华眨了眨眼睛，道：“现在我可是你弟弟了，你以后要照顾我。”
<br/>　　美华无语，对叶少阳道：“我真有点羡慕了，我要真正当你的门人，还要等九十多年。”
<br/>　　“额，这个吧，其实我最早收的就是你，等我回去之后，我又能见到你……”
<br/>　　“那不一样，虽然百年后的我也是我，但是……跟现在的我，其实又不是一回事，至少我没有那些记忆。等你离开了，我还要等九十多年才能遇到你……”
<br/>　　美华细细说着，突然想到一件事，问道：“老大，你说到了那个时候，我见到的你……跟你还是一个人吗，你应该听懂我的意思。”
<br/>　　叶少阳也怔住了，思索了半天，摇摇头道：“我不知道。”
<br/>　　这个问题，对叶少阳来说实在有点困惑，在自己那个时代，穿越类型的小说和影视很多，自己虽然是山上长大的，从小也看过不少，各种脑洞，设定也都是导演说了算，现在自己真遇上这种事，叶少阳整个人完全是懵逼的。仔细想了半天也给不出答案。
<br/>　　美华自己也歪着脑袋琢磨起来，说道：“举个例子，假如你在这里留下来，陪我一起呆上九十多年，到我们人是的那个时候……你说，我还能遇到另外一个你吗？”
<br/>　　叶少阳表情有点呆滞，美华的假设，对他来说脑洞实在有点太大了。
<br/>　　美华看着他呆呆的样子，笑道：“好了老大，不要想这么没用的事情了，我也就是好奇，随便想了一下，其实不管是不是现在的你，对我来说都是一样的……因为你就是他，他也是你，就像那时候的我一样，终归是我自己。”
<br/>　　“什么你你我我的，你们不要讨论这么奇怪的问题吧！”花童抓着叶少阳的手，说道，“老大，我还没有名字，你给我取个名字吧。”
<br/>　　取名字……
<br/>　　叶少阳一时有点犯难。自己哪里会取什么名字，当下各种推脱，表示自己没好好读书，学问不行。
<br/>　　“随便啊，老大，我又不是去学堂读书，不要那种正式的名字，你就随便取个名字给我就好。”花童各种纠缠。
<br/>　　实在没法推脱，叶少阳只好冥思苦想起来……
<br/>　　“有了！孢子，你不是孢子出身吗，正好长得也挺像包子的，你就叫包子好了！能吃的那种包子！”
<br/>　　“包子……”花童露出忧伤的表情。
<br/>　　“挺可爱的名字啊，先叫这个吧。将来想到好的再改吧。”叶少阳实在不想在这件伤脑筋的事情上花太多精力。
<br/>　　“就这个吧。不改了。”花童无精打采地挠了挠头，“我这是讨口才，你是我主人嘛，金口一开，就不能改的了。好啦，我好歹有个名字了，我以后就叫包子啦！”包子到最后居然开心起来。
<br/>　　叶少阳突然想到了瓜瓜，自己门下本来就有个瓜瓜，现在又多了个包子，两人还一样都是男童的外形，将来凑成一对，可真是够好玩的，都是吃的东西。哦，还有个橙子。蔬菜水果食物……这下都有了。
<br/>　　想到橙子和瓜瓜，叶少阳心中又燃起了思念。
<br/>　　你们……在等着我回去吧？
<br/>　　“对了包子，你知不知道那个碧清的来历？”叶少阳想起了碧清，于是问道。
<br/>　　“不知道。其实，我……应该说是之前的佛子花，跟她并不熟。佛子花本来长在西域，在小昭寺附近的一棵柏树下，是被一个黑教弟子移植过来，镇守古墓的……”
<br/>　　包子的神识中，遗传了之前那朵佛子花的部分记忆，对当年的事也是知道一些。几百年前，通灵佛子花被一个苯教的宗师找到，移植到古墓里，并且下了强大的禁制，让它只能以本体的姿态长在花坛里，一步也不能离开（禁制就在花坛下面的泥土中，里面那一具尸体的身上，缠绕住了通灵佛子花的根系）。
<br/>　　碧清，只大概百年之后才进入古墓的，找到炼尸缸，与里面本来有的邪物大战一场，当时没能打得过，逃走了，后来趁着那邪物渡劫的时候出手将其击杀，将它一身修为都吸收了，本来炼尸缸里因为杀人无数，怨气极为充沛，是人世间十分罕见的大凶之物。碧清占据之后，就作茧自缚，在里面躲着修炼，再也不出去了。
<br/>　　通灵佛子花的存在，正好能为她充当一道守卫，万一有人闯进古墓，那里是必经之路，通灵佛子花也可以守上一道，因此，碧清对通灵佛子花的存在不闻不问，也没有伤害它……
<br/>　　至于把它从西域强行带到古墓里来的那个黑教宗师，还有有关古墓里的一切，当时的通灵佛子花并不知道太多，包子就更不知道了。
<br/>　　不过包子说的这些，还是解决了叶少阳心中的一些困惑，大致知道了怎么回事。
<br/>　　叶少阳又问起妙心的祖先当初进入古墓的经过。包子也是知道的，那个时候，通灵佛子花还没有太成气候，而且那群人中的老头（包子的原话，叶少阳猜测说的多半就是那位父子中的父亲），法力极深，破了通灵佛子花的幻境，没有灭了它，是想着收为己用，打算出去古墓的时候把他带上，结果最后死在了地宫里……

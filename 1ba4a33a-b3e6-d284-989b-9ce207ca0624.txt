        俗话说，没有对比，就没有伤害。
<br/>
<br/>        童筱颖只要负责吃就行了，傅夜沉全程为她涮锅，就连虾蟹贝类的壳，他都负责剥得一干二净了才放入她的碗中。
<br/>
<br/>        在傅昼景那儿，她可从来没享受过这样“公主级别”的待遇。
<br/>
<br/>        以前傅昼景还会替她挑鱼刺，但是两个人婚后相处久了，傅昼景在一些细节上便渐渐地开始对她怠慢了。
<br/>
<br/>        看着傅夜沉这么辛苦又殷勤地替她又涮又剥，他自己没吃几口，童筱颖主动夹起碗中的虾肉，送到了傅夜沉的嘴边。
<br/>
<br/>        “看在你这么‘辛苦’的份上，我喂你好了！”她手里拿着筷子，胳膊抬在半空中，微笑着看着他。
<br/>
<br/>        傅夜沉受宠若惊般地睐了童筱颖一眼，下一秒就咬走了童筱颖的筷子上的虾肉。
<br/>
<br/>        这一来二去，候在一旁的女服务员看明白了他俩的关系后，便不再对傅夜沉一个人献殷勤了。
<br/>
<br/>        他俩刚进来的时候，她还以为他俩是兄妹，现在看，倒有点像夫妻了。
<br/>
<br/>        人，不能吃太饱，吃太饱了，就会想睡觉……
<br/>
<br/>        饭后，童筱颖坐在副驾驶座上还没十分钟，就“呼呼”地睡了过去，完全忘记了自己下定决心要跟傅夜沉分居两年而选择逃跑的事情。
<br/>
<br/>        因为中午的那餐饭吃得太久，傅夜沉看了看腕上的时间后，决定先带童筱颖一起去公司，然后让司机再送她回私宅。
<br/>
<br/>        他两点半的时候，有个跨国视频会议要准时开。
<br/>
<br/>        车子驶入傅氏集团大厦的地下室车库后，傅夜沉停稳了车，从驾驶座上下来。
<br/>
<br/>        当他绕过车头，拉开副驾驶座的车门，发现童筱颖还在睡，忍不住笑了笑，索性替她解了安全带，将她从副驾驶座上横抱了出来。
<br/>
<br/>        傅夜沉双手横抱着童筱颖，让她靠在了自己的胸膛上，他好用自己的后背轻轻地推关车门。
<br/>
<br/>        动作一气呵成，轻而易举。
<br/>
<br/>        全过程，他都是小心翼翼的，生怕将她吵醒，扰了她的午休。
<br/>
<br/>        他还发现，他的筱颖比以前要轻了许多，看样子，从今以后，他得把她养胖点才行。
<br/>
<br/>        傅夜沉抱着童筱颖，到了自己的总裁办里二楼休息室，他刚轻轻地把她放床上，她就醒了过来。
<br/>
<br/>        看着她揉眼睛，扭动着身子，一副未睡醒的样子，他拉开她正在揉眼睛的手，温柔地哄道：“筱颖再睡一会。”
<br/>
<br/>        “小卫……”童筱颖下意识地唤了声，然后拉住傅夜沉的手，侧着身子，将他的大手贴在自己的脸颊与枕头之间，又缓缓闭上了惺忪的睡眼，呢喃呓语道，“我好想你，你在哪……”
<br/>
<br/>        潜意识里，她已经将傅昼景和龙之卫臆想成了两个不同的男人。
<br/>
<br/>        傅昼景是季思妍的，龙之卫则是她童筱颖的。
<br/>
<br/>        那个和她自幼相识，长大后在火车站里再次邂逅，免费送她火车票，在滨江城里关照她，后来帮她还债，娶她为妻，一路与她同甘共苦的龙之卫，现在去了哪里呢？
<br/>
<br/>        是啊！
<br/>
<br/>        她的小卫，去了哪儿？
<br/>
<br/>        一滴晶莹剔透的泪珠，从童筱颖的眼睑中溢出，滑过她的鼻梁，滚过她的脸颊，最终落在了傅夜沉的掌中。
<br/>
<br/>        她口中的“小卫”，他知道是谁，阿景小时候的曾用名“龙之卫”。
<br/>
<br/>        傅昼景的母亲名叫龙半夏，他们的父亲名叫冷卫峰。
<br/>
<br/>        父亲的乳名是“阿卫”。
<br/>
<br/>        而“龙之卫”，这个名字的寓意是什么，已经再明确不过了。
<br/>
<br/>        傅夜沉不知道自己的父亲是否爱过傅昼景的母亲，但是单从龙半夏给傅昼景取的“龙之卫”这个名字来说，就能知晓龙半夏对他的父亲到底用情有多深了。
<br/>
<br/>        “大哥，开会的时间到了！你……”
<br/>
<br/>        落以柔上了二楼，刚爬出楼梯口，猛地身心惊颤得欲言又止。
<br/>
<br/>        她只见傅夜沉呆坐在床边，而床上还躺着一个女人。
<br/>
<br/>        “嗯，我现在就去会议室，通知一下你师父，让他帮我做笔录。”傅夜沉轻轻地抽回手，淡定地从床边起身，转过身来朝落以柔走了过去。
<br/>
<br/>        落以柔怔愣了片刻后，立马回过神来，讷讷地问道：“她是……”
<br/>
<br/>        “总裁夫人。”傅夜沉嘴角微扬，与落以柔擦肩而过，只身下了楼。
<br/>
<br/>        总、裁、夫、人？！
<br/>
<br/>        这四个字，犹如一道惊鸿，划破了落以柔脑海里那片平静的心湖。
<br/>
<br/>        傅夜沉已经离开了总裁办，落以柔却蹑手蹑脚地往那张单人床边走去。
<br/>
<br/>        她很好奇这位“总裁夫人”到底是谁？
<br/>
<br/>        只是令她没想到的是，竟然是她……
<br/>
<br/>        女明星苏漫雪！
<br/>
<br/>        落以柔顿时难以置信地瞪大了眼睛。
